# Translation test data

## Generating

The sample outputs with mock data can be generated by running:

### JSON

```console
./ci-support/i18n/translate                                                     \
  --base-directory-source=ci-support/i18n/test/json/en-US                       \
  --base-directory-output=ci-support/i18n/test/json                             \
  --content-type=json                                                           \
  --locale-directory-specifier="{language}-{country}"                           \
  --translation-engine=mock                                                     \
  --languages=fr,ja                                                             \
  --sources=ci-support/i18n/test/json/en-US/a.json,ci-support/i18n/test/json/en-US/b.json,ci-support/i18n/test/json/en-US/c.json,ci-support/i18n/test/json/en-US/d.json,ci-support/i18n/test/json/en-US/e.json,ci-support/i18n/test/json/en-US/f.json
```

### MD

```console
./ci-support/i18n/translate                                                    \
  --base-directory-source=ci-support/i18n/test/markdown/src                    \
  --base-directory-output=ci-support/i18n/test/markdown/i18n                   \
  --content-type=md                                                            \
  --locale-directory-specifier="{language}/docs"                               \
  --translation-engine=mock                                                    \
  --languages=fr,ja                                                            \
  --sources=ci-support/i18n/test/markdown/src/sample.md
```
