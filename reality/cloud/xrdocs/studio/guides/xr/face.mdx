## Enabling Face Effects
To enable face effects in your project, follow these steps:

1. Select the Camera in your scene
2. Set the Type to "Face"
````mdx-code-block
<img src={require('/images/studio/change-camera-to-face.png').default} alt='Camera properties include the ability to change the type of Camera.'/>
````
3. (Optional) Add a “Face” object to your scene hierarchy to visualize spatial face effects.
````mdx-code-block
<img src={require('/images/studio/add-face-object.png').default} alt='Add a Face Object from the New Entity button in the Interface.'/>
````
