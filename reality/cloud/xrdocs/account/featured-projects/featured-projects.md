# Featured Projects

Once your [Public Profile is activated](/account/public-profiles/), you can publish Featured
Projects pages to your Public Profile page. All 8th Wall projects can be featured on a public profile.
All workspaces users have permission to create, save and publish featured projects. Projects can be
added or removed to a Public Profile at any time. Publishing a Featured Project does not impact the
live published experience.
