# Back up your code

:::warning[Note]
This version does not produce a buildable project. Exported code cannot be run or deployed as-is.
:::

Code Export allows you to download a full snapshot of your project exactly as it exists on 8th Wall today. With this version, you can download code, assets, and image targets in one click.

Code Export is designed to give you immediate access and backups of your work as we move through the open source transition or to help as you evaluate using other platforms.

## Studio

To download a copy of a Studio project:

1. Open the Studio project and click **Publish**

![](/images/migration/studio-publish.png)

2. Select **Code** under the **Export** Section, and then click the **Export** button

![](/images/migration/studio-export-code.png)

3. You will recieve a .zip file containing image targets, code, and files.

![](/images/migration/studio-code-files.png)

## Cloud Editor
