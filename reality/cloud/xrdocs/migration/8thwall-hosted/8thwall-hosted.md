# 8th Wall Hosted Projects

:::danger[Important]
Access to the 8th Wall Console will end on **February 28, 2026**. Before this date, you'll need to migrate any projects you want to continue developing or keep live beyond February 2027 to a self-hosted environment.
:::

:::info
Projects containing the following features **can not** be migrated as they are not supported by the open-source 8th Wall engine:
* VPS / Maps
* Hand Tracking
* Modules / Backends
:::

This guide walks through the process of migrating 8th Wall-hosted projects to a self-hosted setup. It applies to all projects previously hosted on 8th Wall, including both **Editor** and **Studio** projects.

Specifically, you'll learn how to:

- Export your project's buildable code
- Understand the exported project structure
- Make any required updates or adjustments
- Run the project locally for development
- Deploy and host the project yourself

If you already have an existing self-hosted project, see the [self-hosted project migration guide](/migration/self-hosted).
