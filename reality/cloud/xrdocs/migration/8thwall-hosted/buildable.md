# Export Buildable Code

Buildable Code Export gives you everything required to build and run an offline version of your project, including project code and assets, image targets, build configuration, XR Engine, XRExtras and Landing Page.

:::info
Buildable code export does **not support** cloud-dependent features or those we don’t have license to distribute such as:
* VPS / Maps
* Hand Tracking
* Modules / Backends
:::

## Export a Studio project

To download a buildable Studio project:

1. Open the Studio project and click **Publish**

![](/images/migration/studio-publish.png)

2. Select **Buildable Code** under the **Export** Section, and then click the **Export** button

![](/images/migration/studio-export-buildable.png)

3. You will recieve a .zip file containing project files.

## Export a Cloud Editor project

To download a buildable Cloud Editor project:

1. Open the Editor project and click **Publish**

![](/images/migration/editor-publish.jpg)

2. Select **Buildable Code** under the **Export** Section, and then click the **Export** button

![](/images/migration/editor-export-buildable.png)

3. You will recieve a .zip file containing project files.
