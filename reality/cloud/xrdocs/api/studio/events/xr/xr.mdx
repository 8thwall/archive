# XR Events

XR events are forwarded from the 8th Wall engine. They are emitted on the camera entity and bubble up to the [world.events.globalId](/api/studio/world/events/#properties).

See the lifecycle events for [World Effects](world), [Face Effects](face), [Image Targets](image-targets), and [VPS](vps).

## Events

### REALITY_READY

This event is emitted when 8th Wall Web has initialized and at least one frame has been successfully processed. This is the recommended time at which any permissions/loading UI should be hidden.

#### Properties

None.
