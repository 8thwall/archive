# Physics Events

## Events

### COLLISION_START_EVENT
Emitted when the entity has started colliding with another entity.

#### Properties

| Property | Type | Description                     |
|----------|------|---------------------------------|
| other    | eid  | The eid of the colliding entity |

#### Example

<EventExample
  ns='ecs.physics'
  event='COLLISION_START_EVENT'
  callbackBody={`console.log('Collided with', event.data.other)`}
/>

### COLLISION_END_EVENT
Emitted when the entity has stopped colliding with another entity.

#### Properties

| Property | Type | Description                     |
|----------|------|---------------------------------|
| other    | eid  | The eid of the colliding entity |

#### Example

<EventExample
  ns='ecs.physics'
  event='COLLISION_END_EVENT'
  callbackBody={`console.log('Stopped colliding with', event.data.other)`}
/>

### UPDATE_EVENT
Emitted on the [world.events.globalId](/api/studio/world/events/#properties) immediately after the physics update before rendering.

#### Properties

None.

#### Example

<EventExample
  ns='ecs.physics'
  event='UPDATE_EVENT'
  target='world.events.globalId'
  hasPayload={false}
  callbackBody={`console.log('Physics update')`}
/>
