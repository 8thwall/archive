# Asset Events

Asset events are emitted on the asset entity and bubble up to the [world.events.globalId](/api/studio/world/events/#properties).

## 3D Model

### GLTF_MODEL_LOADED

Emitted when a model has loaded

#### Properties

| Property | Type                                                    | Description        |
|----------|---------------------------------------------------------|--------------------|
| model    | [Group](https://threejs.org/docs/#api/en/objects/Group) | The three.js Group |

#### Example

<EventExample
  ns='ecs.events'
  event='GLTF_MODEL_LOADED'
  callbackBody={`console.log('Model loaded: ', event.data.model)`}
/>

### GLTF_ANIMATION_FINISHED

Emitted when all loops of an animation clip have finished.

#### Properties

| Property | Type   | Description               |
|----------|--------|---------------------------|
| name     | string | The name of the animation |

#### Example

<EventExample
  ns='ecs.events'
  event='GLTF_ANIMATION_FINISHED'
  callbackBody={`console.log('Animation finished: ', event.data.name)`}
/>

### GLTF_ANIMATION_LOOP

Emitted when a single loop of the animation clip has finished.

#### Properties

| Property | Type   | Description               |
|----------|--------|---------------------------|
| name     | string | The name of the animation |

#### Example

<EventExample
  ns='ecs.events'
  event='GLTF_ANIMATION_LOOP'
  callbackBody={`console.log('Animation loop: ', event.data.name)`}
/>

## Audio

### AUDIO_CAN_PLAY_THROUGH

Emitted when an entity has the capability to play Audio.

#### Properties

None.

#### Example

<EventExample
  ns='ecs.events'
  event='AUDIO_CAN_PLAY_THROUGH'
  hasPayload={false}
  callbackBody={`console.log('Ready to play audio')`}
/>

### AUDIO_END

Emitted when audio has finished playing on an entity.

#### Properties

None.

#### Example

<EventExample
  ns='ecs.events'
  event='AUDIO_END'
  hasPayload={false}
  callbackBody={`console.log('Finished playing audio')`}
/>
