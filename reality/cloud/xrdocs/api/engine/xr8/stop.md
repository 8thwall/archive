# XR8.stop()

`XR8.stop()`

## Description {#description}

While stopped, the camera feed is closed and device motion is not tracked. Must call
`XR8.run()` to restart after the engine is stopped.

## Parameters {#parameters}

None

## Returns {#returns}

None

## Example {#example}

```javascript
XR8.stop()
```
