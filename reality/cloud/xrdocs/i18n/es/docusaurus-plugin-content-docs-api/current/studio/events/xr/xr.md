# Eventos XR

Los eventos XR son emitidos por el motor 8th Wall a través de módulos de canalización de cámara. Los módulos de canalización de cámara que emiten eventos son: `reality` y `facecontroller`.

## realidad {#reality}

Eventos emitidos por el módulo `reality`:

- Eventos Image Target
- Eventos VPS
- Eventos de Efecto Mundial
