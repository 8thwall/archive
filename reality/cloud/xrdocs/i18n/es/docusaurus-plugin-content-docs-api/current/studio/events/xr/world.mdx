# Efectos mundiales Eventos

## Eventos {#events}

### REALITY_TRACKING_STATUS

Este evento es emitido por World Effects cuando el motor arranca y cambia el estado o motivo del seguimiento.

#### Propiedades {#properties}

| Propiedad | Tipo   | Descripción                                       |
| --------- | ------ | ------------------------------------------------- |
| Estado    | Cadena | Una de las opciones `LIMITED` o `NORMAL`.         |
| motivo    | Cadena | Una de las opciones `INITIALIZING` o `UNDEFINED`. |
