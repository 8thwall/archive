# VideoControls

## Descripción

Este componente permite a las entidades controlar la reproducción de vídeo en las texturas aplicadas a los materiales. Se añade automáticamente cuando se configura un vídeo a través del configurador de Studio. Si el vídeo se añade mediante programación, este componente debe añadirse manualmente para habilitar los controles de reproducción.
