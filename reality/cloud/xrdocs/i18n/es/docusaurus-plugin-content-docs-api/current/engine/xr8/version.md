# XR8.version()

`XR8.version()`

## Descripción {#description}

Consigue la versión del motor web de 8th Wall.

## Parámetros {#parameters}

Ninguno

## Devuelve {#returns}

Cadena que indica la versión del motor web de 8th Wall.

## Ejemplo {#example}

```javascript
console.log(XR8.version())
```
