# XR8.XrController.recenter()

`XR8.XrController.recenter()`

## Descripción {#description}

Reposiciona la cámara a la dirección de origen / orientación especificada por [`XR8.XrController.updateCameraProjectionMatrix()`](updatecameraprojectionmatrix.md) y reinicia el seguimiento.

## Parámetros {#parameters}

Ninguna

## Devuelve {#returns}

Ninguna
