# XR8.isInitialized()

`XR8.isInitialized()`

## Descripción {#description}

Indica si el WebAssembly del AR Engine está inicializado o no.

## Parámetros {#parameters}

Ninguno

## Devuelve {#returns}

Un booleano que indica si el WebAssembly del AR Engine está inicializado o no.

## Ejemplo {#example}

```javascript
if (XR8.isInitialized()) {
  console.log(XR8.version())
}
```
