# onRemove()

`onRemove: ({framework})`

## Descripción {#description}

Se llama a `onRemove` cuando se elimina un módulo del pipeline.

## Parámetros {#parameters}

| Parámetro | Descripción                                                          |
| --------- | -------------------------------------------------------------------- |
| marco     | Los enlaces de este módulo para el envío de eventos. |
