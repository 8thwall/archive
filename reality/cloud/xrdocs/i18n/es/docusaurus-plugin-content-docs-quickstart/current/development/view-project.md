# Ver el proyecto público

1. Vuelva al Panel de control del proyecto en la navegación de la izquierda.  En la sección **QR 8.code**, se mostrará la URL pública del proyecto
   junto con un enlace corto a **8th.io** y el código QR asociado.

2. Escanee el código QR con su dispositivo móvil para ver el proyecto público.
