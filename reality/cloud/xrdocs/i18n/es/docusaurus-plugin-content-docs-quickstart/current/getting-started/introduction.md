# Introducción

8th Wall proporciona la solución completa para crear, colaborar y publicar experiencias WebAR, WebVR
y juegos 3D que se ejecutan directamente en un navegador web, sin necesidad de aplicaciones.

**8th Wall's ya está disponible para todo el mundo de forma gratuita, sin necesidad de tarjeta de crédito.**

## Cómo funciona {#how-it-works}

Construido íntegramente con JavaScript y WebGL compatibles con los estándares, 8th Wall Web es una implementación completa en
del motor de localización y mapeo simultáneos (SLAM) de 8th Wall, hiperoptimizado
para RA web en tiempo real en navegadores. Entre las funciones de RA se incluyen el seguimiento del mundo, los objetivos de imagen, los efectos faciales,
y la segmentación del cielo.

**Studio** es un editor visual y motor de juegos XR en tiempo real que combina la velocidad y la potencia de la web
con las herramientas necesarias para crear experiencias 3D y XR hiperinmersivas. Studio cuenta con un editor 3D visual
interfaz para crear XR y juegos web a través de dispositivos. Cuenta con un motor totalmente nuevo que incluye sistemas para la física,
para la iluminación de la cámara y el audio, para los materiales de animación y las partículas, para el seguimiento de las acciones del jugador, y mucho más.
Con Studio, puede crear atractivas experiencias WebAR, juegos 3D y mucho más en tiempo real e implantar fácilmente
en dispositivos móviles, ordenadores de sobremesa y auriculares avanzados. [Más información sobre Studio](/studio).

El alojamiento integrado le permite publicar proyectos en varios estados de despliegue de
alojados en la fiable y segura red global de 8th Wall, incluido un entorno de ensayo protegido por contraseña de
.

## Características de la plataforma {#platform-features}

La plataforma para desarrolladores de 8th Wall te ofrece un montón de funciones útiles, así que echemos un vistazo rápido
a algunas de las principales:

📱 **Crea** ricas experiencias Web AR y Web VR que se ejecutan directamente en un navegador web móvil.

👥 **Colaborar** con los miembros del equipo y almacenar el código en el control de origen.

👀 **previsualiza al instante** los proyectos a medida que los desarrollas.

🛠 **depura** inalámbricamente tu código en tiempo real con registros de consola en vivo desde múltiples dispositivos.

🌎 **Publicar** proyectos alojados en la red mundial de 8th Wall.

📣 Crea un **Perfil público** y **Proyectos destacados** en 8thwall.com, lo que te permitirá mostrar tu trabajo, demos en directo y/o código.
