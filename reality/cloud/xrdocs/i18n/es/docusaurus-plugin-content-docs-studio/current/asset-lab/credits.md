# Créditos

Se necesitan créditos para utilizar la generación de activos mediante IA en Studio. Cada acción consume un número determinado de créditos en función del modelo o servicio utilizado. Todas las cuentas comienzan con un conjunto de créditos gratuitos al registrarse, y se pueden adquirir créditos adicionales mediante una suscripción -o pronto mediante una recarga- a través de la página de su Cuenta.

Puede adquirir una suscripción de crédito o recargar desde su [Configuración de cuenta](/account/settings/manage-credits/).
