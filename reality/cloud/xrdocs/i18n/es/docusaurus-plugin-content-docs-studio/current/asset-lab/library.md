# Biblioteca de activos

En la Biblioteca del Laboratorio de Activos:

- Acceder a todos los activos generados en el espacio de trabajo
- Compruebe su saldo en la parte superior derecha
- Filtrar por tipo: Imágenes, Modelos 3D, Personajes animados
- Utilice las herramientas de búsqueda y clasificación para organizar

![](/images/studio/asset-lab/library1.png)

Haga clic en un activo para ver los metadatos:

- Pregunta utilizada
- Usuario que ha enviado
- Modelo utilizado
- Coste del crédito

Acciones disponibles:

- **Enviar a flujo de trabajo** - Reutilizar activo como entrada
- **Descargar** - Guardar localmente
- **Remix** - Ajustar la configuración de entrada
- **Import** - Añadir a la carpeta de activos del proyecto
