# Componentes

## Introducción

{frontMatter.description} Cada Componente es un bloque reutilizable de funcionalidad que puede adjuntarse a una entidad, permitiéndole exhibir comportamientos o propiedades específicas. Los componentes pueden definir el aspecto visual, las propiedades físicas, la gestión de las entradas o la lógica de juego personalizada. Combinando varios Componentes, puede crear entidades complejas con un comportamiento rico.

## Componentes

Los componentes son los bloques de construcción que dan a las entidades su funcionalidad. Mientras que una entidad representa un objeto en blanco, los Componentes definen cómo ese objeto interactúa con el mundo, qué aspecto tiene o cómo se comporta. En 8th Wall Studio, puedes utilizar componentes integrados o crear tus propios componentes personalizados para definir comportamientos únicos para tu juego.
