# Entidades

## Introducción

{frontMatter.description} Una entidad por sí misma no tiene comportamiento ni apariencia; simplemente actúa como un contenedor al que se pueden adjuntar componentes. Estos componentes definen el comportamiento y las características de la entidad, como su posición, aspecto visual o interacción con sistemas físicos.

## Entidades

Las entidades constituyen la columna vertebral de cualquier juego o simulación en 8th Wall Studio. Combinando varios componentes, se pueden crear objetos, sistemas e interacciones complejos.

\*\*Las entidades están representadas por un número entero único de 64 bits denominado ID de entidad o eid.
