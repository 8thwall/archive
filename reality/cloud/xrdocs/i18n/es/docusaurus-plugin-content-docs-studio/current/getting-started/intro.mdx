# Introducción

:::warning[Important Actualización]
La 8ª Muralla se reducirá a lo largo del próximo año - [lea ahora la actualización completa](/migration).
:::

{frontMatter.description} Con Studio, puede crear fácilmente atractivas experiencias WebAR, juegos 3D interactivos y mucho más en tiempo real, y luego desplegarlos sin problemas en dispositivos móviles, ordenadores de sobremesa y auriculares avanzados.

Studio presenta una solución completa para crear, colaborar y publicar experiencias WebAR, WebVR y juegos 3D que se ejecutan directamente en un navegador web, sin necesidad de aplicaciones.

![Intro-Image](/images/studio/introducing-studio.png)

Studio es una herramienta basada en navegador que no requiere descarga de software. Para utilizar Studio, [inicie sesión en su cuenta de 8th Wall](https://www.8thwall.com/login), o [cree una cuenta nueva](https://www.8thwall.com/get-started) si aún no tiene una.
