# Construir y aterrizar

Haz clic en **Construir** para guardar tu trabajo e iniciar una nueva compilación en la nube de tu proyecto.

![StudioLaunchToolbar](/images/studio/studio-navigate-launch-toolbar.png)

Una vez satisfecho con los cambios, **coloca** el código actualizado en el control de código fuente integrado de Studio.

Cuando el botón Aterrizar está verde, indica que hay cambios en el proyecto que aún no se han aterrizado en el control de origen.

Tras hacer clic en aterrizar, se le pedirá que introduzca un breve mensaje describiendo los cambios realizados. Introduzca un mensaje y haga clic en **Aterrizar**:

![StudioLand](/images/studio/studio-navigate-land.png)
