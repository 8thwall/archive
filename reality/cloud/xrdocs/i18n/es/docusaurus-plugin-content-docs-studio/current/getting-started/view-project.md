# Ver el proyecto público

1. Vaya al panel de control del proyecto. Desde Studio, puedes hacer clic en el menú hamburguesa de la esquina superior izquierda. En la sección **QR 8.code** del panel de control del proyecto, se mostrará la URL pública del proyecto junto con un enlace corto a **8th.io** y el código QR asociado.

2. Escanee el código QR con su dispositivo móvil para ver el proyecto público.
