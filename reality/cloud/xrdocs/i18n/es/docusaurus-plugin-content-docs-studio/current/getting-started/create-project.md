# Crear un proyecto

1. En la página de inicio o en el panel del espacio de trabajo, haga clic en "Nuevo proyecto".

![StartNewProject](/images/console-workspace-start-project.jpg)
2. Seleccione el espacio de trabajo para este proyecto si es miembro de más de un espacio de trabajo.

3. Introduzca la URL del proyecto.

:::warning
La URL del proyecto se utiliza para generar la URL predeterminada del proyecto, así como la URL de la página Proyecto destacado (si está activada).  \*\*No se puede cambiar. Para cambiar la URL de un proyecto, tendrás que clonarlo y especificar una nueva URL.
:::

![ProjectUrl](/images/account/project-create.jpg)

4. Haga clic en **Crear**.

5. **Seleccione una plantilla**: Cuando abras el nuevo proyecto, se te presentarán unas cuantas Plantillas de Proyecto
   para ayudarte a empezar, incluyendo plantillas orientadas al juego y a la RA. Estas plantillas de proyectos de
   muestran diferentes funcionalidades y pueden modificarse fácilmente introduciendo sus propios recursos en
   . Cree un proyecto utilizando una de las plantillas disponibles o cree un proyecto a partir de la plantilla Vacío
   .

![StudioTemplatePicker](/images/studio/studio-template-picker.png)

Cuando tu proyecto haya cargado todo lo que necesita para empezar, puedes empezar a crear.
