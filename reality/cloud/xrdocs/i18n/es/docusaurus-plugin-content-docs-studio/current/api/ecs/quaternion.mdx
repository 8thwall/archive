# Quaternion

## Descripción

This component controls how the entity is rotated.

## Propiedades

| Propiedad | Tipo   | Por defecto | Descripción  |
| --------- | ------ | ----------- | ------------ |
| x         | number | 0           | x coordinate |
| y         | number | 0           | y coordinate |
| z         | number | 0           | z coordinate |
| w         | number | 0           | w coordinate |

## Funciones

        w: 0
    }}
