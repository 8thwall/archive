# RingGeometry

## Descripción

This component establishes geometry on an entity.

## Propiedades

| Propiedad    | Tipo   | Por defecto | Descripción                           |
| ------------ | ------ | ----------- | ------------------------------------- |
| innerRadius  | number | 1           | Radius of the inner hole of the ring. |
| outerRadius  | number | 1           | Radius of the outer edge of the ring. |

## Funciones

<ComponentFunctions
    name="RingGeometry"
    properties={{ innerRadius: 1, outerRadius: 1 }}
    mutate={`cursor.innerRadius += 0.5;
    cursor.outerRadius += 1;
    return false;`}
/>
