# CapsuleGeometry

## Descripción

This component establishes geometry on an entity.

## Propiedades

| Propiedad | Tipo   | Por defecto | Descripción            |
| --------- | ------ | ----------- | ---------------------- |
| radio     | number | 1           | Radius of the capsule. |
| height    | number | 1           | Height of the capsule. |

## Funciones

<ComponentFunctions
    name="CapsuleGeometry"
    properties={{ radius: 1, height: 1 }}
    mutate={`cursor.radius += 1;
    cursor.height *= 2;
    return false;`}
/>
