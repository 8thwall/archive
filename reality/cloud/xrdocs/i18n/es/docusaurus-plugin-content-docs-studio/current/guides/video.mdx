# Vídeo

## Introducción

{frontMatter.description} Por defecto, todas las texturas de vídeo y los vídeos de interfaz de usuario se reproducen automáticamente en modo silenciado. Para personalizar el comportamiento del vídeo, utilice [`VideoControls`](/api/studio/ecs/video-controls).

## Añadir vídeos

Para aplicar una textura de vídeo a una entidad, selecciona un vídeo de tus activos en el desplegable de cualquier mapa de textura.

Las texturas de vídeo sólo pueden aplicarse a materiales estándar y no iluminados para primitivas.
Para las entidades GLB, las texturas de vídeo pueden aplicarse utilizando Material de Vídeo, que anulará todos los mapas de color del modelo.
