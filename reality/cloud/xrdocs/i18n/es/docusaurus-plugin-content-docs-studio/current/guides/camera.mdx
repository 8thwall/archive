## Introducción

{frontMatter.description}

## Añadir una cámara

Una cámara se proporciona por defecto en un nuevo proyecto de Studio, sin embargo, también puede agregar una cámara a una escena utilizando el desplegable + Objeto en la Jerarquía Espacial. Además, se puede añadir una cámara a una entidad mediante la interfaz Añadir componente de Studio o en código.

![camera-new-object.png](/images/studio/camera-new-object.png)

## Cambiar la cámara activa

Haga clic en el botón _Set Active Camera_ en las propiedades de la cámara.

![camera-set-active.png](/images/studio/camera-set-active.png)

## Cambiar el tipo de cámara

Hay tres tipos de cámaras en Studio: Sólo 3D, Cara y Mundo. Cada uno de estos tipos de cámara tendrá una configuración diferente. Para las experiencias de RA se necesita una cámara facial o mundial. \*\*Para obtener más información sobre la creación de experiencias de realidad aumentada, consulte la sección de guías XR.
