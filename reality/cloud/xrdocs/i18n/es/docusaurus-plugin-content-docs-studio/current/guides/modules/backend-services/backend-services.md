# Servicios de backend

8th Wall Studio es ahora compatible con dos nuevas y potentes funciones: Backend Proxies y Backend Functions.

Estas herramientas aportan nuevas capacidades a 8th Wall Studio, equipando a los desarrolladores para manejar la lógica backend
e integrar servicios externos directamente en sus WebAR y juegos web.
