## Activación de los efectos faciales

Para activar los efectos faciales en tu proyecto, sigue estos pasos:

1. Seleccione la cámara en su escena
2. Ajuste el tipo a "Cara

```mdx-code-block
<img src={require('/images/studio/change-camera-to-face.png').default} alt='Camera properties include the ability to change the type of Camera.'/>
```

3. (Opcional) Añade un objeto "Cara" a tu jerarquía de escenas para visualizar efectos espaciales de caras.

```mdx-code-block
<img src={require('/images/studio/add-face-object.png').default} alt='Add a Face Object from the New Entity button in the Interface.'/>
```
