# Cuadro de mando del proyecto

El panel de control del proyecto es el centro de gestión de los proyectos de 8th Wall.  Desde la página del panel de control del proyecto
puede gestionar la configuración del proyecto, acceder al editor de 8th Wall Cloud, comprar suscripciones de marca blanca,
gestionar objetivos de imagen, configurar dominios personalizados y mucho más.

La URL directa a su Cuadro de mandos del proyecto tiene el formato: www.8thwall.com/workspacename/projectname
