# Crear un espacio de trabajo libre

1. Introduzca un nombre para el espacio de trabajo. Este valor es sólo a efectos de visualización y no afecta a ninguna URL
   asociada a su espacio de trabajo.

2. Introduzca una URL del espacio de trabajo. Elige algo relevante para el nombre de tu espacio de trabajo, como el nombre de tu empresa
   . La URL del espacio de trabajo se utilizará como **subdominio** por defecto para **todos** los proyectos alojados en 8th Wall
   de su cuenta (por ejemplo, **miempresa**.8thwall.app/nombredelproyecto). Este valor también se utilizará
   en la URL de su página de perfil público (por ejemplo, www.8thwall.com/mycompany).

:::warning
\*\*Sin embargo, si desea conectar dominios personalizados a sus proyectos alojados en 8th Wall para anular la URL predeterminada de
, consulte [aquí](/account/projects/connected-domains/).
:::
