# Exportar proyectos

## Exportación de códigos

La exportación de código ya está disponible tanto para los proyectos de Studio como para los de Cloud Editor. Esto le permitirá descargar una instantánea completa de su proyecto tal y como existe hoy en 8th Wall. Con esta versión, puedes **descargar código, activos y objetivos de imagen con un solo clic**.

Code Export está diseñado para ofrecerle acceso inmediato y copias de seguridad de su trabajo a medida que avanzamos en la transición al código abierto o para ayudarle a evaluar el uso de otras plataformas.

:::warning[Note]
Esta versión no produce un proyecto compilable. El código exportado no puede ejecutarse ni desplegarse tal cual. La exportación edificable llegará pronto.
:::

### Estudio

Para descargar una copia de un proyecto de Studio:

1. Abre el proyecto de Studio y haz clic en **Publicar**.

![](/images/migration/studio-publish.png)

2. Seleccione **Código** en la sección **Exportar** y, a continuación, haga clic en el botón **Exportar**.

![](/images/migration/studio-export-code.png)

3. Recibirá un archivo .zip con imágenes, código y archivos.

![](/images/migration/studio-files.png)

### Editor de nubes

Para descargar una copia de un proyecto del Editor de Nubes:

1. Abra el proyecto Editor y haga clic en **Publicar**.

![](/images/migration/editor-publish.jpg)

2. Seleccione **Código** en la sección **Exportar** y, a continuación, haga clic en el botón **Exportar**.

![](/images/migration/editor-export-code.png)

3. Recibirá un archivo .zip con imágenes, código y archivos.

![](/images/migration/editor-files.png)

## Exportación de código edificable

En unas semanas, lanzaremos Buildable Code Export. Esta versión incluirá todo lo necesario para construir y ejecutar una versión offline de su proyecto, incluyendo:

* Código del proyecto y activos
* Objetivos de imagen
* Construir la configuración
* Motor XR
* Extras XR
* Página de aterrizaje

Sin embargo, el desarrollo offline **no soportará** funciones dependientes de la nube o aquellas para las que no tengamos licencia de distribución, como:

* VPS / Mapas
* Seguimiento manual
* Módulos / Backends
