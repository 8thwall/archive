# xrlefteyebrowlowered

## Descripción {#description}

Este evento lo emite [`xrface`](/api/aframe/#face-effects) cuando la ceja izquierda de una cara rastreada baja a su posición inicial al encontrarla.

`xrlefteyebrowlowered.detail : {id}`

| Propiedad | Tipo     | Descripción                                      |
| --------- | -------- | ------------------------------------------------ |
| id        | `Number` | Un identificador numérico de la cara localizada. |
