# XR8.FaceController

## Descripción {#description}

`FaceController` proporciona detección y mallado de caras, e interfaces para configurar el seguimiento.

## Funciones {#functions}

| Función                                 | Descripción                                                                                                                                                                                                                                       |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)               | Configura qué procesamiento realiza FaceController.                                                                                                                                                                                               |
| [pipelineModule](pipelinemodule.md)     | Crea un módulo de canalización de cámara que, cuando se instala, recibe llamadas de retorno sobre cuándo se ha iniciado la cámara, eventos de proceso de la cámara y otros cambios de estado. Se utilizan para calcular la posición de la cámara. |
| [AttachmentPoints](attachmentpoints.md) | Puntos de la cara a los que puedes anclar contenidos.                                                                                                                                                                                             |
| [MeshGeometry](meshgeometry.md)         | Opciones para definir qué partes de la cara tienen triángulos de malla devueltos.                                                                                                                                                                 |
