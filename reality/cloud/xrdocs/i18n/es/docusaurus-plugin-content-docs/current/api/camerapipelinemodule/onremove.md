# onRemove()

`onRemove: ({framework})`

## Descripción {#description}

`onRemove` se llama cuando se elimina un módulo de la canalización.

## Parámetros {#parameters}

| Parámetro | Descripción                                               |
| --------- | --------------------------------------------------------- |
| framework | Los enlaces del marco de este módulo para enviar eventos. |
