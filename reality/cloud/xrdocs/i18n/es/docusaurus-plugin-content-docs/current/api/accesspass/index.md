# AccessPass

## Descripción {#description}

Proporciona un módulo para monetizar tu experiencia WebAR y WebVR. Esto sólo está disponible para los proyectos alojados en 8th Wall, y requiere el [Módulo de Pagos](/guides/advanced-topics/monetize-with-8th-wall-payments).

```ts
import {AccessPass} from 'payments'
```

## Funciones {#functions}

| Función                                               | Descripción                                                                                                            |
| ----------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| [requestPurchaseIfNeeded](requestpurchaseifneeded.md) | Abre una ventana de pago en la que el cliente puede realizar de forma segura el pago del pase de acceso proporcionado. |
