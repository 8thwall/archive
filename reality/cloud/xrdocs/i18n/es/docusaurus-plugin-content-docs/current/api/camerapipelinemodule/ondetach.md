# onDetach()

`onDetach: ({framework})`

## Descripción {#description}

`onDetach` se llama después de la última vez que un módulo recibe actualizaciones de tramas. Esto ocurre después de parar el motor o de retirar manualmente el módulo pipeline, lo que ocurra primero.

## Parámetros {#parameters}

| Parámetro | Descripción                                               |
| --------- | --------------------------------------------------------- |
| framework | Los enlaces del marco de este módulo para enviar eventos. |
