# XR8.initialize()

`XR8.initialize()`

## Descripción {#description}

Devuelve una promesa que se cumple cuando se inicializa el WebAssembly del AR Engine.

## Parámetros {#parameters}

Ninguno

## Vuelta {#returns}

Ninguno

## Ejemplo {#example}

```javascript
XR8.initialize().then(() => console.log(XR8.version())
```
