# xrlefteyeopened

## Descripción {#description}

Este evento lo emite [`xrface`](/api/aframe/#face-effects) cuando se abre el ojo izquierdo de una cara rastreada.

`xrlefteyeopened.detail : {id}`

| Propiedad | Tipo     | Descripción                                      |
| --------- | -------- | ------------------------------------------------ |
| id        | `Number` | Un identificador numérico de la cara localizada. |
