# xrinterpupillarydistance

## Descripción {#description}

Este evento lo emite [`xrface`](/api/aframe/#face-effects) cuando se detecta por primera vez la distancia en milímetros entre los centros de cada pupila de una cara rastreada.

`xrinterpupillarydistance.detail : {id}`

| Propiedad              | Tipo     | Descripción                                                          |
| ---------------------- | -------- | -------------------------------------------------------------------- |
| id                     | `Number` | Un identificador numérico de la cara localizada.                     |
| interpupillaryDistance | `Number` | Distancia aproximada en milímetros entre los centros de cada pupila. |
