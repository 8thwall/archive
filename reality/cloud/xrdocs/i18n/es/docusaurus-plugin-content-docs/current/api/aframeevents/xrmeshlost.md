# xrmeshlost

## Descripción {#description}

Este evento es emitido por [`xrweb`](/api/aframe/#world-tracking-image-targets-andor-lightship-vps) cuando se llama a `recenter()`.

`xrmeshlost.detail : { id }`

| Propiedad | Descripción                                               |
| --------- | --------------------------------------------------------- |
| id        | Un id para esta malla que es estable dentro de una sesión |
