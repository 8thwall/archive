# XR8.Babylonjs

Babylon.js (<https://www.babylonjs.com/>) es un completo marco JavaScript para construir juegos 3D y experiencias con HTML5 y WebGL.  Combinado con 8th Wall Web, puedes crear potentes experience WebAR.

Vídeo tutorial:

* Integración de Babylon.js y 8th Wall Web: <https://youtu.be/RhZaebNpaEY>

## Descripción {#description}

Proporciona una integración que interactúa con el entorno BabylonJS y el estilo de vida para manejar la cámara Babylon.js para hacer superposiciones virtuales.

## Funciones {#functions}

| Función                                     | Descripción                                                                                                                       |
| ------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| [xrCameraBehavior](xrcamerabehavior.md)     | Consigue un comportamiento que se pueda asociar a una cámara Babylon para ejecutar seguimiento del mundo y/o objetivos de imagen. |
| [faceCameraBehavior](facecamerabehavior.md) | Consigue un comportamiento que se pueda conectar a una cámara Babylon para ejecutar los efectos faciales.                         |
