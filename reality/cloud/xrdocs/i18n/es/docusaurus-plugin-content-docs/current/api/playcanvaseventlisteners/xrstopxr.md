# xr:stopxr

`this.app.fire('xr:stopxr')`

## Descripción {#description}

Detiene la sesión XR actual.  Mientras está parado, la alimentación de la cámara se detiene y no se sigue el movimiento del dispositivo.

## Parámetros {#parameters}

Ninguno

## Ejemplo {#example}

```javascript
this.app.fire('xr:stopxr')
```
