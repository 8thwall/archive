# XR8.MediaRecorder

## Descripción {#description}

Proporciona un módulo de canalización de cámara que te permite grabar un vídeo en formato MP4.

## Funciones {#functions}

| Función                                   | Descripción                                                                                           |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| [configure](configure.md)                 | Configura los ajustes de grabación de vídeo.                                                          |
| [pipelineModule](pipelinemodule.md)       | Crea un módulo de canalización que grabe vídeo en formato MP4.                                        |
| [recordVideo](recordvideo.md)             | Inicia la grabación.                                                                                  |
| [requestMicrophone](requestmicrophone.md) | Activa la grabación de audio (si no se activa automáticamente), solicitando permisos si es necesario. |
| [stopRecording](stoprecording.md)         | Detén la grabación.                                                                                   |
| [RequestMicOptions](requestmicoptions.md) | Enum para solicitar o no automáticamente permisos de micrófono.                                       |
