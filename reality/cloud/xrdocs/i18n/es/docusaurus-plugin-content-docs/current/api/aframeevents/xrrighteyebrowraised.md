# xrrighteyebrowraised

## Descripción {#description}

Este evento lo emite [`xrface`](/api/aframe/#face-effects) cuando la ceja derecha de una cara rastreada se levanta de su posición cuando se encontró la cara.

`xrrighteyebrowraised.detail : {id}`

| Propiedad | Tipo     | Descripción                                      |
| --------- | -------- | ------------------------------------------------ |
| id        | `Number` | Un identificador numérico de la cara localizada. |
