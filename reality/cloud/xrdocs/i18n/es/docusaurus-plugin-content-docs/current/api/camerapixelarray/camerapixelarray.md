# XR8.CameraPixelArray

## Descripción {#description}

Proporciona un módulo de canalización de cámara que da acceso a los datos de la cámara como una matriz uint8 en escala de grises o en color.

## Funciones {#functions}

| Función                             | Descripción                                                                                                                                                                                           |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [pipelineModule](pipelinemodule.md) | Un módulo de canalización que proporciona la textura de la cámara como una matriz de valores de píxeles RGBA o en escala de grises que puede utilizarse para el procesamiento de imágenes por la CPU. |
