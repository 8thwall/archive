# XR8.FaceController.MeshGeometry

Enumeración

## Descripción {#description}

Opciones para definir qué partes de la cara tienen triángulos de malla devueltos.

## Propiedades {#properties}

| Propiedad | Valor     | Descripción                        |
| --------- | --------- | ---------------------------------- |
| FACE      | `'face'`  | Devuelve la geometría de la cara.  |
| MOUTH     | `'mouth'` | Devuelve la geometría de la boca.  |
| EYES      | `'eyes'`  | Devuelve la geometría de los ojos. |
| IRIS      | `'iris'`  | Devuelve la geometría de los iris. |
