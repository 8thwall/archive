# VpsCoachingOverlay

## Descripción {#description}

Proporciona un módulo que genera una superposición de coaching para tu experiencia de WebAR habilitada para VPS Lightship.

Para obtener información sobre la superposición de coaching de escala absoluta, consulta [aquí](/api/coachingoverlay).

## Funciones {#functions}

| Función                                                 | Descripción                                                                                                                              |
| ------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](vps-coachingoverlay-configure.md)           | Configura los ajustes de la superposición de coaching.                                                                                   |
| [pipelineModule](vps-coachingoverlay-pipelinemodule.md) | Crea un módulo de canalización de cámara que, cuando se instala, añade la funcionalidad de superposición de entrenamiento a tu proyecto. |
