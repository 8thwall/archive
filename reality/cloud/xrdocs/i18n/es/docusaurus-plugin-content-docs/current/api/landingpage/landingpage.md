# LandingPage

## Descripción {#description}

Proporciona un módulo que genera una página de destino personalizada para tu experiencia WebAR.

## Funciones {#functions}

| Función                             | Descripción                                                                                                              |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| [configure](configure.md)           | Configura los ajustes de LandingPage.                                                                                    |
| [pipelineModule](pipelinemodule.md) | Crea un módulo de canalización de cámara que, cuando se instala, añade funcionalidad de página de destino a tu proyecto. |
