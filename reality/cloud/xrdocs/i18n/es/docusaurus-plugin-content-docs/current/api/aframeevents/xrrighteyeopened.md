# xrrighteyeopened

## Descripción {#description}

Este evento lo emite [`xrface`](/api/aframe/#face-effects) cuando se abre el ojo derecho de una cara rastreada.

`xrrighteyeopened.detail : {id}`

| Propiedad | Tipo     | Descripción                                      |
| --------- | -------- | ------------------------------------------------ |
| id        | `Number` | Un identificador numérico de la cara localizada. |
