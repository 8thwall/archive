# xr:meshlost

## Descripción {#description}

Este evento se emite cuando se llama a `recenter()`.

`xr:meshlost.detail : { id }`

| Propiedad | Descripción                                                |
| --------- | ---------------------------------------------------------- |
| id        | Un id para esta malla que es estable dentro de una sesión. |
