# CoachingOverlay

## Descripción {#description}

Proporciona un módulo que genera un Coaching Overlay para su experiencia de WebAR a Escala Absoluta.

Para obtener información sobreCoaching Overlay VPS Lightship, consulte [aquí](/api/vpscoachingoverlay).

## Funciones {#functions}

| Función                             | Descripción                                                                                                                |
| ----------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Configura los ajustes de Coaching Overlay.                                                                                 |
| [pipelineModule](pipelinemodule.md) | Crea un módulo de canalización de cámara que, cuando se instala, añade la funcionalidad de Coaching Overlay a su proyecto. |
