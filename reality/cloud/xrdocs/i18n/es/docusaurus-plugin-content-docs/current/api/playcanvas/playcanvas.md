# XR8.PlayCanvas

PlayCanvas (<https://www.playcanvas.com/>) es un motor de juegos 3D de código abierto/motor de aplicaciones 3D interactivas junto con una plataforma propia de creación hospedada en la nube que permite la edición simultánea desde varios ordenadores a través de una interfaz basada en navegador.

## Descripción {#description}

Proporciona una integración que interactúa con el entorno y el ciclo de vida de PlayCanvas para manejar la cámara PlayCanvas para hacer superposiciones virtuales.

## Funciones {#functions}

| Función                                          | Descripción                                                                                                                |
| ------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------- |
| [run](run.md)                                    | Abre la cámara con los Módulos Pipeline especificados y comienza a ejecutarse en una escena PlayCanvas.                    |
| [runXr (obsoleto)](runxr.md)                     | Abre la cámara y comienza a ejecutar el Seguimiento del Mundo y/o el Seguimiento de la Imagen en una escena de PlayCanvas. |
| [runFaceEffects (obsoleto)](runfaceeffects.md)   | Abre la cámara y empieza a ejecutar Efectos faciales en una escena de PlayCanvas.                                          |
| [stop](stop.md)                                  | Elimina los módulos añadidos en [ejecutar](run.md) y detiene la cámara.                                                    |
| [stopXr (obsoleto)](stopxr.md)                   | Elimina los módulos añadidos en [runXr](runxr.md) y detiene la cámara.                                                     |
| [stopFaceEffects (obsoleto)](stopfaceeffects.md) | Elimina los módulos añadidos en [runFaceEffects](runfaceeffects.md) y detiene la cámara.                                   |
