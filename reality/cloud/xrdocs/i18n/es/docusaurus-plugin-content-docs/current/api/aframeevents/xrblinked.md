# xrblinked

## Descripción {#description}

Este evento lo emite [`xrface`](/api/aframe/#face-effects) cuando los ojos de una cara rastreada parpadean.

`xrblinked.detail : {id}`

| Propiedad | Tipo     | Descripción                                      |
| --------- | -------- | ------------------------------------------------ |
| id        | `Number` | Un identificador numérico de la cara localizada. |
