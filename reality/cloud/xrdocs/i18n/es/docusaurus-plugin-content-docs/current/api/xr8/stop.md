# XR8.stop()

`XR8.stop()`

## Descripción {#description}

Mientras está parado, la alimentación de la cámara está cerrada y no se rastrea el movimiento del dispositivo. Debes llamar a [`XR8.run()`](run.md) para volver a arrancar después de parar el motor.

## Parámetros {#parameters}

Ninguno

## Vuelta {#returns}

Ninguno

## Ejemplo {#example}

```javascript
XR8.stop()
```
