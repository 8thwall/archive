# XR8.XrController.recenter()

`XR8.XrController.recenter()`

## Descripción {#description}

Reposiciona la cámara a la dirección de origen / orientación especificada en [`XR8.XrController.updateCameraProjectionMatrix()`](updatecameraprojectionmatrix.md) y reinicia el seguimiento.

Ninguno
