# XR8.Threejs

## Descripción {#description}

Proporciona un módulo de canalización de cámara que controla la cámara three.js para realizar superposiciones virtuales.

## Funciones {#functions}

| Función                             | Descripción                                                                                                                                     |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| [pipelineModule](pipelinemodule.md) | Un módulo de canalización que interactúa con el entorno y el ciclo de vida de three.js.                                                         |
| [configure](configure.md)           | Configura el renderizador three.js.                                                                                                             |
| [xrScene](xrscene.md)               | Obtiene un manejador de la escena xr, la cámara, el renderizador, la textura de alimentación de la cámara (opcional) y  layerScenes (opcional). |
