# XR8.MediaRecorder.pipelineModule()

`XR8.MediaRecorder.pipelineModule()`

## Descripción {#description}

Proporciona un módulo de canalización de cámara que te permite grabar un vídeo en formato MP4.

## Parámetros {#parameters}

Ninguno

## Devuelve {#returns}

Un módulo de canalización de `MediaRecorder` que permite grabar un vídeo.

## Ejemplo {#example}

```javascript
XR8.addCameraPipelineModule(XR8.MediaRecorder.pipelineModule())
```
