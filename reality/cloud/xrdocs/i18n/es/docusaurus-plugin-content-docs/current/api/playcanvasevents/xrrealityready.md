# xr:realityready

## Descripción {#description}

Este evento se dispara cuando 8th Wall Web se ha inicializado y se ha procesado correctamente al menos un fotograma.

## Ejemplo {#example}

```javascript
this.app.on('xr:realityready', () => {
  // Ocultar la carga de la interfaz de usuario
}, this)
```
