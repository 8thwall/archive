# xrrighteyebrowlowered

## Descripción {#description}

Este evento es emitido por [`xrface`](/api/aframe/#face-effects) cuando la ceja derecha de una cara rastreada se baja a su posición inicial cuando se encontró la cara

`xrrighteyebrowlowered.detail : {id}`

| Propiedad | Tipo     | Descripción                                      |
| --------- | -------- | ------------------------------------------------ |
| id        | `Number` | Un identificador numérico de la cara localizada. |
