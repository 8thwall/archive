# SXR8.XrConfig.cámara()

Enumeración

## Descripción {#description}

Cámara que desea utilizar.

## Propiedades {#properties}

| Propiedad | Valor      | Descripción                         |
| --------- | ---------- | ----------------------------------- |
| FRENTE    | `'frente'` | Utiliza la cámara frontal / selfie. |
| ATRÁS     | `'atrás'`  | Utiliza la cámara trasera.          |
