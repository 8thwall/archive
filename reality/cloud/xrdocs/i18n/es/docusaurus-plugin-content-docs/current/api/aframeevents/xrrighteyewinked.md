# xrrighteyewinked

## Descripción {#description}

Este evento lo emite [`xrface`](/api/aframe/#face-effects) cuando el ojo derecho de una cara rastreada se cierra y se abre en un plazo de 750 ms, mientras que el ojo izquierdo permanece abierto.

`xrrighteyewinked.detail : {id}`

| Propiedad | Tipo     | Descripción                                      |
| --------- | -------- | ------------------------------------------------ |
| id        | `Number` | Un identificador numérico de la cara localizada. |
