# Fijar objetivos {#pinning-targets}
Si importa un módulo desde su propio espacio de trabajo, verá varias opciones de destino de fijación:

**Versión**: Si selecciona un objetivo de fijación "Versión", puede suscribir su módulo importado a actualizaciones de corrección de errores, actualizaciones menores o desactivar las actualizaciones automáticas del módulo. Solo verás esta opción si el módulo se ha desplegado al menos una vez.

**Commit**: te permite anclar el módulo a cualquier commit del código del módulo.

**Desarrollo**: te permite utilizar el código del módulo que tiene en su cliente local y que aún no ha sido aterrizado. Se utiliza para desarrollo y pruebas.
