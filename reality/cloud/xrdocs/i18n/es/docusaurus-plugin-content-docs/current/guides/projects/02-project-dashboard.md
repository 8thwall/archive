# Panel de control del proyecto

El panel de control del proyecto es su centro para gestionar los proyectos de 8th Wall.  Desde la página del panel de control del proyecto puede gestionar la configuración del proyecto, acceder al editor de la nube de la 8th Wall, comprar licencias comerciales, gestionar objetivos de imagen, configurar dominios personalizados y mucho más.

La URL directa a su panel de control del proyecto tiene el formato: www.8thwall.com/workspacename/projectname

## Gestionar objetivos de imagen {#managing-image-targets}

Para gestionar los objetivos de imagen de un proyecto determinado, haga clic en el icono Objetivo de imagen de la barra del navegador de la izquierda , o en el enlace "Gestionar objetivos de Imagen" del panel de ccntrol del proyecto.

![ManageImageTargets](/images/console-appkey-imagetargets.jpg)

Para obtener información detallada sobre los objetivos de omagen, consulte la [documentación sobre objetivos de imagen](/guides/image-targets).

## Gestión de las ubicaciones VPS de Lightship {#managing-lightship-vps-locations}

Para gestionar las ubicaciones VPS de Lightship para un proyecto determinado, haga clic en el icono del mapa en la navegación de la izquierda.

![ManageWayspots](/images/console-wayspot-left-nav.jpg)

Para obtener información detallada sobre Lightship VPS para Web, consulte la [documentación deLightship VPS para Web](/guides/lightship-vps).
