# Áreas de trabajo

Un área de trabajo es una agrupación lógica de proyectos, usuarios y aplicaciones. Espacios de trabajo
puede contener uno o más Usuarios, cada uno con diferentes permisos. Los usuarios pueden pertenecer
a varios espacios de trabajo.

El panel de control del area de trabajo le permite:

- Crear nuevos proyectos.
- Gestionar los proyectos existentes.
- Gestionar los miembros del equipo del área de trabajo y los permisos.
- Gestionar suscripciones y licencias comerciales.

## Área de trabajo inicial {#initial-workspace}

When creating a new 8th Wall account directly from 8thwall.com, you will start
with a workspace on the Basic (free) plan.

Si te registras a través de una invitación de otro usuario de 8th Wall, se te añadirá como
un miembro del equipo de su espacio de trabajo existente.

**Importante: Elija cuidadosamente el nombre de su espacio de trabajo, ya que no se puede cambiar más tarde.** Por defecto, la URL
de su experiencia Web AR publicada estará en el fomato de:

"workspace-name.8thwall.app/project-name"

Para cambiar la URL del área de trabajo es necesario ponerse en contacto con el servicio de asistencia y crear una solicitud.
