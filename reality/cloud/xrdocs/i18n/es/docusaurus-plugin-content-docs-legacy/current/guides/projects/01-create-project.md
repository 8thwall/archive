# Crear un proyecto de editor heredado

1. Acceda a su cuenta de 8th Wall y haga clic en "Ir a mis proyectos":

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Amplíe su espacio de trabajo, si aún no está abierto.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Desplácese hasta la parte inferior de la página y haga clic en **Construir con el editor heredado**.

![BuildWithLegacyEditor](/images/legacy/build-with-legacy-editor.jpg)

4. Introduzca la URL del proyecto y haga clic en **Crear**.

![CreateLegacyProject](/images/legacy/legacy-create-project.jpg)

:::info
Una **App Key** se utiliza para desarrollar localmente y para proyectos autoalojados. Esta opción sólo está disponible en un [Plan personalizado](https://8thwall.com/custom).
:::
