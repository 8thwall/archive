# Cuadro de mando del proyecto

El panel de control del proyecto es el centro de gestión de los proyectos de 8th Wall.  Desde la página del panel de control del proyecto
puede gestionar la configuración del proyecto, acceder al editor de 8th Wall Cloud, adquirir licencias comerciales, gestionar objetivos de imagen
, configurar dominios personalizados, etc.

La URL directa a su Cuadro de mandos del proyecto tiene el formato: www.8thwall.com/workspacename/projectname

## Gestión de objetivos de imagen {#managing-image-targets}

Para gestionar los objetivos de imagen de un proyecto determinado, haga clic en el icono Objetivo de imagen de la barra de navegación de la izquierda
o en el enlace "Gestionar objetivos de imagen" del panel de control del proyecto.

![ManageImageTargets](/images/console-appkey-imagetargets.jpg)

Para obtener información detallada sobre los objetivos de imagen, consulte
[Image Target documentation](/legacy/guides/image-targets).

## Gestión de las ubicaciones VPS de Lightship {#managing-lightship-vps-locations}

Para gestionar las ubicaciones VPS de Lightship para un proyecto determinado, haga clic en el icono del mapa en la navegación de la izquierda.

![ManageWayspots](/images/console-wayspot-left-nav.jpg)

Para información detallada sobre Lightship VPS para Web, por favor refiérase a
[Documentación de Lightship VPS para Web](/legacy/guides/lightship-vps).
