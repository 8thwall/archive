# Fijación de objetivos {#pinning-targets}

Si importa un módulo desde su propio espacio de trabajo, verá varias opciones de destino de fijación:

**Versión**:
Si selecciona un objetivo de anclaje "Versión", puede suscribir su módulo importado a actualizaciones de corrección de errores, actualizaciones menores o desactivar las actualizaciones automáticas del módulo. Sólo verá esta opción si el módulo se ha desplegado al menos una vez.

**Confirmación**:
Permite anclar el módulo a cualquier commit del código del módulo.

**Desarrollo**:
Te permite utilizar código de módulo que tengas en tu cliente local y que aún no haya sido aterrizado. Se utiliza para el desarrollo y las pruebas.
