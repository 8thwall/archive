# XR8.stop()

`XR8.stop()`

## Descripción {#description}

Mientras está detenido, la alimentación de la cámara está cerrada y no se rastrea el movimiento del dispositivo. Debe llamar a
[`XR8.run()`](run.md) para volver a arrancar después de parar el motor.

## Parámetros {#parameters}

Ninguno

## Devuelve {#returns}

Ninguno

## Ejemplo {#example}

```javascript
XR8.stop()
```
