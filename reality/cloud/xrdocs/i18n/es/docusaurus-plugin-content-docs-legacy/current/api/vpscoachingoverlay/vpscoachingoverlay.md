# VpsCoachingOverlay

## Descripción {#description}

Proporciona un módulo que genera un Coaching Overlay para su experiencia Web AR habilitada con Lightship VPS.

Para obtener más información sobre la superposición de revestimiento de escala absoluta, consulte [aquí](/legacy/api/coachingoverlay).

## Funciones {#functions}

| Función                                                 | Descripción                                                                                                                                         |
| ------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](vps-coachingoverlay-configure.md)           | Configura los ajustes de Coaching Overlay.                                                                                          |
| [pipelineModule](vps-coachingoverlay-pipelinemodule.md) | Crea un módulo de canalización de cámara que, cuando se instala, añade la funcionalidad de superposición de coaching a su proyecto. |
