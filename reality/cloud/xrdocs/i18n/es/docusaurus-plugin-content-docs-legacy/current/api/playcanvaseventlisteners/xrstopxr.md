# xr:detenerxr

`this.app.fire('xr:stopxr')`

## Descripción {#description}

Detener la sesión XR actual.  Mientras está detenido, la alimentación de la cámara se detiene y el movimiento del dispositivo no se rastrea.

## Parámetros {#parameters}

Ninguno

## Ejemplo {#example}

```javascript
this.app.fire('xr:stopxr')
```
