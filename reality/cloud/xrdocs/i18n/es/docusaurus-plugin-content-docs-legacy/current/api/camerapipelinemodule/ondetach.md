# onDetach()

`onDetach: ({framework})`

## Descripción {#description}

Se llama a `onDetach` después de la última vez que un módulo recibe actualizaciones de trama. Esto ocurre después de que el motor se detiene o el módulo se retira manualmente de la tubería, lo que ocurra primero.

## Parámetros {#parameters}

| Parámetro | Descripción                                                          |
| --------- | -------------------------------------------------------------------- |
| marco     | Los enlaces de este módulo para el envío de eventos. |
