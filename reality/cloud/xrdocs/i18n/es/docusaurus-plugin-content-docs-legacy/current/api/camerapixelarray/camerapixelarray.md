# XR8.CameraPixelArray

## Descripción {#description}

Proporciona un módulo de canalización de cámara que da acceso a los datos de la cámara como una matriz uint8 en escala de grises o color.

## Funciones {#functions}

| Función                             | Descripción                                                                                                                                                                                                           |
| ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [pipelineModule](pipelinemodule.md) | Un módulo de canalización que proporciona la textura de la cámara como una matriz de valores de píxeles RGBA o en escala de grises que se puede utilizar para el procesamiento de imágenes de la CPU. |
