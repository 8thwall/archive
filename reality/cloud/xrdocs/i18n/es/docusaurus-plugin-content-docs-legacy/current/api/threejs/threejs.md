# XR8.Threejs

## Descripción {#description}

Proporciona un módulo de canalización de cámara que controla la cámara three.js para realizar superposiciones virtuales.

## Funciones {#functions}

| Función                             | Descripción                                                                                                                                                                              |
| ----------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [pipelineModule](pipelinemodule.md) | Un módulo de canalización que interactúa con el entorno y el estilo de vida de three.js.                                                                 |
| [configure](configure.md)           | Configura el renderizador de three.js.                                                                                                                   |
| [xrScene](xrscene.md)               | Obtiene un manejador a la escena xr, cámara, renderizador, (opcional) textura de alimentación de cámara, y (opcional) layerScenes. |
