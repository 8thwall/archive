# XR8.XrController.recenter()

`XR8.XrController.recenter()`

## Descripción {#description}

Reposiciona la cámara al origen / dirección de orientación especificada por [`XR8.XrController.updateCameraProjectionMatrix()`](updatecameraprojectionmatrix.md) y reinicia el seguimiento.

## Parámetros {#parameters}

Ninguno

## Devuelve {#returns}

Ninguno
