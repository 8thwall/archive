# XR8.LayersController.getLayerNames()

`XR8.LayersController.getLayerNames()`

## Descripción {#description}

Devuelve las capas configuradas por el `LayersController`.

## Parámetros {#parameters}

Ninguno

## Devuelve {#returns}

Una lista de cadenas donde cada una es un nombre de capa.
