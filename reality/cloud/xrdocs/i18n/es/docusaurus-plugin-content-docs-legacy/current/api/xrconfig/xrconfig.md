## XR8.XrConfig {#xr8xrconfig}

## Descripción {#description}

Utilidades para especificar la clase de dispositivos y cámaras en los que deben ejecutarse los módulos de canalización.

## Propiedades {#properties}

| Propiedad                                     | Tipo | Descripción                                                                                      |
| --------------------------------------------- | ---- | ------------------------------------------------------------------------------------------------ |
| [camera()](camera.md)      | Enum | Cámara deseada.                                                                  |
| [dispositivo()](device.md) | Enum | Especifique la clase de dispositivos en los que debe ejecutarse la canalización. |
