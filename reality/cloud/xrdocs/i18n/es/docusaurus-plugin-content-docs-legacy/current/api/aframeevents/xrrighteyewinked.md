# xrrighteyewinked

## Descripción {#description}

Este evento es emitido por [`xrface`](/legacy/api/aframe/#face-effects) cuando el ojo derecho de una cara rastreada se cierra y se abre en 750ms mientras el ojo izquierdo permanece abierto.

`xrrighteyewinked.detail : {id}`

| Propiedad | Tipo     | Descripción                                        |
| --------- | -------- | -------------------------------------------------- |
| id        | `Número` | Id numérico de la cara localizada. |
