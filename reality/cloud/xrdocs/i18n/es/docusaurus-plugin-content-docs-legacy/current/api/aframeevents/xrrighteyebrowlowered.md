# xrrighteyebrowlowered

## Descripción {#description}

Este evento es emitido por [`xrface`](/legacy/api/aframe/#face-effects) cuando la ceja derecha de una cara rastreada es bajada a su posición inicial cuando la cara fue encontrada

`xrrighteyebrowlowered.detail : {id}`

| Propiedad | Tipo     | Descripción                                        |
| --------- | -------- | -------------------------------------------------- |
| id        | `Número` | Id numérico de la cara localizada. |
