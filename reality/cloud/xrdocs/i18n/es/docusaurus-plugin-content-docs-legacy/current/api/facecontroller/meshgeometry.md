# XR8.FaceController.MeshGeometry

Enumeración

## Descripción {#description}

Opciones para definir qué partes de la cara tienen triángulos de malla devueltos.

## Propiedades {#properties}

| Propiedad | Valor    | Descripción                                           |
| --------- | -------- | ----------------------------------------------------- |
| CARA      | `'cara'` | Devuelve la geometría de la cara.     |
| BOCA      | `'boca'` | Geometría de retorno para la boca.    |
| OJOS      | `'ojos'` | Geometría de retorno para los ojos.   |
| IRIS      | `'iris'` | Geometría de retorno para los lirios. |
