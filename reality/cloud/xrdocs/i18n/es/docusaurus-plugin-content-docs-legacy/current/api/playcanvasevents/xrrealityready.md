# xr:realityready

## Descripción {#description}

Este evento se dispara cuando 8th Wall Web se ha inicializado y al menos un fotograma ha sido procesado con éxito.

## Ejemplo {#example}

```javascript
this.app.on('xr:realityready', () => {
  // Ocultar la carga de la interfaz de usuario
}, this)
```
