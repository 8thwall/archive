# xrblinked

## Descripción {#description}

Este evento es emitido por [`xrface`](/legacy/api/aframe/#face-effects) cuando los ojos de una cara rastreada parpadean.

`xrblinked.detail : {id}`

| Propiedad | Tipo     | Descripción                                        |
| --------- | -------- | -------------------------------------------------- |
| id        | `Número` | Id numérico de la cara localizada. |
