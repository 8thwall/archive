# XR8.XrConfig.cámara()

Enumeración

## Descripción {#description}

Cámara deseada.

## Propiedades {#properties}

| Propiedad | Valor      | Descripción                                         |
| --------- | ---------- | --------------------------------------------------- |
| FRENTE    | `'frente'` | Utiliza la cámara frontal / selfie. |
| VOLVER    | `'atrás'`  | Utiliza la cámara trasera.          |
