# xrrighteyebrowraised

## Descripción {#description}

Este evento es emitido por [`xrface`](/legacy/api/aframe/#face-effects) cuando la ceja derecha de una cara rastreada se levanta de su posición cuando la cara fue encontrada.

`xrrighteyebrowraised.detail : {id}`

| Propiedad | Tipo     | Descripción                                        |
| --------- | -------- | -------------------------------------------------- |
| id        | `Número` | Id numérico de la cara localizada. |
