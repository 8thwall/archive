# Página de aterrizaje

## Descripción {#description}

Proporciona un módulo genera una página de destino personalizada para su experiencia Web AR.

## Funciones {#functions}

| Función                             | Descripción                                                                                                                              |
| ----------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Configura los ajustes de LandingPage.                                                                                    |
| [pipelineModule](pipelinemodule.md) | Crea un módulo de canalización de cámara que, cuando se instala, añade funcionalidad de página de destino a su proyecto. |
