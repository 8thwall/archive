# xrinterpupilardistancia

## Descripción {#description}

Este evento es emitido por [`xrface`](/legacy/api/aframe/#face-effects) cuando se detecta por primera vez la distancia en milímetros entre los centros de cada pupila de una cara rastreada.

`xrinterpupillarydistance.detail : {id}`

| Propiedad              | Tipo     | Descripción                                                                          |
| ---------------------- | -------- | ------------------------------------------------------------------------------------ |
| id                     | `Número` | Id numérico de la cara localizada.                                   |
| distancia interpupilar | `Número` | Distancia aproximada en milímetros entre los centros de cada pupila. |
