# AccessPass

## Descripción {#description}

Proporciona un módulo para monetizar su experiencia Web AR y Web VR. Esta opción sólo está disponible para los proyectos alojados en el 8º muro de
y requiere el [Módulo de pagos](/legacy/guides/advanced-topics/monetize-with-8th-wall-payments).

```ts
import {AccessPass} from 'pagos'
```
