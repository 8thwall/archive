# XR8.Babylonjs

Babylon.js (<https://www.babylonjs.com/>) es un completo framework JavaScript para crear juegos 3D y experiencias
con HTML5 y WebGL.  Combinado con 8th Wall Web, puede crear potentes experiencias Web AR
.

Vídeo tutorial:

- Integración de Babylon.js y 8th Wall Web: <https://youtu.be/RhZaebNpaEY>

## Descripción {#description}

Proporciona una integración que interactúa con el entorno BabylonJS y el estilo de vida para manejar la cámara Babylon.js para hacer superposiciones virtuales.

## Funciones {#functions}

| Función                                     | Descripción                                                                                                                                       |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| [xrCameraBehavior](xrcamerabehavior.md)     | Obtenga un comportamiento que se puede adjuntar a una cámara Babylon para ejecutar Seguimiento del Mundo y/o Objetivos de Imagen. |
| [faceCameraBehavior](facecamerabehavior.md) | Consigue un comportamiento que se pueda conectar a una cámara Babylon para ejecutar Face Effects.                                 |
