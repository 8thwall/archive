# CoachingOverlay

## Descripción {#description}

Proporciona un módulo que genera una Superposición de Coaching para su experiencia Absolute Scale Web AR.

Para obtener información sobre la Superposición de Entrenamiento VPS de Lightship, consulte [aquí](/legacy/api/vpscoachingoverlay).

## Funciones {#functions}

| Función                             | Descripción                                                                                                                                         |
| ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Configura los ajustes de Coaching Overlay.                                                                                          |
| [pipelineModule](pipelinemodule.md) | Crea un módulo de canalización de cámara que, cuando se instala, añade la funcionalidad de superposición de coaching a su proyecto. |
