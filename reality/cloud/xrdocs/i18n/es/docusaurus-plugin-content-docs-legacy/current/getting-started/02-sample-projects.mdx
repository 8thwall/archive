# Ejemplos de proyectos

8th Wall ha creado una serie de proyectos de muestra que puede clonar y utilizar como puntos de partida para
ayudarle a empezar. 

## Biblioteca del proyecto 8th Wall {#8th-wall-project-library}

Empieza con proyectos de muestra. Están diseñados para ayudarle con los fundamentos del desarrollo de la 8ª Pared. Para cada proyecto puedes ver el código, probarlo en WebAR y clonarlo en tu espacio de trabajo. [Ir a: Biblioteca de proyectos](https://www.8thwall.com/projects)
