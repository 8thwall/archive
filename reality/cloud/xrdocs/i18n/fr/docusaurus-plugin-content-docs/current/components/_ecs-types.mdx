Les types de données suivants sont utiles pour créer des propriétés de schéma sur un composant personnalisé ou des références à un type spécifique.

| Type        | Description                           |
| ----------- | ------------------------------------- |
| ecs.eid     | Référence de l'entité                 |
| ecs.f32     | Nombre à virgule flottante de 32 bits |
| ecs.f64     | Nombre à virgule flottante de 64 bits |
| ecs.i32     | Entier de 32 bits                     |
| ecs.ui8     | Entier non signé de 8 bits            |
| ecs.ui32    | Entier non signé de 32 bits           |
| ecs.string  | Chaîne                                |
| ecs.boolean | Booléen                               |
