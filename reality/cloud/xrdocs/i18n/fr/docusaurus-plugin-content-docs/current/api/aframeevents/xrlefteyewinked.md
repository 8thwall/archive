# xrlefteyewinked

## Description {#description}

Cet événement est émis par [`xrface`](/api/aframe/#face-effects) lorsque l'œil gauche d'un visage suivi se ferme et s'ouvre dans un délai de 750 ms alors que l'œil droit reste ouvert.

`xrlefteyewinked.detail : {id}`

| Propriété | Type     | Description                                  |
| --------- | -------- | -------------------------------------------- |
| id        | `Nombre` | Un identifiant numérique du visage localisé. |
