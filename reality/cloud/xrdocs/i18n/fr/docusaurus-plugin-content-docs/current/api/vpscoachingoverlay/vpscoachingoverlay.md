# VpsCoachingOverlay

## Description {#description}

Fournit un module qui génère une superposition de coaching pour votre expérience WebAR activée par Lightship VPS.

Pour plus d'informations sur la superposition de l'échelle absolue de l'encadrement, veuillez consulter [ici](/api/coachingoverlay).

## Fonctions {#functions}

| Fonction                                                 | Description                                                                                                                           |
| -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| [configurer](vps-coachingoverlay-configure.md)           | Configure les paramètres de superposition de la couverture.                                                                           |
| [module pipeline](vps-coachingoverlay-pipelinemodule.md) | Crée un module de pipeline de caméra qui, une fois installé, ajoute la fonctionnalité de superposition d'entraînement à votre projet. |
