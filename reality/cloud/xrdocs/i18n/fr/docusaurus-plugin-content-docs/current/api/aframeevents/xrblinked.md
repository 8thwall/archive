# xrblinked

## Description {#description}

Cet événement est émis par [`xrface`](/api/aframe/#face-effects) lorsque les yeux d'un visage suivi clignotent.

`xrblinked.detail : {id}`

| Propriété | Type     | Description                                  |
| --------- | -------- | -------------------------------------------- |
| id        | `Nombre` | Un identifiant numérique du visage localisé. |
