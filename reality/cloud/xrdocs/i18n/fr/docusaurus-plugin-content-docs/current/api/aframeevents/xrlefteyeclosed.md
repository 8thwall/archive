# xrlefteyeclosed

## Description {#description}

Cet événement est émis par [`xrface`](/api/aframe/#face-effects) lorsque l'œil gauche d'un visage suivi se ferme.

`xrlefteyeclosed.detail : {id}`

| Propriété | Type     | Description                                  |
| --------- | -------- | -------------------------------------------- |
| id        | `Nombre` | Un identifiant numérique du visage localisé. |
