# XR8.FaceController.MeshGeometry

Enumération

## Description {#description}

Options permettant de définir les parties du visage dont les triangles de maillage sont renvoyés.

## Propriétés {#properties}

| Propriété | Valeur     | Description                         |
| --------- | ---------- | ----------------------------------- |
| FACE      | `visage`   | Renvoi de la géométrie du visage.   |
| BOUCHE    | `bouche`   | Géométrie de retour pour la bouche. |
| YEUX      | `les yeux` | Géométrie de retour pour les yeux.  |
| IRIS      | `iris`     | Géométrie de retour pour les iris.  |
