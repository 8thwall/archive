# XR8.XrController.recenter()

`XR8.XrController.recenter()`

## Description {#description}

Repositionne la caméra sur l'origine / la direction d'orientation spécifiée par [`XR8.XrController.updateCameraProjectionMatrix()`](updatecameraprojectionmatrix.md) et redémarre le suivi.

## Paramètres {#parameters}

Aucun

## Retours {#returns}

Aucun
