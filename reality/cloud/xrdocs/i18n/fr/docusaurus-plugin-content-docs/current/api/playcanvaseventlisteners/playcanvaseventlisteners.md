# Audience d'événements PlayCanvas

Cette section décrit les événements écoutés par 8th Wall Web dans un environnement PlayCanvas.

Vous pouvez déclencher ces événements dans votre application web pour effectuer diverses actions :

| Écouteur d'événements                                   | Description                                                                                                                                                                                                                                                                                                      |
| ------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [xr:hidecamerafeed](xrhidecamerafeed.md)                | Masque le flux de la caméra. Le suivi ne s'arrête pas.                                                                                                                                                                                                                                                           |
| [xr:recenter](xrrecenter.md)                            | Recentre le flux de la caméra sur son origine. Si un nouveau point d'origine est fourni en argument, le point d'origine de la caméra sera réinitialisé à ce point, puis la caméra se recentrera.                                                                                                                 |
| [xr:demande de capture d'écran](xrscreenshotrequest.md) | Émet une requête au moteur pour capturer une capture d'écran du support PlayCanvas. Le moteur émettra un événement [`xr:screenshotready`](/api/playcanvasevents/xrscreenshotready) avec l'image compressée JPEG ou [`xr:screenshoterror`](/api/playcanvasevents/xrscreenshoterror) si une erreur s'est produite. |
| [xr:showcamerafeed](xrshowcamerafeed.md)                | Affiche le flux de la caméra.                                                                                                                                                                                                                                                                                    |
| [xr:stopxr](xrstopxr.md)                                | Arrêtez la session XR en cours. Lorsque l'appareil est arrêté, le flux de la caméra est interrompu et les mouvements de l'appareil ne sont pas suivis.                                                                                                                                                           |
