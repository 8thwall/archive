# XR8.initialize()

`XR8.initialize()`

## Description {#description}

Renvoie une promesse qui est remplie lorsque le WebAssembly de l'AR Engine est initialisé.

## Paramètres {#parameters}

Aucun

## Retours {#returns}

Aucun

## Exemple {#example}

```javascript
XR8.initialize().then(() => console.log(XR8.version())
```
