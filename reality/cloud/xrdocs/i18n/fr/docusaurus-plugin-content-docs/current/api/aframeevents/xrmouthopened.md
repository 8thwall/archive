# xrmouthopened

## Description {#description}

Cet événement est émis par [`xrface`](/api/aframe/#face-effects) lorsque la bouche d'un visage suivi s'ouvre.

`xrmouthopened.detail : {id}`

| Propriété | Type     | Description                                  |
| --------- | -------- | -------------------------------------------- |
| id        | `Nombre` | Un identifiant numérique du visage localisé. |
