# xrrighteyebrowraised

## Description {#description}

Cet événement est émis par [`xrface`](/api/aframe/#face-effects) lorsque le sourcil droit d'un visage suivi est relevé par rapport à sa position au moment où le visage a été trouvé.

`xrrighteyebrowraised.detail : {id}`

| Propriété | Type     | Description                                  |
| --------- | -------- | -------------------------------------------- |
| id        | `Nombre` | Un identifiant numérique du visage localisé. |
