# XR8.LayersController

## Description {#description}

`LayersController` fournit une détection sémantique des couches et des interfaces pour configurer le rendu des couches.

## Fonctions {#functions}

| Fonction                             | Description                                                                                               |
| ------------------------------------ | --------------------------------------------------------------------------------------------------------- |
| [configurer](configure.md)           | Configure le traitement effectué par `LayersController`.                                                  |
| [getLayerNames](getlayernames.md)    | Renvoie les couches configurées par `LayersController`.                                                   |
| [module pipeline](pipelinemodule.md) | Crée un module de pipeline de caméra qui, une fois installé, assure la détection des couches sémantiques. |
| [recentrer](recenter.md)             | Repositionne l'appareil photo dans la direction de l'origine ou de l'orientation.                         |
