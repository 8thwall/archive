# xrrighteyebrowlowered

## Description {#description}

Cet événement est émis par [`xrface`](/api/aframe/#face-effects) lorsque le sourcil droit d'un visage suivi est abaissé à sa position initiale lorsque le visage a été trouvé

`xrrighteyebrowlowered.detail : {id}`

| Propriété | Type     | Description                                  |
| --------- | -------- | -------------------------------------------- |
| id        | `Nombre` | Un identifiant numérique du visage localisé. |
