# XR8.version()

`XR8.version()`

## Description {#description}

Obtenez la version du moteur 8th Wall Web.

## Paramètres {#parameters}

Aucun

## Retours {#returns}

Une chaîne indiquant la version du moteur Web 8th Wall.

## Exemple {#example}

```javascript
console.log(XR8.version())
```
