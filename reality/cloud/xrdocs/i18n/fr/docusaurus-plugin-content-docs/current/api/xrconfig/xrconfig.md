## XR8.XrConfig {#xr8xrconfig}

## Description {#description}

Utilitaires permettant de spécifier les classes d'appareils et de caméras sur lesquels les modules de pipeline doivent être exécutés.

## Propriétés {#properties}

| Propriété                 | Type | Description                                                                |
| ------------------------- | ---- | -------------------------------------------------------------------------- |
| [caméra()](camera.md)     | Enum | Appareil photo souhaité.                                                   |
| [dispositif()](device.md) | Enum | Spécifiez la classe d'appareils sur lesquels le pipeline doit fonctionner. |
