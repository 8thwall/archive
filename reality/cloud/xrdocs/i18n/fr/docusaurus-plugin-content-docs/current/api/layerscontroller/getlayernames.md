# XR8.LayersController.getLayerNames()

`XR8.LayersController.getLayerNames()`

## Description {#description}

Renvoie les couches configurées par `LayersController`.

## Paramètres {#parameters}

Aucun

## Retours {#returns}

Une liste de chaînes de caractères où chacune est un nom de couche.
