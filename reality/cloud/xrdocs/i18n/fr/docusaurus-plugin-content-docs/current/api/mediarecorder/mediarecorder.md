# XR8.MediaRecorder

## Description {#description}

Fournit un module de pipeline de caméra qui vous permet d'enregistrer une vidéo au format MP4.

## Fonctions {#functions}

| Fonction                                   | Description                                                                                                          |
| ------------------------------------------ | -------------------------------------------------------------------------------------------------------------------- |
| [configurer](configure.md)                 | Configurez les paramètres d'enregistrement vidéo.                                                                    |
| [module pipeline](pipelinemodule.md)       | Crée un module de pipeline qui enregistre des vidéos au format MP4.                                                  |
| [enregistrerVidéo](recordvideo.md)         | Démarrer l'enregistrement.                                                                                           |
| [demanderMicrophone](requestmicrophone.md) | Active l'enregistrement audio (s'il n'est pas activé automatiquement), en demandant des autorisations si nécessaire. |
| [stopRecording](stoprecording.md)          | Arrêter l'enregistrement.                                                                                            |
| [RequestMicOptions](requestmicoptions.md)  | Enum indiquant s'il faut ou non demander automatiquement les autorisations pour le microphone.                       |
