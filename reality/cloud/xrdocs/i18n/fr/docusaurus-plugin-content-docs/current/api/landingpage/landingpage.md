# LandingPage

## Description {#description}

Fournit un module qui génère une page d'atterrissage personnalisée pour votre expérience WebAR.

## Fonctions {#functions}

| Fonction                             | Description                                                                                                                   |
| ------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------- |
| [configurer](configure.md)           | Configure les paramètres de la page d'atterrissage.                                                                           |
| [module pipeline](pipelinemodule.md) | Crée un module de pipeline de caméra qui, une fois installé, ajoute une fonctionnalité de page d'atterrissage à votre projet. |
