# Épingler des cibles {#pinning-targets}
Si vous importez un module à partir de votre propre espace de travail, vous verrez plusieurs options de cibles d’épinglage :

**Version**: Si vous sélectionnez une cible d'épinglage "Version", vous pouvez abonner votre module importé à des mises à jour de corrections de bugs, à des mises à jour mineures ou désactiver les mises à jour automatiques du module. Vous ne verrez cette option que si le module a été déployé au moins une fois.

**Commit**: Permet de rattacher le module à n'importe quel commit du code du module.

**Développement** : Vous permet d'utiliser le code du module que vous avez dans votre client local et qui n'a pas encore été débarqué. Utilisé pour le développement et les tests.
