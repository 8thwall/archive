# Créer un projet

1. À partir de la page d'accueil (connecté) ou du tableau de bord de l'espace de travail, cliquez sur "Démarrer un nouveau projet"

2. Sélectionnez l'espace de travail pour ce projet.

3. Saisissez les informations de base du projet : Veuillez fournir : Titre, URL, Description (facultatif) et Image de couverture (facultatif).  Tous ces champs, **sauf URL**, peuvent être modifiés ultérieurement dans la page Paramètres du projet.

4. Sélectionnez un type de projet

## Types de projets {#project-types}

**Tous les projets** doivent afficher le badge [Powered by 8th Wall](https://drive.google.com/drive/folders/1c9d23c5hS_HspHTUD7VceV6ocqdbPN7J?usp=sharing) sur la page de chargement. </code> Il est inclus par défaut dans le module de chargement et ne peut pas être supprimé.
Veuillez consulter <a href="/guides/advanced-topics/load-screen/">ici</a> pour obtenir des instructions sur la personnalisation de l'écran de chargement.</p>

<ul>
<li><p spaces-before="0"><strong x-id="1">Commercial</strong>: Les projets commerciaux sont destinés à un usage commercial. Lorsque vous êtes prêt à lancer un projet commercial
afin que le monde entier puisse le voir, vous devez acheter une licence commerciale
. Veuillez consulter la <a href="https://www.8thwall.com/pricing#commercial-license">page des prix</a> sur
pour connaître les options de licence commerciale. NOTE : Pour acheter une licence commerciale, l'espace de travail
doit d'abord avoir un plan Pro actif.</p></li>
<li><p spaces-before="0"><strong x-id="1">Non commercial</strong>: Votre abonnement payant vous permet de développer et de publier un nombre illimité de
projets non commerciaux. Une indication "Non-Commercial" apparaîtra sur l'écran de chargement. Si vous décidez
de commercialiser votre projet à un moment donné, passez à "Commercial" dans le tableau de bord du projet.</p></li>
<li><p spaces-before="0"><strong x-id="1">Utilisation démo</strong>: Vous pouvez créer un nombre illimité de projets démo qui sont visibles par le public et strictement
destinés à la présentation d'un travail potentiel. Une indication « Utilisation en démonstration seulement" apparaîtra sur l'écran de chargement.
Si vous décidez de commercialiser votre projet à un moment donné, passez à "Commercial" dans le tableau
de bord du projet.</p></li>
<li><p spaces-before="0"><strong x-id="1">Utilisation à des fins éducatives</strong>: Les projets éducatifs sont destinés à des fins éducatives uniquement, par exemple dans le cadre
d'une salle de classe. Vous pouvez développer et publier un nombre illimité de projets éducatifs. Une étiquette « Utilisation à des fins
éducatives » apparaît sur l'écran de chargement. Si votre projet est destiné à un usage commercial, vous
devez sélectionner "Commercial". Si vous êtes un établissement d'enseignement, veuillez contacter 8th Wall pour
obtenir des informations sur un plan d'éducation personnalisé.</p></li>
</ul>

<h2 id="change-project-type" spaces-before="0">Modifier le type de projet</h2>

<ol start="1">
<li>Accédez à la page du tableau de bord du projet</li>
<li>Cliquez sur "Modifier le type de projet"</li>
<li>Sélectionnez un type de projet approprié</li>
<li>Cliquez sur "Enregistrer"</li>
</ol>
