# Tableau de bord du projet

Le tableau de bord des projets est votre centre de gestion des projets 8th Wall.  À partir de la page du tableau de bord du projet , vous pouvez gérer les paramètres du projet, accéder au Cloud Editor 8th Wall, acheter des licences commerciales, gérer les images cible, configurer des domaines personnalisés, et bien plus encore.

L'URL directe de votre tableau de bord de projet se présente sous la forme suivante : www.8thwall.com/workspacename/projectname

## Gestion des images cible {#managing-image-targets}

Pour gérer les images cible d'un projet donné, cliquez sur l'icône Cible d'image dans le menu de gauche ou sur le lien "Gérer les images cible" dans le tableau de bord du projet.

![ManageImageTargets](/images/console-appkey-imagetargets.jpg)

Pour des informations détaillées sur les images cible, veuillez vous référer à la [documentation sur les images cible](/guides/image-targets).

## Gestion des emplacements des VPS de Lightship {#managing-lightship-vps-locations}

Pour gérer les emplacements des VPS pour un projet donné, cliquez sur l'icône de la carte dans la navigation de gauche.

![Gérer les points de passage](/images/console-wayspot-left-nav.jpg)

Pour des informations détaillées sur Lightship VPS for Web, veuillez vous référer à la documentation [Lightship VPS pour le Web](/guides/lightship-vps).
