# Exemples de projets

8th Wall a créé un certain nombre de projets types que vous pouvez cloner et utiliser comme points de départ pour
vous aider à démarrer. 

## Bibliothèque de projet de 8th Wall {#8th-wall-project-library}

Commencez avec des exemples de projets. Ils sont conçus pour vous aider à maîtriser les principes fondamentaux du développement du 8e mur. Pour chaque projet, vous pouvez visualiser le code, l'essayer dans WebAR et le cloner dans votre espace de travail. [Aller à : Bibliothèque de projets](https://www.8thwall.com/projects)
