# Aperçu de Lightship VPS {#lightship-vps-overview}

:::info
**La localisation VPS n'est pas encore disponible dans le [Simulateur]
(/legacy/getting-started/quick-start-guide/#simulator).**
:::

Grâce au système de positionnement visuel (VPS) de Niantic, les développeurs peuvent créer de riches expériences de localisation AR
qui superposent du contenu virtuel au monde physique. Le contenu WebAR peut être ancré dans le monde réel (
), ce qui permet aux objets virtuels d'interagir avec leur environnement physique. L'expérience de la RA sur le site
est ainsi plus personnelle, plus significative et plus immersive, et donne aux utilisateurs de nouvelles raisons d'explorer
le monde qui les entoure.
