# Espaces de travail

Un espace de travail est un regroupement logique de projets, d'utilisateurs et de facturation. Espaces de travail
peut contenir un ou plusieurs utilisateurs, chacun avec des autorisations différentes. Les utilisateurs peuvent appartenir
à plusieurs espaces de travail.

Le tableau de bord de l'espace de travail vous permet de :

- Créer de nouveaux projets.
- Gérer les projets existants.
- Gérer les membres de l'équipe de l'espace de travail et les autorisations.
- Gérer les abonnements et les licences commerciales.

## Espace de travail initial {#initial-workspace}

Lorsque vous créez un nouveau compte 8th Wall directement à partir de 8thwall.com, vous démarrez
avec un espace de travail sur le plan Basic (gratuit).

Si vous vous inscrivez via une invitation d'un autre utilisateur de 8th Wall, vous serez ajouté à
en tant que membre de l'équipe de son espace de travail existant.

\*\*Par défaut, l'URL
de votre expérience Web AR publiée sera dans le format suivant :

"nom-espace-de-travail.8ewall.app/nom-du-projet"

Pour modifier l'URL de l'espace de travail, il faut contacter l'assistance et créer une demande.
