# Tableau de bord du projet

Le tableau de bord des projets est le point central de la gestion des projets 8th Wall.  À partir de la page du tableau de bord du projet
, vous pouvez gérer les paramètres du projet, accéder à 8th Wall Cloud Editor, acheter des licences commerciales,
gérer les cibles d'image, configurer des domaines personnalisés, et bien plus encore.

L'URL directe de votre tableau de bord de projet se présente sous la forme suivante : www.8thwall.com/workspacename/projectname

## Gestion des cibles d'image {#managing-image-targets}

Pour gérer les cibles d'image d'un projet donné, cliquez sur l'icône Cible d'image dans le menu de gauche
ou sur le lien "Gérer les cibles d'image" dans le tableau de bord du projet.

![ManageImageTargets](/images/console-appkey-imagetargets.jpg)

Pour des informations détaillées sur les cibles d'image, veuillez consulter le site
[Documentation sur les cibles d'image] (/legacy/guides/image-targets).

## Gérer les emplacements du VPS Lightship {#managing-lightship-vps-locations}

Pour gérer les emplacements des VPS pour un projet donné, cliquez sur l'icône de la carte dans la navigation de gauche.

![ManageWayspots](/images/console-wayspot-left-nav.jpg)

Pour des informations détaillées sur Lightship VPS for Web, veuillez vous référer à
[Lightship VPS for Web documentation](/legacy/guides/lightship-vps).
