# Créer un projet d'éditeur patrimonial

1. Connectez-vous à votre compte 8th Wall et cliquez sur "Aller à mes projets" :

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Agrandissez votre espace de travail, s'il n'est pas déjà ouvert.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Descendez au bas de la page et cliquez sur **Construire avec l'éditeur patrimonial**.

![BuildWithLegacyEditor](/images/legacy/build-with-legacy-editor.jpg)

4. Saisissez l'URL du projet et cliquez sur **Créer**.

![CreateLegacyProject](/images/legacy/legacy-create-project.jpg)

:::info
Une **clé d'application** est utilisée pour développer localement et pour les projets auto-hébergés. Cette option n'est disponible que sur un [Plan personnalisé] (https://8thwall.com/custom).
:::
