# Page d'atterrissage

## Description {#description}

Ce module génère une page d'atterrissage personnalisée pour votre expérience Web AR.

## Fonctions {#functions}

| Fonction                            | Description                                                                                                                                   |
| ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Configure les paramètres de la page d'atterrissage.                                                                           |
| [pipelineModule](pipelinemodule.md) | Crée un module de pipeline de caméra qui, une fois installé, ajoute une fonctionnalité de page d'atterrissage à votre projet. |
