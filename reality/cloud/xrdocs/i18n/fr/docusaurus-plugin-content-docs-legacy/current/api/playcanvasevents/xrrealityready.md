# xr:realityready

## Description {#description}

Cet événement est déclenché lorsque 8th Wall Web a été initialisé et qu'au moins une image a été traitée avec succès.

## Exemple {#example}

```javascript
this.app.on('xr:realityready', () => {
  // Cacher l'interface de chargement
}, this)
```
