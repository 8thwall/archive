# XR8.MediaRecorder.pipelineModule()

`XR8.MediaRecorder.pipelineModule()`

## Description {#description}

Fournit un module de pipeline de caméra qui vous permet d'enregistrer une vidéo au format MP4.

## Paramètres {#parameters}

Aucun

## Retourne {#returns}

Un module de pipeline `MediaRecorder` permet d'enregistrer une vidéo.

## Exemple {#example}

```javascript
XR8.addCameraPipelineModule(XR8.MediaRecorder.pipelineModule())
```
