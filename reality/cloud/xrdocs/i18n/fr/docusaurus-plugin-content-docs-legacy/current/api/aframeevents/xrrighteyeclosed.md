# xrrighteyeclosed

## Description {#description}

Cet événement est émis par [`xrface`](/legacy/api/aframe/#face-effects) lorsque l'œil droit d'un visage suivi se ferme.

`xrrighteyeclosed.detail : {id}`

| Propriété | Type     | Description                                                    |
| --------- | -------- | -------------------------------------------------------------- |
| id        | `Nombre` | Un identifiant numérique de la face localisée. |
