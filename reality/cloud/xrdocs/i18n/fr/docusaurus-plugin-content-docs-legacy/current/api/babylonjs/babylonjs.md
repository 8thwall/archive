# XR8.Babylonjs

Babylon.js (<https://www.babylonjs.com/>) est un cadre JavaScript complet pour la création de jeux 3D et d'expériences
avec HTML5 et WebGL.  Combiné avec 8th Wall Web, vous pouvez créer de puissantes expériences de RA Web
.

Tutoriel vidéo :

- Intégration de Babylon.js et de 8th Wall Web : <https://youtu.be/RhZaebNpaEY>

## Description {#description}

Fournit une intégration qui s'interface avec l'environnement BabylonJS et le cycle de vie pour piloter la caméra Babylon.js afin de réaliser des superpositions virtuelles.

## Fonctions {#functions}

| Fonction                                    | Description                                                                                                                                  |
| ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| [xrCameraBehavior](xrcamerabehavior.md)     | Obtenir un comportement qui peut être attaché à une caméra Babylon pour exécuter le suivi du monde et/ou des cibles d'image. |
| [faceCameraBehavior](facecamerabehavior.md) | Obtenez un comportement qui peut être attaché à une caméra Babylon pour exécuter les effets de visage.                       |
