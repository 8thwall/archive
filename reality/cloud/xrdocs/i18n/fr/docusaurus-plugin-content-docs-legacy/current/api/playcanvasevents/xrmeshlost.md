# xr:meshlost

## Description {#description}

Cet événement est émis lorsque la fonction `recenter()` est appelée.

`xr:meshlost.detail : { id }`

| Propriété | Description                                                                           |
| --------- | ------------------------------------------------------------------------------------- |
| id        | Un identifiant pour ce maillage qui est stable au sein d'une session. |
