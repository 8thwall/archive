# AccessPass

## Description {#description}

Fournit un module pour monétiser votre expérience Web AR et Web VR. Cette option n'est disponible que pour les projets hébergés sur le 8e
Wall et nécessite le [Module de paiement] (/legacy/guides/advanced-topics/monetize-with-8th-wall-payments).

```ts
import {AccessPass} from 'paiements'
```

## Fonctions {#functions}

| Fonction                                            | Description                                                                                                                          |
| --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| [requestPurchaseIfNeed](requestpurchaseifneeded.md) | Ouvre une fenêtre de paiement où le client peut effectuer en toute sécurité le paiement de la carte d'accès fournie. |
