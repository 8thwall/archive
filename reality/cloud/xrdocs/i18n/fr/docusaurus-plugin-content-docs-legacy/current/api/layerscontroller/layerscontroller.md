# XR8.LayersController

## Description {#description}

`LayersController` fournit une détection sémantique des couches et des interfaces pour configurer le rendu des couches.

## Fonctions {#functions}

| Fonction                            | Description                                                                                                               |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Configure les traitements effectués par `LayersController`.                                               |
| [getLayerNames](getlayernames.md)   | Retourne les couches configurées par `LayersController`.                                                  |
| [pipelineModule](pipelinemodule.md) | Crée un module de pipeline de caméra qui, une fois installé, assure la détection des couches sémantiques. |
| [recentrer](recenter.md)            | Repositionne l'appareil photo dans la direction de l'origine ou de l'orientation.                         |
