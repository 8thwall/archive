# CoachingOverlay

## Description {#description}

Fournit un module qui génère une superposition de coaching pour votre expérience Absolute Scale Web AR.

Pour plus d'informations sur le Lightship VPS Coaching Overlay, veuillez consulter [ici](/legacy/api/vpscoachingoverlay).

## Fonctions {#functions}

| Fonction                            | Description                                                                                                                                           |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Configure les paramètres de superposition de la couverture.                                                                           |
| [pipelineModule](pipelinemodule.md) | Crée un module de pipeline de caméra qui, une fois installé, ajoute la fonctionnalité de superposition d'entraînement à votre projet. |
