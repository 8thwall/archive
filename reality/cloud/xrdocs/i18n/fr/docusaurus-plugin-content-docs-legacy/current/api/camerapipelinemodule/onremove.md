# onRemove()

`onRemove : ({framework})`

## Description {#description}

`onRemove` est appelé lorsqu'un module est retiré du pipeline.

## Paramètres {#parameters}

| Paramètres | Description                                                                        |
| ---------- | ---------------------------------------------------------------------------------- |
| cadre      | Les liaisons de ce module avec le cadre pour l'envoi d'événements. |
