# xrlefteyeopened

## Description {#description}

Cet événement est émis par [`xrface`](/legacy/api/aframe/#face-effects) lorsque l'œil gauche d'un visage suivi s'ouvre.

`xrlefteyeopened.detail : {id}`

| Propriété | Type     | Description                                                    |
| --------- | -------- | -------------------------------------------------------------- |
| id        | `Nombre` | Un identifiant numérique de la face localisée. |
