# XR8.version()

`XR8.version()`

## Description {#description}

Obtenir la version du moteur Web du 8e mur.

## Paramètres {#parameters}

Aucun

## Retourne {#returns}

Chaîne indiquant la version du moteur Web 8th Wall.

## Exemple {#example}

```javascript
console.log(XR8.version())
```
