# XR8.Threejs

## Description {#description}

Fournit un module de pipeline de caméra qui pilote la caméra three.js pour réaliser des superpositions virtuelles.

## Fonctions {#functions}

| Fonction                            | Description                                                                                                                                                                                                             |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [pipelineModule](pipelinemodule.md) | Un module de pipeline qui s'interface avec l'environnement et le style de vie de three.js.                                                                                              |
| [configure](configure.md)           | Configure le moteur de rendu three.js.                                                                                                                                                  |
| [xrScene](xrscene.md)               | Obtenir un handle sur la scène xr, la caméra, le moteur de rendu, la texture (facultative) de l'alimentation de la caméra et la couche (facultative) de la scène. |
