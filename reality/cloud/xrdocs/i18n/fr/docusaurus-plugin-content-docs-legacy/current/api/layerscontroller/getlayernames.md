# XR8.LayersController.getLayerNames()

`XR8.LayersController.getLayerNames()`

## Description {#description}

Retourne les couches configurées par le contrôleur `LayersController`.

## Paramètres {#parameters}

Aucun

## Retourne {#returns}

Une liste de chaînes de caractères dont chacune est un nom de couche.
