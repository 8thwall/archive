# xrinterpupillarydistance

## Description {#description}

Cet événement est émis par [`xrface`](/legacy/api/aframe/#face-effects) lorsque la distance en millimètres entre les centres de chaque pupille d'un visage suivi est détectée pour la première fois.

`xrinterpupillarydistance.detail : {id}`

| Propriété                | Type     | Description                                                                                |
| ------------------------ | -------- | ------------------------------------------------------------------------------------------ |
| id                       | `Nombre` | Un identifiant numérique de la face localisée.                             |
| distance interpupillaire | `Nombre` | Distance approximative en millimètres entre les centres de chaque pupille. |
