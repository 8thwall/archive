# Exporter des projets

## Exportation du code

L'exportation de code est désormais disponible pour les projets Studio et Cloud Editor. Cela vous permettra de télécharger un instantané complet de votre projet tel qu'il existe aujourd'hui sur 8th Wall. Avec cette version, vous pouvez **télécharger le code, les actifs et les images cibles en un seul clic**.

Code Export est conçu pour vous donner un accès immédiat et des sauvegardes de votre travail lors de la transition vers l'open source ou pour vous aider à évaluer l'utilisation d'autres plateformes.

:::warning[Note]
Cette version ne produit pas de projet constructible. Le code exporté ne peut pas être exécuté ou déployé tel quel. L'exportation de produits constructibles est imminente.
:::

### Studio

Pour télécharger une copie d'un projet Studio :

1. Ouvrez le projet Studio et cliquez sur **Publish**.

![](/images/migration/studio-publish.png)

2. Sélectionnez **Code** dans la section **Export**, puis cliquez sur le bouton **Export**.

![](/images/migration/studio-export-code.png)

3. Vous recevrez un fichier .zip contenant les images cibles, le code et les fichiers.

![](/images/migration/studio-files.png)

### Editeur de nuages

Pour télécharger une copie d'un projet de l'Editeur de nuages :

1. Ouvrez le projet de l'éditeur et cliquez sur **Publier**.

![](/images/migration/editor-publish.jpg)

2. Sélectionnez **Code** dans la section **Export**, puis cliquez sur le bouton **Export**.

![](/images/migration/editor-export-code.png)

3. Vous recevrez un fichier .zip contenant les images cibles, le code et les fichiers.

![](/images/migration/editor-files.png)

## Exportation de code constructible

Dans quelques semaines, nous publierons Buildable Code Export. Cette version comprendra tout ce qui est nécessaire pour construire et exécuter une version hors ligne de votre projet, y compris :

* Code du projet et actifs
* Cibles d'image
* Configuration de la construction
* Moteur XR
* Extras XR
* Page d'atterrissage

Cependant, le développement hors ligne ne supportera pas les fonctionnalités dépendantes du cloud ou celles que nous n'avons pas la licence de distribuer, comme par exemple :

* VPS / Cartes
* Suivi des mains
* Modules / Backends
