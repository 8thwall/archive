# Abonnements en marque blanche

## Achat Abonnement {#purchase-subscription}

:::info
Par défaut, les abonnements White Label **sont illimités et se renouvellent automatiquement** à chaque période de facturation jusqu'à ce que vous les annuliez ou programmiez une date de fin. Une fois l'abonnement acheté, les paramètres de durée de l'abonnement peuvent être gérés à partir de la page Tableau de bord du projet.
:::

1. Accéder à la page du tableau de bord du projet
2. Cliquez sur "Acheter maintenant"

![PurchaseSubscription](/images/account/subscription-purchase.jpg)

3. Consulter et accepter l'accord de marque blanche

![SubscriptionAgreement](/images/account/subscription-agreement.jpg)

2. Sélectionner la période de facturation. Vous avez le choix entre les options suivantes :
   - **Mensuel** : L'abonnement est renouvelé automatiquement tous les mois.
   - **Annuel** : L'abonnement est automatiquement renouvelé tous les 12 mois.

![SubscriptionBillingPeriod](/images/account/subscription-billing-period.jpg)

3. Sélectionnez le mode de paiement.

![SubscriptionPaymentMethod](/images/account/subscription-payment-method.jpg)

:::info
Si vous voyez l'erreur suivante, cliquez sur le lien pour accéder aux paramètres de votre compte et ajouter une adresse de facturation.

![SubscriptionBillingError](/images/account/subscription-billing-error.jpg)
:::

4. Vérifiez **Total dû aujourd'hui** et comprenez quand le prochain prélèvement automatique aura lieu, puis cliquez sur **Terminer l'achat**.

![SubscriptionSuccess](/images/account/subscription-success.jpg)

5. **Républier votre projet pour supprimer l'écran de démarrage par défaut**

## Annuler l'abonnement {#cancel-subscription}

Par défaut, les abonnements White Label **sont illimités et se renouvellent automatiquement** à chaque période de facturation jusqu'à ce que vous les annuliez ou programmiez une date de fin. Une fois l'abonnement acheté, les paramètres de durée de l'abonnement peuvent être gérés à partir de la page Tableau de bord du projet.

Les paramètres de la durée de l'abonnement peuvent être gérés à partir du tableau de bord du projet. Les options suivantes sont disponibles :

- **En cours** : Le projet se poursuivra indéfiniment et votre abonnement sera automatiquement renouvelé chaque mois. La date et l'heure du prochain renouvellement s'affichent.
- **Fin après le cycle de facturation en cours** :  L'abonnement se poursuivra jusqu'à la fin de la période de facturation en cours, puis s'achèvera.
- **Planifier une date et une heure de fin** : Sélectionnez une date et une heure personnalisées pour la fin de l'abonnement.
- **Terminer le projet maintenant** : Mettre fin à l'abonnement immédiatement (**Plans personnalisés avec Label blanc illimité uniquement**).

Pour modifier la durée de l'abonnement :

1. Accédez au tableau de bord de votre projet.
2. Dans la case **White Label Subscription**, cliquez sur **Edit** :
3. Définissez la date de fin d'abonnement souhaitée et cliquez sur **Mise à jour** pour enregistrer vos modifications :

![SubscriptionDuration](/images/account/subscription-duration.jpg)

## Réactiver l'abonnement {#reactivate-subscription}

Lorsqu'un abonnement White Label est annulé ou atteint une date de fin programmée, la facturation s'arrête et le projet est relocalisé sous l'onglet **Projets terminés** dans le tableau de bord du projet.

Pour relancer un projet et souscrire un nouvel abonnement en marque blanche, veuillez suivre l'une des options suivantes :

**Depuis la page Espace de travail** :

1. Dans la page Espace de travail, sélectionnez l'onglet **Projets terminés**.
2. Localisez le projet à réactiver.
3. Cliquez sur le menu "..." et sélectionnez **Reactivate White Label** dans le menu déroulant.
4. Complétez l'assistant [Purchase White Label](#purchase-subscription) pour réactiver l'abonnement White Label.

**Dans le tableau de bord du projet** :

1. Dans la page Espace de travail, sélectionnez l'onglet **Projets terminés**.
2. Sélectionnez le projet souhaité pour accéder à la page du tableau de bord du projet.
3. Cliquez sur "Lancer maintenant".
4. Complétez l'assistant [Purchase White Label](#purchase-subscription) pour réactiver l'abonnement White Label.

## URL de redirection du projet {#project-redirect-url}

Lorsqu'un projet lancé est annulé ou terminé, vous pouvez rediriger les utilisateurs vers une autre URL.

Spécifiez une URL de redirection de projet pour rediriger automatiquement vos utilisateurs vers un autre site lorsque votre abonnement
White Label est terminé.

1. Dans le tableau de bord du projet, repérez la section **Redirection de l'URL du projet**.
2. Cliquez sur **Set a URL** (Définir une URL)
3. Saisir l'URL de redirection souhaitée
