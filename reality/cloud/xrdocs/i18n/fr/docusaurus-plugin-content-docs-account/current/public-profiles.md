# Profils publics

:::info
Seuls les rôles d'utilisateur `Owner` et `Admin` peuvent modifier le profil public de l'espace de travail.
:::

Votre profil public est votre propre page sur 8thwall.com où vous pouvez présenter votre travail, démontrer votre expérience sur
et partager votre code avec d'autres développeurs si vous le souhaitez.

Votre profil public est défini au cours de la procédure de création du compte et affiche les informations suivantes :

- Photo de profil (minimum 400x400px PNG ou JPEG)
- Nom (3-30 caractères)
- URL du site web (https://www.mycompany.com)
- Lieu principal (ville)
- Biographie (240 caractères ou moins)
- Courriel de contact (par exemple : contact@mycompany.com)

## Mise à jour du profil public

Vous pouvez mettre à jour les détails de votre profil public à tout moment à partir de l'onglet **Profil public** :

1. Développez votre espace de travail dans la navigation de gauche.

![ExpandWorkspace](/images/console-home-my-workspaces.jpg)

2. Cliquez sur **Profils publics**

![SelectPublicProfile](/images/console-workspace-nav-public-profile.jpg)

3. Mise à jour des détails du profil public

4. Cliquez sur **Save** pour enregistrer les informations que vous avez saisies.

5. Accédez à votre profil public en ajoutant le nom court de votre espace de travail à https://8thwall.com/ (par exemple https://8thwall.com/mycompany).
