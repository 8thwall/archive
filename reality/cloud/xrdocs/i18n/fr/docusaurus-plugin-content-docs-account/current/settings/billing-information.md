# Mise à jour des informations de facturation

La section "Informations de facturation" de la page Compte vous permet de spécifier les informations de contact
que vous souhaitez voir apparaître sur les futures factures, ainsi que l'adresse électronique à laquelle vous souhaitez que les factures/reçus soient envoyés
.

:::info
Les informations de facturation mises à jour figureront sur les **factures/reçus ultérieurs**.
:::

Pour mettre à jour les informations de facturation du compte :

1. Connectez-vous à votre compte 8th Wall et cliquez sur "Aller à mes projets" :

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Agrandissez votre espace de travail, s'il n'est pas déjà ouvert.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Cliquez sur **Compte** dans la navigation de gauche.

![SelectAccount](/images/account/select-account.jpg)

4. Cliquez sur **Editer** pour modifier vos informations de facturation.

![BillingInformation](/images/console-workspace-account-invoice-details.jpg)

5. Cliquez sur **Mise à jour** pour enregistrer vos modifications.
