# Gérer la facturation et les modes de paiement

8th Wall utilise le portail de facturation Stripe pour gérer les méthodes de paiement et les informations de facturation. Sur cette page, vous pouvez

- Ajouter un nouveau mode de paiement
- Supprimer un mode de paiement
- Modifier votre mode de paiement par défaut
- Mise à jour des informations de facturation

1. Connectez-vous à votre compte 8th Wall et cliquez sur "Aller à mes projets" :

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Agrandissez votre espace de travail, s'il n'est pas déjà ouvert.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Cliquez sur **Compte** dans la navigation de gauche.

![SelectAccount](/images/account/select-account.jpg)

4. Cliquez sur **Gestion de la facturation et des modes de paiement**.

![ManageBilling](/images/account/manage-billing.jpg)

:::info
Les informations de facturation mises à jour ne figureront que sur les **factures/reçus ultérieurs**.
:::
