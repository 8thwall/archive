# XR8.clearCameraPipelineModules()

`XR8.clearCameraPipelineModules()`

## Description {#description}

Retirer tous les modules de canalisation de la caméra de la boucle de la caméra.

## Paramètres {#parameters}

Aucun

## Retourne {#returns}

Aucun

## Exemple {#example}

```javascript
XR8.clearCameraPipelineModules()
```
