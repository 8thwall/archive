# XR8.stop()

`XR8.stop()`

## Description {#description}

Lorsque l'appareil est arrêté, le flux de la caméra est fermé et les mouvements de l'appareil ne sont pas suivis. Doit appeler
`XR8.run()` pour redémarrer après l'arrêt du moteur.

## Paramètres {#parameters}

Aucun

## Retourne {#returns}

Aucun

## Exemple {#example}

```javascript
XR8.stop()
```
