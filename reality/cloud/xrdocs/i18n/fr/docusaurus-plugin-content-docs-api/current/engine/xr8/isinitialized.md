# XR8.isInitialized()

`XR8.isInitialized()`

## Description {#description}

Indique si le WebAssembly du moteur AR est initialisé ou non.

## Paramètres {#parameters}

Aucun

## Retourne {#returns}

Un booléen indiquant si le WebAssembly du moteur AR est initialisé ou non.

## Exemple {#example}

```javascript
if (XR8.isInitialized()) {
  console.log(XR8.version())
}
```
