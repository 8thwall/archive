# Matériel vidéo

## Description

Ce composant permet à l'utilisateur d'ajouter une texture vidéo à l'entité.

<VideoMaterialTable />

## Fonctions

<ComponentFunctions
    name="UnlitMaterial"
    properties={{
        r: 0,
        g: 0,
        b: 0,
        textureSrc: '',
        opacity: 1
    }}
    mutate={`cursor.r = 255;
    cursor.g = 255;
    cursor.b = 255;
    cursor.textureSrc = 'assets/my-video.mp4';
    cursor.opacity = 0.5;
    return false;`}
/>
