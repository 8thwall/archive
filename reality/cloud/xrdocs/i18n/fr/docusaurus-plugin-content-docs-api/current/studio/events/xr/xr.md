# Événements XR

Les événements XR sont émis par le moteur 8th Wall via des modules de pipeline de caméra. Les modules du pipeline de la caméra qui émettent des événements sont les suivants : `reality` et `facecontroller` : `reality` et `facecontroller`.

## réalité {#reality}

Événements émis par le module `reality` :

- Image Target Events
- Événements VPS
- Événements à effet mondial

## facecontroller {#facecontroller}

Événements émis par le module `facecontroller` :

- Événements de visage
