# Matériau d'ombre

## Description

Ce composant crée un matériau qui ne rend que les ombres sur une entité avec géométrie.

## Fonctions

<ComponentFunctions
    name="ShadowMaterial"
    properties={{
        r: 0,
        g: 0,
        b: 0,
        opacity: 0.5,
    }}
    mutate={`cursor.opacity = 0.5;
    cursor.r = 128;
    cursor.g = 128;
    cursor.b = 128;
    return false;`}
/>
