## Permettre les effets mondiaux

Pour activer les effets de monde dans votre projet, procédez comme suit :

1. Sélectionnez l'appareil photo dans votre scène.
2. Réglez le type sur "Monde".

```mdx-code-block
<img src={require('/images/studio/change-camera-to-world.png').default} alt='Camera properties include the ability to change the type of Camera.'/>
```

3. (Facultatif) Ajoutez un plan avec un composant d'ombre pour représenter le sol.

:::info
Le niveau du sol dans votre scène est défini comme 0 sur l'axe Y, par rapport à l'alimentation de la caméra.
:::
