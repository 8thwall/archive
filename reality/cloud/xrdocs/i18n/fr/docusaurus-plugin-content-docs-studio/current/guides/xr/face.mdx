## Activation des effets de visage

Pour activer les effets de visage dans votre projet, procédez comme suit :

1. Sélectionnez la caméra dans votre scène
2. Définir le type sur "Face"

```mdx-code-block
<img src={require('/images/studio/change-camera-to-face.png').default} alt='Camera properties include the ability to change the type of Camera.'/>
```

3. (Facultatif) Ajoutez un objet "Visage" à la hiérarchie de votre scène pour visualiser les effets spatiaux des visages.

```mdx-code-block
<img src={require('/images/studio/add-face-object.png').default} alt='Add a Face Object from the New Entity button in the Interface.'/>
```
