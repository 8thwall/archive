# Introduction

:::warning[Important Mise à jour ]
8th Wall se terminera au cours de l'année prochaine - [lire la mise à jour complète](/migration).
:::

{frontMatter.description} Avec Studio, vous pouvez facilement créer des expériences WebAR attrayantes, des jeux interactifs en 3D et bien plus encore en temps réel, puis les déployer de manière transparente sur les appareils mobiles, les ordinateurs de bureau et les casques avancés.

Studio est une solution complète pour créer, collaborer et publier des expériences WebAR, WebVR et des jeux 3D qui s'exécutent directement dans un navigateur web, sans aucune application.

![Intro-Image](/images/studio/introducing-studio.png)

Studio est un outil basé sur un navigateur qui ne nécessite aucun téléchargement de logiciel. Pour utiliser Studio, [connectez-vous à votre compte 8th Wall existant](https://www.8thwall.com/login), ou [créez un nouveau compte](https://www.8thwall.com/get-started) si vous n'en avez pas encore.
