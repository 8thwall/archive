# Voir le projet public

1. Accédez au tableau de bord du projet. Dans Studio, vous pouvez cliquer sur le menu hamburger dans le coin supérieur gauche. Dans la section **QR 8.code** du tableau de bord du projet, l'URL publique du projet sera affichée avec un lien court **8th.io** et un code QR associé.

2. Scannez le code QR avec votre appareil mobile pour voir le projet public.
