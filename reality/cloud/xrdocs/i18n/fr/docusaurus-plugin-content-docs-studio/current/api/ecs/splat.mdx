# Splat

## Description

Ce composant établit un Splat gaussien sur une entité.

## Propriétés

| Propriété | Type                 | Défaut  | Description         |
| --------- | -------------------- | ------- | ------------------- |
| url       | chaîne de caractères | ''      | L'url de la source. |
