# Quaternion

## Description

Ce composant contrôle la façon dont l'entité est tournée.

## Propriétés

| Propriété | Type   | Défaut  | Description  |
| --------- | ------ | ------- | ------------ |
| x         | nombre | 0       | Coordonnée x |
| y         | nombre | 0       | coordonnée y |
| z         | nombre | 0       | coordonnée z |
| w         | nombre | 0       | coordonnée w |

## Fonctions

    }}
