# CapsuleGéométrie

## Description

Ce composant établit la géométrie d'une entité.

## Propriétés

| Propriété | Type   | Défaut  | Description            |
| --------- | ------ | ------- | ---------------------- |
| rayon     | nombre | 1       | Rayon de la capsule.   |
| hauteur   | nombre | 1       | Hauteur de la capsule. |

## Fonctions

<ComponentFunctions
    name="CapsuleGeometry"
    properties={{ radius: 1, height: 1 }}
    mutate={`cursor.radius += 1;
    cursor.height *= 2;
    return false;`}
/>
