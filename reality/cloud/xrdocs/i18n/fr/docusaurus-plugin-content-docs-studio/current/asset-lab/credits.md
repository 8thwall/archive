# Crédits

Des crédits sont nécessaires pour utiliser la génération de ressources alimentée par l'IA dans Studio. Chaque action consomme un nombre spécifique de crédits en fonction du modèle ou du service utilisé. Tous les comptes commencent avec un pool de crédits gratuits lors de l'inscription, et des crédits supplémentaires peuvent être achetés par le biais d'un abonnement - ou bientôt par le biais d'un rechargement - sur votre page Compte.

Vous pouvez acheter un abonnement de crédit ou le recharger à partir de vos [Paramètres du compte] (/account/settings/manage-credits/).
