# Bibliothèque des actifs

Dans la bibliothèque Asset Lab :

- Accéder à tous les actifs générés par l'espace de travail
- Vérifiez votre solde de crédit affiché en haut à droite
- Filtrer par type : Images, Modèles 3D, Personnages animés
- Utiliser des outils de recherche et de tri pour organiser

![](/images/studio/asset-lab/library1.png)

Cliquez sur un bien pour obtenir des métadonnées :

- Invite utilisée
- Utilisateur qui a soumis la demande
- Modèle utilisé
- Coût du crédit

Actions disponibles :

- **Envoyer au flux de travail** - Réutiliser l'actif en tant qu'intrant
- **Télécharger** - Enregistrer localement
- **Remix** - Ajuster les paramètres d'entrée
- **Import** - Ajouter au dossier des actifs du projet

![](/images/studio/asset-lab/library2.png)
