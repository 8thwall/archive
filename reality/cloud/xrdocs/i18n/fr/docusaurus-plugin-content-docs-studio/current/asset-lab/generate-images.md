# Générer des images

## Étape 1 : Générer des images

Commencez par sélectionner un modèle dans le menu déroulant.

![](/images/studio/asset-lab/image-model.png)

Chaque modèle présente des atouts et des coûts de crédit qui lui sont propres. Voir la section [Prix du crédit](/account/settings/manage-credits/) pour plus de détails sur les prix.

### Modèles pris en charge

**Cœur d'image stable**
Stability AI's core model for fast, high-quality generation.
Entrées :

- Invite de texte
- Invite négative (facultatif)
- Rapport d'aspect (par défaut : carré)
- Taille du lot (1 ou 4)

**FLUX1.1 [pro]**
Version améliorée de FLUX.1 avec une fidélité et une composition accrues.
Entrées :

- Invite de texte
- Rapport d'aspect (par défaut : carré)
- Taille du lot (1 ou 4)

**FLUX Kontext [pro]**
Prend en charge les entrées de texte et d'images de référence.
Entrées :

- Invite de texte
- Demande d'image
- Rapport d'aspect (par défaut : carré)
- Taille du lot (1 ou 4)

**GPT-Image-1**
Le dernier modèle d'image multimodale d'OpenAI.
Entrées :

- Invite de texte
- Demande d'image (facultatif)
- Rapport d'aspect (par défaut : carré)
- Taille du lot (1 ou 4)

Sélectionnez le bouton Générer pour traiter la demande.

![](/images/studio/asset-lab/image-generate.png)

## Étape 2 : Importer dans un projet ou télécharger

Utilisez les boutons en bas de page pour importer ou télécharger toutes les images.

![](/images/studio/asset-lab/image-import.png)

Accédez à tous les actifs de votre espace de travail dans la **Bibliothèque**, disponible dans la barre latérale de Studio ou dans le laboratoire d'actifs en plein écran.

![](/images/studio/asset-lab/image-library.png)
