# Voir le projet public

1. Retournez au tableau de bord du projet dans la navigation de gauche.  Dans la section **QR 8.code**, l'URL du projet public
   sera affichée avec un lien court **8th.io** et un code QR associé.

2. Scannez le code QR avec votre appareil mobile pour voir le projet public.
