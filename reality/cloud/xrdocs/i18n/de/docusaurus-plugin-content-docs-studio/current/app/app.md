# 8th Wall Desktop App

:::info[Public Beta]
Die **8th Wall Desktop App befindet sich in der Public Beta** und die Funktionalität kann sich in einer zukünftigen Version ändern. Wir freuen uns über Ihr Feedback und haben ein [spezielles Support-Forum für die Beta-Benutzer der Desktop-Anwendung] (https://forum.8thwall.com/c/desktop-beta/17) eingerichtet - bitte melden Sie hier alle Probleme oder Vorschläge, die Sie haben.
:::

Die 8th Wall Desktop App bringt 8th Wall über den Browser hinaus in Ihren lokalen Workflow. Sie können jetzt Projekte direkt von Ihrem eigenen Rechner aus erstellen und gleichzeitig mit allem verbunden bleiben, was Sie bereits in 8th Wall kennen. Das bedeutet, dass Sie in Ihrem eigenen Code-Editor schneller vorankommen, Änderungen speichern und die Ergebnisse sofort sehen können, und dass Sie im Fluss bleiben, während Sie experimentieren und iterieren.
