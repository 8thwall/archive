## Ermöglichung von Welteffekten

Um Welteffekte in Ihrem Projekt zu aktivieren, gehen Sie folgendermaßen vor:

1. Wählen Sie die Kamera in Ihrer Szene aus.
2. Setzen Sie den Typ auf "Welt".

```mdx-code-block
<img src={require('/images/studio/change-camera-to-world.png').default} alt='Camera properties include the ability to change the type of Camera.'/>
```

3. (Optional) Fügen Sie eine Ebene mit einer Schattenkomponente hinzu, um den Boden darzustellen.

:::info
Die Bodenhöhe in Ihrer Szene ist als 0 auf der Y-Achse definiert, relativ zur Kameraführung.
:::
