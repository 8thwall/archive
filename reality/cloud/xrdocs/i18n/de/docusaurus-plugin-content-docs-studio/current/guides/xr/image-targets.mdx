## Aktivieren von Bildzielen

Um Bildziele in Ihrem Projekt zu aktivieren, müssen Sie eine Weltkamera verwenden.

1. Wählen Sie die **Kamera** in Ihrer Szene  
2. Setzen Sie den **Typ** auf `World`.

## Verwendung von Bild-Zielscheiben

Wenn Sie ein Bildziel zu Ihrem Projekt hinzufügen, wird ein Bildzielnetz angezeigt, so dass Sie andere 3D-Inhalte an dem Bildziel ausrichten und positionieren können.

## Bildziele simulieren

Mit dem **Simulator** von Studio können Sie Ihre Bildzielerfahrung aus der Ferne testen, indem Sie die Bewegung mit **WASD**-Steuerungen simulieren.

### Verwendung des Simulators mit Bildzielen

Um die Bewegung und die Platzierung von Inhalten um ein der Szene hinzugefügtes Bildziel zu simulieren:

1. Stellen Sie sicher, dass Ihre **Kameraeinstellungen** auf **Welt** eingestellt sind.
2. Drücken Sie **Play**, um den Simulator zu öffnen
3. Verwenden Sie die **Umgebungsoptionen des Simulators** (Auswahlfeld unten links), um das gewünschte Bildziel zu finden.
4. Verwenden Sie die **WASD-Tastaturkürzel**, um Bewegungen zu simulieren. _(Sie müssen zuerst auf das Simulator-Panel klicken, bevor Sie die Tastatur benutzen können)_

**Bewegungstasten:**

- W" = Vorwärts  
- `A` = Links  
- `S` = Rückwärts  
- `D` = Rechts  
- `Q` = Aufwärts  
- `E` = Abwärts

Wenn Sie sich mit der WASD-Taste um das Zielbild herum bewegen, wird das Ereignis **Bild gefunden** ausgelöst, um eine erfolgreiche Verfolgung des Zielbildes zu simulieren.
