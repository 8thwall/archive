## Aktivieren von Gesichtseffekten

Um Gesichtseffekte in Ihrem Projekt zu aktivieren, gehen Sie folgendermaßen vor:

1. Wählen Sie die Kamera in Ihrer Szene aus
2. Setzen Sie den Typ auf "Gesicht".

```mdx-code-block
<img src={require('/images/studio/change-camera-to-face.png').default} alt='Camera properties include the ability to change the type of Camera.'/>
```

3. (Optional) Fügen Sie ein "Face"-Objekt zu Ihrer Szenenhierarchie hinzu, um räumliche Gesichtseffekte zu visualisieren.

```mdx-code-block
<img src={require('/images/studio/add-face-object.png').default} alt='Add a Face Object from the New Entity button in the Interface.'/>
```
