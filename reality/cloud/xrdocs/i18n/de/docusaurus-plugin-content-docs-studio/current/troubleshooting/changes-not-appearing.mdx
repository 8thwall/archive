# Nicht angezeigte Änderungen

## Einführung

{frontMatter.description}

## Warum ist das so?

Wahrscheinlich haben Sie vergessen, Ihre Änderungen zu landen. Die Landung ist eine Voraussetzung dafür, dass andere Ihre Änderungen synchronisieren oder veröffentlichen können. Die Veröffentlichung ist der einzige Weg, wie Ihre Änderungen in Staging oder Produktion gelangen können.

## Wie kann ich es reparieren?

1. Bauen Sie
2. Land
3. In Staging oder Produktion veröffentlichen.
