# Anlagenbibliothek

In der Asset-Lab-Bibliothek:

- Zugriff auf alle im Arbeitsbereich erstellten Assets
- Prüfen Sie Ihr Guthaben, das oben rechts angezeigt wird
- Nach Typ filtern: Bilder, 3D-Modelle, Animierte Charaktere
- Verwenden Sie Such- und Sortierwerkzeuge zum Organisieren

![](/images/studio/asset-lab/library1.png)

Klicken Sie auf ein Asset, um Metadaten anzuzeigen:

- Verwendete Aufforderung
- Benutzer, der eingereicht hat
- Verwendetes Modell
- Kreditkosten

Verfügbare Aktionen:

- **An Workflow senden** - Asset als Input wiederverwenden
- **Download** - Lokal speichern
- **Remix** - Eingangseinstellungen anpassen
- **Importieren** - Dem Asset-Ordner des Projekts hinzufügen

![](/images/studio/asset-lab/library2.png)
