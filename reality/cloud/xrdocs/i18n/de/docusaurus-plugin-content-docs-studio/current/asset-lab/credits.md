# Kredite

Für die Nutzung der KI-gestützten Asset-Generierung in Studio sind Credits erforderlich. Jede Aktion verbraucht eine bestimmte Anzahl von Credits, die sich nach dem verwendeten Modell oder Dienst richtet. Alle Konten beginnen mit einem Pool kostenloser Credits bei der Anmeldung, und zusätzliche Credits können über ein Abonnement - oder bald auch über eine Aufladung - über Ihre Kontoseite erworben werden.

Sie können ein Guthaben-Abonnement erwerben oder über Ihre [Kontoeinstellungen](/account/settings/manage-credits/) aufladen.
