# Asset-Labor

Asset Lab ist eine KI-gestützte Suite von Funktionen zur Erstellung von Assets in 8th Wall Studio. Es wurde entwickelt, um die Arbeitsabläufe für die Erstellung von Bildern, 3D-Modellen und animierten Charakteren für die Verwendung in Studio-Projekten zu optimieren.

### Wesentliche Merkmale

- Anwendungsfalloptimierte Workflows für die Erstellung verschiedener Asset-Typen
- Integration mit branchenführenden KI-Modellen
- Einfacher Projektimport
- Auf Arbeitsplatzebene generierte Asset-Bibliothek & Guthaben

## Vermögenswerte generieren

1. Klicken Sie auf die Registerkarte **Asset Lab** im linken Bereich von Studio.
2. Klicken Sie auf die Schaltfläche "Generate…", um einen Arbeitsablauf auszuwählen.
3. Führen Sie die Schritte entsprechend dem von Ihnen gewählten Arbeitsablauf aus.

Die Schritte des von Ihnen gewählten Arbeitsablaufs werden im oberen Bereich angezeigt.
