# BoxGeometry

## Beschreibung

Diese Komponente legt die Geometrie einer Entität fest.

## Eigenschaften

| Eigenschaft | Typ    | Standard | Beschreibung                        |
| ----------- | ------ | -------- | ----------------------------------- |
| width       | Nummer | 1        | Breite der Seiten auf der X-Achse.  |
| height      | Nummer | 1        | Höhe der Seiten auf der Y-Achse.    |
| Tiefe       | Nummer | 1        | Tiefe der Seiten auf der Z-Achse.   |

## Funktionen

<ComponentFunctions
    name="BoxGeometry"
    properties={{width: 1, height: 2, depth: 1}}
    mutate={`cursor.width += 1;
    cursor.height *= 2;
    return false;`}
/>
