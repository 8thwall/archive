# ConeGeometry

## Beschreibung

Diese Komponente legt die Geometrie einer Entität fest.

## Eigenschaften

| Eigenschaft | Typ    | Standard | Beschreibung                           |
| ----------- | ------ | -------- | -------------------------------------- |
| radius      | Nummer | 1        | Radius des unteren Endes des Kegels.   |
| height      | Nummer | 1        | Höhe des Kegels.                       |

## Funktionen

<ComponentFunctions
    name="ConeGeometry"
    properties={{ radius: 1, height: 1 }}
    mutate={`cursor.radius += 1;
    cursor.height *= 2;
    return false;`}
/>
