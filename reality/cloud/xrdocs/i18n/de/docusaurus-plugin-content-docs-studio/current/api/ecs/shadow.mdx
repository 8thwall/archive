# Schatten

## Beschreibung

Diese Komponente steuert, wie die Entität mit Schatten umgeht.

## Eigenschaften

| Eigenschaft      | Typ     | Standard | Beschreibung                                      |
| ---------------- | ------- | -------- | ------------------------------------------------- |
| castShadow       | boolean | false    | Gibt an, ob ein Objekt Schatten werfen kann.      |
| receiveShadow    | boolean | false    | Gibt an, ob eine Entität Schatten empfangen kann. |

## Funktionen

<ComponentFunctions
    name="Shadow"
    properties={{
        castShadow: false,
        receiveShadow: false
    }}
    mutate={`cursor.castShadow = true;
    cursor.receiveShadow = true;
    return false;`}
/>
