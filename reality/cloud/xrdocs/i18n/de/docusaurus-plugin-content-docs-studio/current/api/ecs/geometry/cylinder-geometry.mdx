# CylinderGeometry

## Beschreibung

Diese Komponente legt die Geometrie einer Entität fest.

## Eigenschaften

| Eigenschaft | Typ    | Standard | Beschreibung             |
| ----------- | ------ | -------- | ------------------------ |
| radius      | Nummer | 1        | Radius des Zylinders.    |
| height      | Nummer | 1        | Höhe des Zylinders.      |

## Funktionen

<ComponentFunctions
    name="CylinderGeometry"
    properties={{ radius: 1, height: 1 }}
    mutate={`cursor.radius += 1;
    cursor.height *= 2;
    return false;`}
/>
