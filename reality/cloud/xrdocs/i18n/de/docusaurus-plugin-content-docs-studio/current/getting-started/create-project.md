# Ein Projekt erstellen

1. Klicken Sie auf der eingeloggten Homepage oder im Workspace Dashboard auf "Neues Projekt".

![StartNewProject](/images/console-workspace-start-project.jpg)
2. Wählen Sie den Arbeitsbereich für dieses Projekt aus, wenn Sie Mitglied in mehr als einem Arbeitsbereich sind.

3. Geben Sie die Projekt-URL ein.

:::warning
Die Projekt-URL wird verwendet, um die Standard-URL des Projekts sowie die URL der Seite "Featured Project" (falls aktiviert) zu generieren.  **Sie kann nicht geändert werden**. Um eine Projekt-URL zu ändern, müssen Sie das Projekt klonen und eine neue URL angeben.
:::

![ProjectUrl](/images/account/project-create.jpg)

4. Klicken Sie auf **Erstellen**.

5. **Wählen Sie eine Vorlage**: Wenn Sie das neue Projekt öffnen, werden Ihnen einige Projektvorlagen (
   ) angezeigt, die Ihnen den Einstieg erleichtern, darunter spielorientierte und AR-orientierte Vorlagen. Diese
   Projektvorlagen zeigen verschiedene Funktionalitäten und können durch den Austausch Ihrer
   eigenen Assets leicht verändert werden. Erstellen Sie ein Projekt unter Verwendung einer der verfügbaren Vorlagen oder erstellen Sie ein Projekt aus der Vorlage Empty
   .

![StudioTemplatePicker](/images/studio/studio-template-picker.png)

Wenn Ihr Projekt alles geladen hat, was Sie brauchen, können Sie mit der Erstellung beginnen.
