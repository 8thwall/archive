# Das öffentliche Projekt ansehen

1. Navigieren Sie zum Projekt-Dashboard. In Studio können Sie auf das Hamburger-Menü in der oberen linken Ecke klicken. Im Abschnitt **QR 8.code** des Projekt-Dashboards wird die öffentliche Projekt-URL zusammen mit einem **8th.io** Shortlink und einem zugehörigen QR-Code angezeigt.

2. Scannen Sie den QR-Code mit Ihrem Mobilgerät, um das öffentliche Projekt zu sehen.
