# Einführung

:::warning[Important Update]
8th Wall wird im Laufe des nächsten Jahres abgebaut - [lesen Sie jetzt die vollständige Aktualisierung](/migration).
:::

{frontMatter.description} Mit Studio können Sie auf einfache Weise ansprechende WebAR-Erlebnisse, interaktive 3D-Spiele und vieles mehr in Echtzeit erstellen und diese dann nahtlos auf mobilen Geräten, Desktops und modernen Headsets einsetzen.

Studio ist eine Komplettlösung für die Erstellung, Zusammenarbeit und Veröffentlichung von WebAR, WebVR-Erlebnissen und 3D-Spielen, die direkt in einem Webbrowser ausgeführt werden können - eine App ist nicht erforderlich.

![Intro-Image](/images/studio/introducing-studio.png)

Studio ist ein browserbasiertes Tool, das keinen Software-Download erfordert. Um Studio zu benutzen, [melden Sie sich bei Ihrem bestehenden 8th Wall-Konto an](https://www.8thwall.com/login), oder [erstellen Sie ein neues Konto](https://www.8thwall.com/get-started), wenn Sie noch keines haben.
