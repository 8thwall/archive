# Backend-Dienste

8th Wall Studio unterstützt jetzt zwei leistungsstarke neue Funktionen: Backend Proxies und Backend Funktionen.

Diese Tools bieten neue Möglichkeiten für 8th Wall Studio, mit denen Entwickler die Backend-Logik
handhaben und externe Dienste direkt in ihre WebAR und Webspiele integrieren können.
