# Komponenten

## Einführung

{frontMatter.description} Jede Komponente ist ein wiederverwendbarer Funktionsblock, der an eine Entität angehängt werden kann, um ihr bestimmte Verhaltensweisen oder Eigenschaften zu verleihen. Komponenten können das visuelle Erscheinungsbild, die physikalischen Eigenschaften, die Eingabeverarbeitung oder die benutzerdefinierte Spiellogik definieren. Durch die Kombination mehrerer Komponenten können Sie komplexe Entitäten mit vielfältigem Verhalten erstellen.

## Komponenten

Komponenten sind die Bausteine, die Entitäten ihre Funktionalität verleihen. Während eine Entität ein leeres Objekt darstellt, definieren Komponenten, wie dieses Objekt mit der Welt interagiert, aussieht oder sich verhält. In 8th Wall Studio können Sie integrierte Komponenten verwenden oder Ihre eigenen Komponenten erstellen, um einzigartige Verhaltensweisen für Ihr Spiel zu definieren.
