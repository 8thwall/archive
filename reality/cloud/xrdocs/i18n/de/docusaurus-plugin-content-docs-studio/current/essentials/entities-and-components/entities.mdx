# Entitäten

## Einführung

{frontMatter.description} Eine Entität an sich hat kein Verhalten oder Aussehen; sie fungiert lediglich als Container, an den Komponenten angehängt werden können. Diese Komponenten definieren das Verhalten und die Eigenschaften der Entität, wie z. B. ihre Position, ihr Erscheinungsbild oder ihre Interaktion mit physikalischen Systemen.

## Entitäten

Entitäten bilden das Rückgrat eines jeden Spiels oder einer Simulation in 8th Wall Studio. Durch die Kombination verschiedener Komponenten können Sie komplexe Objekte, Systeme und Interaktionen erstellen.

\*\*Entitäten werden durch eine eindeutige 64-Bit-Ganzzahl dargestellt, die als Entitäts-ID oder eid bezeichnet wird.
