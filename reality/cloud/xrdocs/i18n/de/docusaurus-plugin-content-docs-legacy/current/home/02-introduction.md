# Einführung

8th Wall bietet eine Komplettlösung für die Erstellung, Zusammenarbeit und Veröffentlichung von WebAR, WebVR-Erlebnissen
und 3D-Spielen, die direkt in einem Webbrowser ausgeführt werden können - eine App ist nicht erforderlich. 8. Wand frei zu verwenden.

## So funktioniert es {#how-it-works}

8th Wall Web ist eine vollständige
Implementierung der 8th Wall Simultaneous Localization and Mapping (SLAM) Engine, die unter
für Echtzeit-Web-AR auf Browsern optimiert wurde. Zu den AR-Funktionen gehören World Tracking, Bildziele, Gesichtseffekte,
und Himmelssegmentierung.

**Studio** ist ein visueller Echtzeit-XR-Editor und eine Spiele-Engine, die die Geschwindigkeit und Leistung des
Webs mit den Tools kombiniert, die für die Erstellung von hyper-immersiven 3D- und XR-Erlebnissen erforderlich sind.
Studio verfügt über eine visuelle 3D-Editor-Oberfläche zur Erstellung von XR- und Webspielen für verschiedene Geräte. Es verfügt über eine
komplett neue Engine mit Systemen für Physik, für Kamerabeleuchtung und Audio, für Animationsmaterialien
und Partikel, für die Verfolgung von Spieleraktionen und vieles mehr. Mit Studio können Sie fesselnde WebAR
Erlebnisse, 3D-Spiele und vieles mehr in Echtzeit erstellen und einfach auf mobilen Geräten, Desktops und
modernen Headsets einsetzen. [Erfahren Sie mehr über Studio](/studio).

Mit dem 8th Wall **Cloud Editor** können Sie voll funktionsfähige WebAR-Projekte entwickeln und
mit Teammitgliedern in Echtzeit zusammenarbeiten. Integriertes Hosting ermöglicht die Veröffentlichung von Projekten in mehreren
Bereitstellungsstatus, die im zuverlässigen und sicheren globalen Netzwerk von 8th Wall gehostet werden, einschließlich einer
passwortgeschützten Staging-Umgebung. Bei bestimmten Tarifen ist auch Self-Hosting möglich.

## Plattformmerkmale {#platform-features}

Die Entwicklerplattform von 8th Wall bietet Ihnen viele nützliche Funktionen. Werfen wir einen kurzen Blick
auf einige der wichtigsten Funktionen:

**Erstellen** Sie reichhaltige Web-AR- und Web-VR-Erlebnisse, die direkt in einem mobilen Webbrowser ausgeführt werden.

👥 **Zusammenarbeit** mit Teammitgliedern und Speicherung von Code in der Versionsverwaltung.

👀 Sofortige **Vorschau** von Projekten während der Entwicklung.

Kabelloses **Debugging** Ihres Codes in Echtzeit mit Live-Konsolenprotokollen von mehreren Geräten.

🌎 **Veröffentlichen** Sie Projekte, die im globalen Netzwerk von 8th Wall gehostet werden.

Erstellen Sie ein **Öffentliches Profil** und **Feature Projects** auf 8thwall.com, damit Sie Ihre Arbeit, Live-Demos und/oder Ihren Code präsentieren können.

## Kompatibilität der Rahmenbedingungen {#framework-compatibility}

Der 8th Wall Cloud Editor lässt sich leicht in 3D-JavaScript-Frameworks wie z.B.:

- [A-Frame](https://aframe.io/)
- [three.js](https://threejs.org/)
- [PlayCanvas](https://www.playcanvas.com)
- [Babylon.js](https://www.babylonjs.com/)

Hinweis: Um App-basierte AR mit Unity zu entwickeln, verwenden Sie [Niantic Lightship ARDK](https://lightship.dev/)
