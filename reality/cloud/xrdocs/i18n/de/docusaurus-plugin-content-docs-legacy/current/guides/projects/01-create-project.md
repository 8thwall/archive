# Legacy-Editor-Projekt erstellen

1. Loggen Sie sich in Ihr 8th Wall-Konto ein und klicken Sie auf "Zu meinen Projekten":

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Erweitern Sie Ihren Arbeitsbereich, falls er noch nicht geöffnet ist.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Führen Sie einen Bildlauf zum Ende der Seite durch, und klicken Sie auf **Mit Legacy-Editor erstellen**.

![BuildWithLegacyEditor](/images/legacy/build-with-legacy-editor.jpg)

4. Geben Sie die Projekt-URL ein und klicken Sie auf **Erstellen**.

![CreateLegacyProject](/images/legacy/legacy-create-project.jpg)

:::info
Ein **App Key** wird für die lokale Entwicklung und für selbst gehostete Projekte verwendet. Diese Option ist nur bei einem [Benutzerdefinierten Plan] (https://8thwall.com/custom) verfügbar.
:::
