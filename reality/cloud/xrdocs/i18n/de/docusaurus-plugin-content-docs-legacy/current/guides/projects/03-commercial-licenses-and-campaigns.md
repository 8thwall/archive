# Kommerzielle Lizenzen & Kampagnen

Marken und Organisationen benötigen kommerzielle Lizenzen oder einen Enterprise-Plan für die öffentliche Nutzung von 8th Wall-Projekten. Agenturen und Studios, die den Pro-Plan nutzen, müssen im Namen ihrer Kunden projektbezogene kommerzielle Lizenzen erwerben. [Weitere Informationen](https://www.8thwall.com/pricing)

## Kauf einer kommerziellen Lizenz {#purchase-commercial-license}

**Hinweis**: Kommerzielle Lizenzen laufen standardmäßig **unbefristet und verlängern sich automatisch** in jedem Abrechnungszeitraum, bis Sie kündigen oder ein Enddatum festlegen. Nach dem Kauf können die [Einstellungen für die Lizenzdauer] (/legacy/guides/projects/commercial-licenses-and-campaigns/#campaign-duration) über die Projekt-Dashboard-Seite für das betreffende Projekt verwaltet werden.

1. Gehen Sie zur Projekt-Dashboard-Seite
2. Ändern Sie den Projekttyp in "Kommerziell", wenn er es nicht schon ist.
3. Klicken Sie auf "Lizenz kaufen".

![PurchaseCommercialLicense](/images/console-appkey-project-purchase-license.jpg)

Folgen Sie dem Assistenten und erwerben Sie die gewünschte kommerzielle Lizenz:

1. Wählen Sie Commercial Agreement (und akzeptieren Sie die kommerzielle Lizenz, falls erforderlich)
2. Wählen Sie den Abrechnungszeitraum. Sie können zwischen den folgenden Optionen wählen:
   - **Monatlich**: Die Lizenz verlängert sich automatisch jeden Monat.
   - **Quartalsweise**: Die Lizenz wird automatisch alle 3 Monate erneuert.
   - **Halbjährlich**: Die Lizenz verlängert sich automatisch alle 6 Monate.
   - **Jährlich**: Die Lizenz wird automatisch alle 12 Monate erneuert.
3. Wählen Sie die Zahlungsmethode.
4. Überprüfen Sie "Heute fälliger Gesamtbetrag" und erfahren Sie, wann die nächste automatische Abbuchung erfolgen wird.
5. Klicken Sie auf "Kauf abschließen".

![PurchaseCommercialLicense2](/images/console-appkey-project-purchase-license2.jpg)

## Gewerbliche Lizenz reaktivieren {#reactivate-commercial-license}

Wenn eine kommerzielle Lizenz gekündigt wird oder ein geplantes Enddatum erreicht, wird die Rechnungsstellung eingestellt und das Projekt ist nicht mehr zugänglich.

Um ein Projekt neu zu starten und eine neue kommerzielle Lizenz zu erwerben, folgen Sie bitte einer dieser Optionen:

**Von der Seite Arbeitsbereich**:

1. Wählen Sie auf der Seite Arbeitsbereich die Registerkarte **Erledigte Projekte**.
2. Suchen Sie das zu reaktivierende Projekt.
3. Klicken Sie auf das Menü "..." und wählen Sie aus dem Dropdown-Menü **Kommerzielle Lizenz aktivieren**.
4. Schließen Sie den Assistenten [Kauf einer kommerziellen Lizenz](#purchase-commercial-license) ab, um eine neue Lizenz zu erwerben und das Projekt zu reaktivieren.

![ReactivateCommercialLicenseWorkspace](/images/completed-project-reactivate-workspace.jpg)

**Aus dem Projekt-Dashboard**:

1. Wählen Sie auf der Seite Arbeitsbereich die Registerkarte **Erledigte Projekte**.
2. Wählen Sie das gewünschte Projekt aus, um zur Seite Projekt-Dashboard zu navigieren.
3. Klicken Sie auf "Lizenz reaktivieren".
4. Schließen Sie den Assistenten [Kauf einer kommerziellen Lizenz](#purchase-commercial-license) ab, um eine neue Lizenz zu erwerben und das Projekt zu reaktivieren.

![ReactivateCommercialLicenseProjDashboard](/images/completed-project-reactivate-project-dashboard.jpg)

## Dauer der Kampagne {#campaign-duration}

Kommerzielle Lizenzen laufen standardmäßig **unbegrenzt und verlängern sich automatisch** in jedem Abrechnungszeitraum, bis Sie kündigen oder ein Enddatum festlegen. Wenn Sie eine Kampagne beenden, wird die kommerzielle Lizenz aufgehoben und das WebAR-Projekt wird deaktiviert.

Die Einstellungen für die Kampagnendauer können über das Projekt-Dashboard verwaltet werden. Die folgenden Optionen sind verfügbar:

- **Laufend**: Das Projekt läuft auf unbestimmte Zeit und Ihre kommerzielle Lizenz wird automatisch jeden Monat verlängert. Das Datum und die Uhrzeit der nächsten Erneuerung werden angezeigt.
- **Ende nach aktuellem Abrechnungszeitraum**:  Die Kampagne läuft bis zum Ende des aktuellen Abrechnungszeitraums und endet dann.
- **Ein Enddatum und -Uhrzeit festlegen**: Wählen Sie ein benutzerdefiniertes Datum und eine Uhrzeit für das Ende der Kampagne.

So ändern Sie die Dauer der Kampagne:

1. Rufen Sie das Projekt-Dashboard Ihres Projekts auf.
2. Klicken Sie im Feld "Projektdauer" auf "Bearbeiten":

![project-duration](/images/console-project-duration.jpg)

3. Legen Sie das gewünschte Enddatum der Lizenz fest und klicken Sie auf "Aktualisieren", um Ihre Änderungen zu speichern:

![project-duration-edit](/images/console-project-duration-edit.jpg)

Um die Kampagne sofort abzubrechen, besuchen Sie die Seite Arbeitsbereich-Konto und
[kommerzielle Lizenzen verwalten](/legacy/guides/account-settings/#manage-commercial-licenses).

## Kampagnen-Weiterleitungs-URL {#campaign-redirect-url}

Wenn ein gestartetes Projekt abgebrochen oder abgeschlossen wird, kann das WebAR-Projekt nicht mehr angezeigt werden.  Benutzer, die die Website
besuchen, sehen eine Fehlermeldung, die besagt, dass das Projekt nicht mehr verfügbar ist.  Es ist
eine bewährte Praxis, die Nutzer nach Beendigung Ihrer Kampagne an eine andere URL weiterzuleiten.

Geben Sie eine Kampagnenumleitungs-URL an, um Ihre Nutzer automatisch auf eine andere Website umzuleiten, wenn Ihre
Kampagne beendet ist.

Kampagnen-Umleitungs-URLs werden sowohl bei von 8th Wall gehosteten als auch bei selbst gehosteten Projekten unterstützt.

Klicken Sie im Project Dashboard auf "Connect a URL" und geben Sie die gewünschte Weiterleitungs-URL ein.
