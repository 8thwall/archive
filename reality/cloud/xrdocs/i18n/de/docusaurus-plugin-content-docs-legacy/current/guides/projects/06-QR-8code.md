# QR 8Code

Zur Erleichterung können für ein Projekt QR-Codes mit der Marke 8th Wall (auch "8 Codes" genannt) generiert werden, so dass
einfach von einem mobilen Gerät gescannt werden kann, um auf Ihr WebAR-Projekt zuzugreifen.  Sie können jederzeit unter
Ihre eigenen QR-Codes erstellen oder Websites oder Dienste von Drittanbietern zur Erstellung von QR-Codes nutzen.

Der QR-Code auf dem Projekt-Dashboard verweist auf einen einzigartigen "8th.io"-Kurzlink für Ihr Projekt. Dieser
Shortlink leitet den Nutzer dann auf die URL Ihres Web AR-Erlebnisses weiter.

<u>Sowohl der QR-Code als auch der "8th.io"-Code für ein bestimmtes Projekt sind statisch und ändern sich nicht je nach Projekttyp oder Lizenz.</u>

### 8th Wall Hosted Projects (KEINE verbundene Domain) {#8th-wall-hosted-projects-no-connected-domain}

Wenn Ihr Projekt die von 8th Wall gehostete Standard-URL verwendet (im Format
"**arbeitsplatzname**.8thwall.app/**projektname**"), werden der QR-Code und der 8th.io Shortlink immer auf die Standard-URL umgeleitet
.  Es ist nicht möglich, die Ziel-URL zu ändern.

![ProjectDashboard8wHostedQR](/images/console-appkey-qrcode-8w-hosted.png)

### 8th Wall Hosted Projects (MIT angeschlossener Domain) {#8th-wall-hosted-projects-with-connected-domain}

Wenn Sie eine [verbundene Domäne] (/legacy/guides/projects/connected-domains) für Ihr von der 8. Wall gehostetes Projekt konfiguriert haben, haben Sie die Möglichkeit, den QR-Code / Shortlink entweder auf die
Standard-URL des Projekts oder auf die primäre verbundene Domäne zu setzen.

Verwenden Sie die Optionsschaltfläche, um Ihr QR-Code- / Shortlink-Ziel festzulegen:

![ProjectDashboard8wHostedQRConnectedDomain](/images/console-appkey-qrcode-8w-hosted-connected-domain.png)

### Selbst gehostete Projekte {#self-hosted-projects}

Um einen QR-Code und einen Shortlink zu generieren, geben Sie die vollständige URL zu Ihrem selbst gehosteten Projekt ein und klicken Sie auf **Speichern**:

![ProjectDashboardSelfHostedQR](/images/console-appkey-qrcode.png)

Der generierte QR-Code kann entweder im PNG- oder SVG-Format heruntergeladen werden, um ihn auf einer Website, auf
oder an anderen Stellen anzubringen, so dass die Nutzer ihn mit ihrem Smartphone scannen können, um die selbst gehostete URL
zu besuchen.  Klicken Sie auf das Bleistiftsymbol, um den Zielort des Shortlinks zu bearbeiten, falls sich die selbst gehostete URL
in Zukunft ändern sollte.

Beispiel:

![ProjectDashboardSelfHostedQRResult](/images/console-appkey-qrcode-result.png)
