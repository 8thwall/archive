# Selbst gehostete Domains

Wenn Sie einen kostenpflichtigen Pro- oder Enterprise-Tarif haben, können Sie Web AR-Erlebnisse auf Ihrem eigenen Webserver
hosten (und sie ohne Geräteautorisierung anzeigen). Um dies zu tun
, müssen Sie eine Liste von Domains angeben, die zum Hosten Ihres
Projekts zugelassen sind.

1. Wählen Sie auf der Seite "Projekt-Dashboard" die Option "Domänen einrichten".

2. Erweitern Sie "Dieses Projekt für die Selbstverwaltung oder lokale Entwicklung einrichten".

3. Geben Sie die Domain(s) oder IP(s) des Webservers ein, auf dem Sie das Projekt
   selbst hosten werden. Eine Domäne darf keinen Platzhalter, Pfad oder Port enthalten. Klicken Sie auf das "+"
   , um mehrere hinzuzufügen.

Hinweis: Selbst gehostete Domains sind **subdomain-spezifisch** - z.B. ist `mydomain.com` NICHT dasselbe wie
`www.mydomain.com`. Wenn Sie sowohl auf mydomain.com als auch auf `www.mydomain.com` hosten wollen, müssen Sie
**BOTH** angeben.
