# Feuerschiff VPS

## Standortberechtigungen verweigert {#location-permissions-denied}

#### Ausgabe {#issue}

Wenn Sie ein Erlebnis starten, das das Lightship Maps Modul oder Lightship VPS verwendet, kann es sein, dass Sie die Meldung "LOCATION PERMISSION DENIED" erhalten. BITTE ERLAUBEN SIE ES UND VERSUCHEN SIE ES NOCH EINMAL."\\`

#### Auflösung {#resolution}

Aktivieren Sie Standortberechtigungen sowohl auf Browser- als auch auf Systemebene.

**Browser-Ebene (iOS)**

Bitte stellen Sie sicher, dass der Standort unter "Einstellungen > Safari > Standort" auf "Fragen" oder "Zulassen" eingestellt ist.

**Systemebene (iOS)**

Bitte stellen Sie sicher, dass die Standortdienste unter "Einstellungen > Datenschutz & Sicherheit > Standortdienste > Safari" aktiviert sind.

https://support.apple.com/en-us/HT207092

## Anmeldung bei Wayfarer mit 8th Wall über Google {#signing-into-wayfarer-with-8th-wall-using-google}

#### Ausgabe {#issue}

Wenn Sie versuchen, sich mit 8th Wall über die Google-Authentifizierung bei Wayfarer anzumelden, friert die App auf einem weißen Bildschirm ein.

#### Auflösung {#resolution}

Die Wayfarer-App unterstützt derzeit keine Google-Authentifizierung. Wenn Sie ein aktives 8th Wall-Konto haben, können Sie unter https://www.8thwall.com/profile eine E-Mail/Passwort-Kombination eingeben und sich damit bei Wayfarer anmelden.

## Der Test-Scan wird im Geospatial Browser nicht angezeigt {#test-scan-not-showing-up-in-the-geospatial-browser}

#### Ausgabe {#issue}

Sie führen einen Test-Scan in der Wayfarer-App durch, aber er erscheint nie im 8th Wall Geospatial Browser.

#### Auflösung {#resolution}

Wahrscheinlich ist auf der Profilseite in der Wayfarer-App der falsche Arbeitsbereich für die 8. Wand ausgewählt. Bitte wählen Sie den richtigen Arbeitsbereich der 8. Wand und scannen Sie den Standort erneut.

## Aktivierungs- oder Test-Scan bleibt in der Verarbeitung stecken {#activation-or-test-scan-stuck-in-processing}

#### Auflösung {#resolution}

Bitte wenden Sie sich an support@lightship.dev und geben Sie Einzelheiten zum VPS-Standort oder zum Test-Scan an.

## Fehlgeschlagene Aktivierung {#failed-activation}

#### Auflösung {#resolution}

Bitte wenden Sie sich an support@lightship.dev und geben Sie Einzelheiten zum VPS-Standort an.
