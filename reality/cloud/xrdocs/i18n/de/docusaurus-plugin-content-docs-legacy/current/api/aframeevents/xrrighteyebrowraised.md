# xrrighteyebrowraised

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/legacy/api/aframe/#face-effects) ausgelöst, wenn die rechte Augenbraue eines verfolgten Gesichts aus der Position angehoben wird, in der sie sich befand, als das Gesicht gefunden wurde.

`xrrighteyebrowraised.detail : {id}`

| Eigentum | Typ    | Beschreibung                                                      |
| -------- | ------ | ----------------------------------------------------------------- |
| id       | Nummer | Eine numerische Kennung für die gefundene Fläche. |
