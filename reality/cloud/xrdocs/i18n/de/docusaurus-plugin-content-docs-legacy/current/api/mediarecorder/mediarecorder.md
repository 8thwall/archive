# XR8.MediaRecorder

## Beschreibung {#description}

Bietet ein Kamera-Pipeline-Modul, mit dem Sie ein Video im MP4-Format aufnehmen können.

## Funktionen {#functions}

| Funktion                                  | Beschreibung                                                                                                                                       |
| ----------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)                 | Konfigurieren Sie die Einstellungen für die Videoaufzeichnung.                                                                     |
| [pipelineModule](pipelinemodule.md)       | Erstellt ein Pipeline-Modul, das Videos im MP4-Format aufzeichnet.                                                                 |
| [recordVideo](recordvideo.md)             | Aufnahme starten.                                                                                                                  |
| [requestMicrophone](requestmicrophone.md) | Ermöglicht die Aufnahme von Audio (falls nicht automatisch aktiviert) und fordert bei Bedarf Berechtigungen an. |
| [stopRecording](stoprecording.md)         | Aufnahme beenden.                                                                                                                  |
| [RequestMicOptions](requestmicoptions.md) | Enum für die Angabe, ob automatisch Mikrofonberechtigungen angefordert werden sollen oder nicht.                                   |
