# onDetach()

`onDetach: ({framework})`

## Beschreibung {#description}

onDetach" wird aufgerufen, nachdem ein Modul zum letzten Mal Rahmenaktualisierungen erhalten hat. Dies geschieht entweder, nachdem der Motor abgestellt oder das Modul manuell aus der Rohrleitung entfernt wurde, je nachdem, was zuerst eintritt.

## Parameter {#parameters}

| Parameter  | Beschreibung                                                                           |
| ---------- | -------------------------------------------------------------------------------------- |
| Rahmenwerk | Die Framework-Bindings für dieses Modul zum Versenden von Ereignissen. |
