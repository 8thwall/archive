# XR8.XrConfig.camera()

Aufzählung

## Beschreibung {#description}

Gewünschte Kamera zu verwenden.

## Eigenschaften {#properties}

| Eigentum | Wert   | Beschreibung                                                                      |
| -------- | ------ | --------------------------------------------------------------------------------- |
| FRONT    | Front  | Verwenden Sie die Frontkamera/Selfie-Kamera.                      |
| ZURÜCK   | Rücken | Verwenden Sie die nach hinten gerichtete Kamera / Rückfahrkamera. |
