# AccessPass

## Beschreibung {#description}

Bietet ein Modul zur Monetarisierung Ihrer Web-AR- und Web-VR-Erfahrung. Dies ist nur für 8th
Wall Hosted-Projekte verfügbar und erfordert das [Payments Module](/legacy/guides/advanced-topics/monetize-with-8th-wall-payments).

```ts
{AccessPass} von 'payments' importieren
```

## Funktionen {#functions}

| Funktion                                              | Beschreibung                                                                                                                         |
| ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| [requestPurchaseIfNeeded](requestpurchaseifneeded.md) | Öffnet ein Kassenfenster, in dem der Kunde eine sichere Zahlung für den bereitgestellten Zugangspass vornehmen kann. |
