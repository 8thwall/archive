# xrlefteyebrowraised

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/legacy/api/aframe/#face-effects) ausgelöst, wenn sich die linke Augenbraue eines verfolgten Gesichts gegenüber der Ausgangsposition beim Auffinden des Gesichts hebt.

`xrlefteyebrowraised.detail : {id}`

| Eigentum | Typ    | Beschreibung                                                      |
| -------- | ------ | ----------------------------------------------------------------- |
| id       | Nummer | Eine numerische Kennung für die gefundene Fläche. |
