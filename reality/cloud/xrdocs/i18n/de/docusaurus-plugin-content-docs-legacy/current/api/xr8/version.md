# XR8.version()

`XR8.version()`

## Beschreibung {#description}

Holen Sie sich die 8. Version der Wall-Web-Engine.

## Parameter {#parameters}

Keine

## Rückgabe {#returns}

Eine Zeichenkette, die die Version der 8th Wall Web Engine angibt.

## Beispiel {#example}

```javascript
console.log(XR8.version())
```
