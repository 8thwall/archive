# VpsCoachingOverlay

## Beschreibung {#description}

Stellt ein Modul zur Verfügung, das ein Coaching-Overlay für Ihr Lightship VPS-fähiges Web AR-Erlebnis erzeugt.

Informationen über das Absolute Scale Coaching Overlay finden Sie [hier](/legacy/api/coachingoverlay).

## Funktionen {#functions}

| Funktion                                                | Beschreibung                                                                                                                                  |
| ------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](vps-coachingoverlay-configure.md)           | Konfiguriert die Einstellungen für das Coaching-Overlay.                                                                      |
| [pipelineModule](vps-coachingoverlay-pipelinemodule.md) | Erstellt ein Kamera-Pipeline-Modul, das nach der Installation die Coaching-Overlay-Funktionalität zu Ihrem Projekt hinzufügt. |
