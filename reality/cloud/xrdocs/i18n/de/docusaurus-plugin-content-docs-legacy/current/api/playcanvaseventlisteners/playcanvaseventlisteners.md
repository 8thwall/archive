# PlayCanvas-Ereignis-Listener

Dieser Abschnitt beschreibt die Ereignisse, die von 8th Wall Web in einer PlayCanvas-Umgebung abgehört werden.

Sie können diese Ereignisse in Ihrer Webanwendung auslösen, um verschiedene Aktionen durchzuführen:

| Ereignis-Listener                                              | Beschreibung                                                                                                                                                                                                                                                                                                                                                                       |
| -------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [xr:hidecamerafeed](xrhidecamerafeed.md)       | Blendet das Kamerabild aus. Die Verfolgung hört nicht auf.                                                                                                                                                                                                                                                                                         |
| [xr:recenter](xrrecenter.md)                   | Führt das Kamerabild zu seinem Ursprung zurück. Wenn ein neuer Ursprung als Argument angegeben wird, wird der Ursprung der Kamera auf diesen zurückgesetzt, und sie wird dann neu zentriert.                                                                                                                                                       |
| [xr:screenshotrequest](xrscreenshotrequest.md) | Sendet eine Anfrage an die Engine, um einen Screenshot der PlayCanvas-Leinwand zu erstellen. Die Engine gibt ein [`xr:screenshotready`](/legacy/api/playcanvasevents/xrscreenshotready)-Ereignis mit dem JPEG-komprimierten Bild aus oder [`xr:screenshoterror`](/legacy/api/playcanvasevents/xrscreenshoterror), wenn ein Fehler aufgetreten ist. |
| [xr:showcamerafeed](xrshowcamerafeed.md)       | Zeigt das Kamerabild an.                                                                                                                                                                                                                                                                                                                                           |
| [xr:stopxr](xrstopxr.md)                       | Beendet die aktuelle XR-Sitzung. Im angehaltenen Zustand wird die Kameraübertragung gestoppt und die Bewegung des Geräts wird nicht verfolgt.                                                                                                                                                                                                      |
