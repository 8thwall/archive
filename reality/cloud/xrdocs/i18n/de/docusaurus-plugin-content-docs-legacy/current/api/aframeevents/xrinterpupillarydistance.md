# xrinterpupillarydistance

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/legacy/api/aframe/#face-effects) ausgelöst, wenn der Abstand in Millimetern zwischen den Mittelpunkten der einzelnen Pupillen eines verfolgten Gesichts zum ersten Mal erkannt wird.

`xrinterpupillarydistance.detail : {id}`

| Eigentum              | Typ    | Beschreibung                                                                                         |
| --------------------- | ------ | ---------------------------------------------------------------------------------------------------- |
| id                    | Nummer | Eine numerische Kennung für die gefundene Fläche.                                    |
| interpupillareDistanz | Nummer | Ungefährer Abstand in Millimetern zwischen den Mittelpunkten der einzelnen Pupillen. |
