# XR8.XrController.recenter()

`XR8.XrController.recenter()`

## Beschreibung {#description}

Positioniert die Kamera neu auf die durch [`XR8.XrController.updateCameraProjectionMatrix()`](updatecameraprojectionmatrix.md) angegebene Ausgangs- bzw. Blickrichtung und startet die Verfolgung neu.

## Parameter {#parameters}

Keine

## Rückgabe {#returns}

Keine
