## XR8.XrConfig {#xr8xrconfig}

## Beschreibung {#description}

Dienstprogramme zur Angabe der Geräte- und Kameraklasse, auf denen Pipeline-Module laufen sollen.

## Eigenschaften {#properties}

| Eigentum                                 | Typ  | Beschreibung                                                                            |
| ---------------------------------------- | ---- | --------------------------------------------------------------------------------------- |
| [camera()](camera.md) | Enum | Gewünschte Kamera zu verwenden.                                         |
| [Gerät()](device.md)  | Enum | Geben Sie die Klasse der Geräte an, auf denen die Pipeline laufen soll. |
