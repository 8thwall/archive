# AFrame Ereignis-Listener

Dieser Abschnitt beschreibt die Ereignisse, auf die die A-Frame-Komponente "xrweb" wartet

Sie können diese Ereignisse in Ihrer Webanwendung ausgeben, um verschiedene Aktionen durchzuführen:

| Ereignis-Listener                         | Beschreibung                                                                                                                                                                                                                                                                                                                                            |
| ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [hidecamerafeed](hidecamerafeed.md)       | Blendet das Kamerabild aus. Die Verfolgung hört nicht auf.                                                                                                                                                                                                                                                              |
| [recenter](recenter.md)                   | Führt das Kamerabild zu seinem Ursprung zurück. Wenn ein neuer Ursprung als Argument angegeben wird, wird der Ursprung der Kamera auf diesen zurückgesetzt, und sie wird dann neu zentriert.                                                                                                                            |
| [screenshotrequest](screenshotrequest.md) | Sendet eine Anfrage an die Engine, um einen Screenshot des AFrame-Canvas zu erstellen. Die Engine sendet ein [`screenshotready`](/legacy/api/aframeevents/screenshotready) Ereignis mit dem JPEG komprimierten Bild oder [`screenshoterror`](/legacy/api/aframeevents/screenshoterror) wenn ein Fehler aufgetreten ist. |
| [showcamerafeed](showcamerafeed.md)       | Zeigt das Kamerabild an.                                                                                                                                                                                                                                                                                                                |
| [stopxr](stopxr.md)                       | Beendet die aktuelle XR-Sitzung. Im angehaltenen Zustand wird die Kameraübertragung gestoppt und die Bewegung des Geräts wird nicht verfolgt.                                                                                                                                                                           |
