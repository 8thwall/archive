# XR8.stop()

`XR8.stop()`

## Beschreibung {#description}

Im angehaltenen Zustand ist die Kameraübertragung geschlossen und die Bewegung des Geräts wird nicht verfolgt. Muss
[`XR8.run()`](run.md) aufrufen, um nach dem Abstellen des Motors neu zu starten.

## Parameter {#parameters}

Keine

## Rückgabe {#returns}

Keine

## Beispiel {#example}

```javascript
XR8.stop()
```
