# XR8.LayersController.getLayerNames()

`XR8.LayersController.getLayerNames()`

## Beschreibung {#description}

Gibt die Ebenen zurück, die vom `LayersController` konfiguriert wurden.

## Parameter {#parameters}

Keine

## Rückgabe {#returns}

Eine Liste von Strings, wobei jeder ein Ebenenname ist.
