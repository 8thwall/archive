# LandingPage

## Beschreibung {#description}

Stellt ein Modul zur Verfügung, das eine benutzerdefinierte Landing Page für Ihr Web AR-Erlebnis generiert.

## Funktionen {#functions}

| Funktion                            | Beschreibung                                                                                                                                    |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Konfiguriert die Einstellungen der LandingPage.                                                                                 |
| [pipelineModule](pipelinemodule.md) | Erstellt ein Kamera-Pipeline-Modul, das nach der Installation die Funktionalität einer Landing Page zu Ihrem Projekt hinzufügt. |
