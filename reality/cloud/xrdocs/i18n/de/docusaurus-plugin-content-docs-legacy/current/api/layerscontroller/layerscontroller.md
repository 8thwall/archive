# XR8.LayersController

## Beschreibung {#description}

LayersController" bietet eine semantische Erkennung von Ebenen und Schnittstellen für die Konfiguration der Darstellung von Ebenen.

## Funktionen {#functions}

| Funktion                            | Beschreibung                                                                                                               |
| ----------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Legt fest, welche Verarbeitung von `LayersController` durchgeführt wird.                                   |
| [getLayerNames](getlayernames.md)   | Gibt die Ebenen zurück, die von `LayersController` konfiguriert wurden.                                    |
| [pipelineModule](pipelinemodule.md) | Erstellt ein Kamera-Pipelinemodul, das nach der Installation die Erkennung semantischer Ebenen ermöglicht. |
| [recenter](recenter.md)             | Richtet die Kamera auf den Ursprung / die Blickrichtung aus.                                               |
