# xrrighteyebrowlowered

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/legacy/api/aframe/#face-effects) ausgelöst, wenn die rechte Augenbraue eines verfolgten Gesichts auf ihre ursprüngliche Position gesenkt wird, wenn das Gesicht gefunden wurde

xrrighteyebrowlowered.detail : {id}\\`

| Eigentum | Typ    | Beschreibung                                                      |
| -------- | ------ | ----------------------------------------------------------------- |
| id       | Nummer | Eine numerische Kennung für die gefundene Fläche. |
