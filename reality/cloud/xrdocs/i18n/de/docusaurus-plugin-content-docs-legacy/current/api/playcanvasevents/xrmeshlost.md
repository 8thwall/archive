# xr:meshlost

## Beschreibung {#description}

Dieses Ereignis wird ausgelöst, wenn "recenter()" aufgerufen wird.

`xr:meshlost.detail : { id }`

| Eigentum | Beschreibung                                                                     |
| -------- | -------------------------------------------------------------------------------- |
| id       | Eine ID für dieses Netz, die innerhalb einer Sitzung stabil ist. |
