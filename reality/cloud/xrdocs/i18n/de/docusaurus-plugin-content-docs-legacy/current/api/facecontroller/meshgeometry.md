# XR8.FaceController.MeshGeometry

Aufzählung

## Beschreibung {#description}

Optionen zur Festlegung, für welche Teile der Fläche Maschendreiecke zurückgegeben werden sollen.

## Eigenschaften {#properties}

| Eigentum | Wert    | Beschreibung                                             |
| -------- | ------- | -------------------------------------------------------- |
| FACE     | Gesicht | Rückgabe der Geometrie für die Fläche.   |
| MOUTH    | "Mund   | Rückführungsgeometrie für den Mund.      |
| AUGEN    | "Augen  | Rückkehrgeometrie für die Augen.         |
| IRIS     | 'iris'  | Rückgabegeometrie für die Schwertlilien. |
