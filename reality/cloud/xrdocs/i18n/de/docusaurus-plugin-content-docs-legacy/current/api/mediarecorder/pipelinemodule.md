# XR8.MediaRecorder.pipelineModule()

`XR8.MediaRecorder.pipelineModule()`

## Beschreibung {#description}

Bietet ein Kamera-Pipeline-Modul, mit dem Sie ein Video im MP4-Format aufnehmen können.

## Parameter {#parameters}

Keine

## Rückgabe {#returns}

Mit dem Pipelinemodul `MediaRecorder` können Sie ein Video aufnehmen.

## Beispiel {#example}

```javascript
XR8.addCameraPipelineModule(XR8.MediaRecorder.pipelineModule())
```
