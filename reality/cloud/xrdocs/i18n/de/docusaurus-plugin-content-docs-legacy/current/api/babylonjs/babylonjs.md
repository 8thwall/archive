# XR8.Babylonjs

Babylon.js (<https://www.babylonjs.com/>) ist ein komplettes JavaScript-Framework zur Erstellung von 3D-Spielen und
-Erlebnissen mit HTML5 und WebGL.  In Kombination mit 8th Wall Web können Sie leistungsstarke Web AR
Erlebnisse schaffen.

Tutorial-Video:

- Integration von Babylon.js und 8th Wall Web: <https://youtu.be/RhZaebNpaEY>

## Beschreibung {#description}

Stellt eine Integration zur Verfügung, die eine Schnittstelle zur BabylonJS-Umgebung und zum Lebenszyklus bildet, um die Babylon.js-Kamera für virtuelle Überlagerungen zu steuern.

## Funktionen {#functions}

| Funktion                                    | Beschreibung                                                                                                                                            |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [xrCameraBehavior](xrcamerabehavior.md)     | Holen Sie sich ein Verhalten, das mit einer Babylon-Kamera verbunden werden kann, um World Tracking und/oder Image Targets auszuführen. |
| [faceCameraBehavior](facecamerabehavior.md) | Besorgen Sie sich ein Verhalten, das an eine Babylon-Kamera angeschlossen werden kann, um Face Effects auszuführen.                     |
