# xrlefteyebrowlowered

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/legacy/api/aframe/#face-effects) ausgelöst, wenn die linke Augenbraue eines verfolgten Gesichts in ihre Ausgangsposition gesenkt wird, wenn das Gesicht gefunden wurde.

`xrlefteyebrowlowered.detail : {id}`

| Eigentum | Typ    | Beschreibung                                                      |
| -------- | ------ | ----------------------------------------------------------------- |
| id       | Nummer | Eine numerische Kennung für die gefundene Fläche. |
