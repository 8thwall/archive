# Rechnungsinformationen aktualisieren

Im Abschnitt "Rechnungsinformationen" auf der Seite "Konto" können Sie die Kontaktinformationen
angeben, die auf künftigen Rechnungen erscheinen sollen, sowie die E-Mail-Adresse, an die Rechnungen/Belege gesendet werden sollen
.

:::info
Die aktualisierten Rechnungsdaten werden auf **künftigen Rechnungen/Belege** ausgewiesen.
:::

So aktualisieren Sie die Abrechnungsinformationen Ihres Kontos:

1. Loggen Sie sich in Ihr 8th Wall-Konto ein und klicken Sie auf "Zu meinen Projekten":

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Erweitern Sie Ihren Arbeitsbereich, falls er noch nicht geöffnet ist.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Klicken Sie in der linken Navigation auf **Konto**.

![SelectAccount](/images/account/select-account.jpg)

4. Klicken Sie auf **Bearbeiten**, um Ihre Rechnungsdaten zu ändern.

![BillingInformation](/images/console-workspace-account-invoice-details.jpg)

5. Klicken Sie auf **Aktualisieren**, um Ihre Änderungen zu speichern.
