# Zahlungsarten verwalten

Das Widget Zahlungsmethoden ermöglicht es Ihnen,:

- Fügen Sie eine neue Zahlungsmethode hinzu.
- Zahlungsmethode als Standard festlegen (für künftige Gebühren zu verwenden)
- Zahlungsarten entfernen.

Hinzufügen, Entfernen oder Festlegen einer neuen Standardzahlungsmethode:

1. Loggen Sie sich in Ihr 8th Wall-Konto ein und klicken Sie auf "Zu meinen Projekten":

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Erweitern Sie Ihren Arbeitsbereich, falls er noch nicht geöffnet ist.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Klicken Sie in der linken Navigation auf **Konto**.

![SelectAccount](/images/account/select-account.jpg)

Auf dieser Seite können Sie Ihre Zahlungsarten sowie die Rechnungsinformationen, die auf Ihren Rechnungen erscheinen sollen, verwalten:
.

Klicken Sie auf "Zahlungsmethode hinzufügen", um eine neue Kreditkarte zu Ihrem Konto hinzuzufügen. Wenn Sie möchten, dass diese neu
hinzugefügte Kreditkarte für künftige Rechnungen verwendet wird, klicken Sie bitte auf "Standard machen".

![PaymentMethods](/images/console-workspace-account-payment-methods.jpg)
