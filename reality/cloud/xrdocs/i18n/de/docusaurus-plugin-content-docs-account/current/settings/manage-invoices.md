# Rechnungen verwalten

1. Loggen Sie sich in Ihr 8th Wall-Konto ein und klicken Sie auf "Zu meinen Projekten":

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Erweitern Sie Ihren Arbeitsbereich, falls er noch nicht geöffnet ist.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Klicken Sie in der linken Navigation auf **Konto**.

![SelectAccount](/images/account/select-account.jpg)

4. Klicken Sie auf **Rechnung und Zahlungsmethoden verwalten**.

![ManageBilling](/images/account/manage-billing.jpg)

5. Eine Liste der Rechnungen finden Sie im unteren Bereich des Stripe Billing Portals.

![ViewInvoices](/images/account/view-invoices.jpg)

6. Klicken Sie auf eine Rechnung, um die Details zu öffnen. Hier haben Sie auch die Möglichkeit, die Rechnung oder den Beleg herunterzuladen.

![InvoiceDetails](/images/account/invoice-details.jpg)
