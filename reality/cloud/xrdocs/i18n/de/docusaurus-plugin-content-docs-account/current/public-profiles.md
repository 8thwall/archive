# Öffentliche Profile

:::info
Nur die Benutzerrollen `Owner` und `Admin` können Änderungen am öffentlichen Profil des Arbeitsbereichs vornehmen.
:::

Ihr öffentliches Profil ist Ihre eigene Seite auf 8thwall.com, auf der Sie Ihre Arbeit vorstellen, Ihre
Erfahrung demonstrieren und Ihren Code mit anderen Entwicklern teilen können, wenn Sie möchten.

Ihr öffentliches Profil wird während der Kontoerstellung eingerichtet und enthält die folgenden Informationen:

- Profilfoto (mindestens 400x400px PNG oder JPEG)
- Name (3-30 Zeichen)
- URL der Website (https://www.mycompany.com)
- Hauptstandort (Stadt)
- Bio (240 Zeichen oder weniger)
- Kontakt-E-Mail (z. B. contact@mycompany.com)

## Öffentliches Profil aktualisieren

Sie können die Details Ihres öffentlichen Profils jederzeit über die Registerkarte **Öffentliches Profil** aktualisieren:

1. Erweitern Sie Ihren Arbeitsbereich in der linken Navigation.

![ExpandWorkspace](/images/console-home-my-workspaces.jpg)

2. Klicken Sie auf **Öffentliche Profile**

![SelectPublicProfile](/images/console-workspace-nav-public-profile.jpg)

3. Öffentliche Profildaten aktualisieren

4. Klicken Sie auf **Speichern**, um die von Ihnen eingegebenen Informationen zu speichern.

5. Greifen Sie auf Ihr öffentliches Profil zu, indem Sie Ihren Kurznamen für den Arbeitsbereich an https://8thwall.com/ anhängen (z. B. https://8thwall.com/mycompany).
