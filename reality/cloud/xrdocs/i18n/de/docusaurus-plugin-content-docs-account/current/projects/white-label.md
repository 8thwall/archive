# White-Label-Abonnements

## Kauf eines Abonnements {#purchase-subscription}

:::info
Standardmäßig laufen White Label-Abonnements **unbefristet und verlängern** sich automatisch in jedem Abrechnungszeitraum, bis Sie kündigen oder ein Enddatum festlegen. Nach dem Kauf können die Einstellungen für die Abonnementdauer auf der Seite Projekt-Dashboard verwaltet werden.
:::

1. Gehen Sie zur Projekt-Dashboard-Seite
2. Klicken Sie auf "Jetzt kaufen".

![PurchaseSubscription](/images/account/subscription-purchase.jpg)

3. Lesen und akzeptieren Sie die White Label-Vereinbarung

![SubscriptionAgreement](/images/account/subscription-agreement.jpg)

2. Wählen Sie den Abrechnungszeitraum. Sie können zwischen den folgenden Optionen wählen:
   - **Monatlich**: Das Abonnement verlängert sich automatisch jeden Monat.
   - **Jahresabonnement**: Das Abonnement verlängert sich automatisch alle 12 Monate.

![SubscriptionBillingPeriod](/images/account/subscription-billing-period.jpg)

3. Wählen Sie die Zahlungsmethode.

![SubscriptionPaymentMethod](/images/account/subscription-payment-method.jpg)

:::info
Wenn Sie die folgende Fehlermeldung sehen, klicken Sie auf den Link, um zu Ihren Kontoeinstellungen zu navigieren und eine Rechnungsadresse hinzuzufügen.

![SubscriptionBillingError](/images/account/subscription-billing-error.jpg)
:::

4. Überprüfen Sie die **Summe, die heute fällig ist**, und erfahren Sie, wann die nächste automatische Aufladung stattfinden wird, und klicken Sie auf **Kauf abschließen**.

![SubscriptionSuccess](/images/account/subscription-success.jpg)

5. **Das Projekt neu veröffentlichen, um den Standard-Splash-Bildschirm zu entfernen**

## Abonnement kündigen {#cancel-subscription}

Standardmäßig laufen White Label-Abonnements **unbefristet und verlängern** sich automatisch in jedem Abrechnungszeitraum, bis Sie kündigen oder ein Enddatum festlegen. Nach dem Kauf können die Einstellungen für die Abonnementdauer auf der Seite Projekt-Dashboard verwaltet werden.

Die Einstellungen für die Abonnementdauer können über das Project Dashboard verwaltet werden. Die folgenden Optionen sind verfügbar:

- **Laufend**: Das Projekt läuft auf unbestimmte Zeit und Ihr Abonnement wird automatisch jeden Monat verlängert. Das Datum und die Uhrzeit der nächsten Erneuerung werden angezeigt.
- **Ende nach dem aktuellen Abrechnungszeitraum**:  Das Abonnement läuft bis zum Ende des aktuellen Abrechnungszeitraums und endet dann.
- **Ein Enddatum und eine Endzeit festlegen**: Wählen Sie ein benutzerdefiniertes Datum und eine Uhrzeit für das Ende des Abonnements.
- **Projekt jetzt beenden**: Beenden Sie das Abonnement sofort (**nur bei kundenspezifischen Plänen mit unbegrenztem White Label**).

So ändern Sie die Abonnementdauer:

1. Rufen Sie das Projekt-Dashboard Ihres Projekts auf.
2. Klicken Sie im Feld **White Label Subscription** auf **Bearbeiten**:
3. Legen Sie das gewünschte Enddatum des Abonnements fest und klicken Sie auf **Aktualisieren**, um Ihre Änderungen zu speichern:

![SubscriptionDuration](/images/account/subscription-duration.jpg)

## Reaktivierung des Abonnements {#reactivate-subscription}

Wenn ein White Label-Abonnement gekündigt wird oder ein geplantes Enddatum erreicht, wird die Abrechnung gestoppt und das Projekt wird auf der Registerkarte **Erledigte Projekte** im Projekt-Dashboard verschoben.

Um ein Projekt neu zu starten und ein neues White Label-Abonnement zu erwerben, folgen Sie bitte einer dieser Optionen:

**Von der Seite Arbeitsbereich**:

1. Wählen Sie auf der Seite Arbeitsbereich die Registerkarte **Erledigte Projekte**.
2. Suchen Sie das zu reaktivierende Projekt.
3. Klicken Sie auf das Menü "..." und wählen Sie aus dem Dropdown-Menü die Option **White-Label reaktivieren**.
4. Schließen Sie den Assistenten [White Label kaufen] (#purchase-subscription) ab, um das White Label-Abonnement zu reaktivieren.

**Aus dem Projekt-Dashboard**:

1. Wählen Sie auf der Seite Arbeitsbereich die Registerkarte **Erledigte Projekte**.
2. Wählen Sie das gewünschte Projekt aus, um zur Seite Projekt-Dashboard zu navigieren.
3. Klicken Sie auf "Jetzt starten".
4. Schließen Sie den Assistenten [White Label kaufen] (#purchase-subscription) ab, um das White Label-Abonnement zu reaktivieren.

## Projekt-URL-Umleitung {#project-redirect-url}

Wenn ein gestartetes Projekt abgebrochen oder abgeschlossen wird, möchten Sie die Benutzer möglicherweise zu einer anderen URL umleiten.

Geben Sie eine Projektumleitungs-URL an, um Ihre Benutzer automatisch auf eine andere Website umzuleiten, wenn Ihr
White Label-Abonnement beendet ist.

1. Suchen Sie im Project Dashboard den Abschnitt **Project Redirect URL**.
2. Klicken Sie auf **Eine URL festlegen**.
3. Geben Sie die gewünschte Weiterleitungs-URL ein
