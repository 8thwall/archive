# Arbeitsbereiche

:::info
Wenn Sie sich über eine Einladung eines anderen Nutzers von 8th Wall anmelden, werden Sie als Teammitglied zu dessen bestehendem Arbeitsbereich hinzugefügt. Erfahren Sie mehr [hier](/account/teams).
:::

Ein Arbeitsbereich ist eine logische Gruppierung von Projekten, Benutzern und Rechnungen. Arbeitsbereiche
können einen oder mehrere Benutzer enthalten, die jeweils unterschiedliche Berechtigungen haben. Benutzer können
zu mehreren Arbeitsbereichen gehören.

Das Arbeitsbereich-Dashboard ermöglicht Ihnen Folgendes:

- Neue Projekte erstellen.
- Verwalten Sie bestehende Projekte.
- Zugangsmodule.
- Verwalten Sie Teammitglieder und Berechtigungen für den Arbeitsbereich.
- Verwalten Sie Ihr öffentliches Profil.
- Verwalten von Abonnements.
