# Einen freien Arbeitsbereich schaffen

1. Geben Sie einen Arbeitsbereichsnamen ein. Dieser Wert dient nur zu Anzeigezwecken und hat keine Auswirkungen auf die mit Ihrem Arbeitsbereich verbundenen URLs
   .

2. Geben Sie eine Arbeitsbereichs-URL ein. Wählen Sie für den Namen Ihres Arbeitsbereichs etwas Relevantes, z. B. Die Arbeitsbereichs-URL wird als Standard-URL für **alle** von 8th Wall gehosteten
   Projekte in Ihrem Konto verwendet (z.B. **meinefirma**.8thwall.app/projekt-name). Dieser Wert wird auch in der URL Ihrer öffentlichen Profilseite
   verwendet (z. B. www.8thwall.com/mycompany).

:::warning
**Sie können Ihre Workspace-URL später nicht mehr ändern**. Wenn Sie jedoch benutzerdefinierte Domains mit Ihren von 8th Wall gehosteten Projekten verbinden möchten, um die Standard-URL
zu überschreiben, lesen Sie bitte [hier](/account/projects/connected-domains/).
:::
