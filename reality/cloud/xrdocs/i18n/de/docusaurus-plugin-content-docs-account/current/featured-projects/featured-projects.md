# Ausgewählte Projekte

Sobald Ihr [öffentliches Profil aktiviert ist](/account/public-profiles/), können Sie Seiten mit vorgestellten
Projekten auf Ihrer öffentlichen Profilseite veröffentlichen. Alle 8th Wall-Projekte können in einem öffentlichen Profil vorgestellt werden.
Alle Benutzer der Arbeitsbereiche haben die Berechtigung, besondere Projekte zu erstellen, zu speichern und zu veröffentlichen. Projekte können unter
jederzeit zu einem öffentlichen Profil hinzugefügt oder entfernt werden. Die Veröffentlichung eines vorgestellten Projekts hat keinen Einfluss auf die Live-Veröffentlichung auf
.
