# xrlefteyebrowraised

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/api/aframe/#face-effects) ausgelöst, wenn die linke Augenbraue eines verfolgten Gesichts aus ihrer Ausgangsposition angehoben wird, als das Gesicht gefunden wurde.

`xrlefteyebrowraised.detail : {id}`

| Eigentum | Typ      | Beschreibung                              |
| -------- | -------- | ----------------------------------------- |
| id       | `Nummer` | Eine numerische ID der gefundenen Fläche. |
