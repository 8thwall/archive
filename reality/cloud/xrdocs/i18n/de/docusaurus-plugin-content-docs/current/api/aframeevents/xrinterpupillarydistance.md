# xrinterpupillarydistance

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/api/aframe/#face-effects) ausgelöst, wenn der Abstand in Millimetern zwischen den Mittelpunkten der Pupillen eines verfolgten Gesichts zum ersten Mal erkannt wird.

`xrinterpupillarydistance.detail : {id}`

| Eigentum               | Typ      | Beschreibung                                                                         |
| ---------------------- | -------- | ------------------------------------------------------------------------------------ |
| id                     | `Nummer` | Eine numerische ID der gefundenen Fläche.                                            |
| interpupillaryDistance | `Nummer` | Ungefährer Abstand in Millimetern zwischen den Mittelpunkten der einzelnen Pupillen. |
