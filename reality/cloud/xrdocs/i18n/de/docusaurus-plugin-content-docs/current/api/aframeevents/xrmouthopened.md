# xrmouthopened

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/api/aframe/#face-effects) ausgelöst, wenn sich der Mund eines verfolgten Gesichts öffnet.

`xrmouthopened.detail : {id}`

| Eigentum | Typ      | Beschreibung                              |
| -------- | -------- | ----------------------------------------- |
| id       | `Nummer` | Eine numerische ID der gefundenen Fläche. |
