# AccessPass

## Beschreibung {#description}

Bietet ein Modul zur Monetarisierung Ihrer WebAR- und WebVR-Erfahrung. Dies ist nur für 8th Wall Hosted Projekte verfügbar und erfordert das [Zahlungsmodul](/guides/advanced-topics/monetize-with-8th-wall-payments).

```ts
importieren Sie {AccessPass} von 'payments'
```

## Funktionen {#functions}

| Funktion                                              | Beschreibung                                                                                                         |
| ----------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| [requestPurchaseIfNeeded](requestpurchaseifneeded.md) | Öffnet ein Kassenfenster, in dem der Kunde eine sichere Zahlung für den bereitgestellten Zugangspass vornehmen kann. |
