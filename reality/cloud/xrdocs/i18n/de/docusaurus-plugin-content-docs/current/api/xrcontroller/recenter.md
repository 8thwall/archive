# XR8.XrController.recenter()

`XR8.XrController.recenter()`

## Beschreibung {#description}

Positioniert die Kamera neu auf den Ursprung / die Blickrichtung, die durch [`angegeben wurde. XR8.XrController.updateCameraProjectionMatrix()`](updatecameraprojectionmatrix.md) und startet die Verfolgung neu.

## Parameter {#parameters}

Keine

## Returns {#returns}

Keine
