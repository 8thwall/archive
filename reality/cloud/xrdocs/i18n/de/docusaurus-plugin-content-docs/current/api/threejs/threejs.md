# XR8.Threejs

## Beschreibung {#description}

Stellt ein Kamera-Pipeline-Modul zur Verfügung, das die three.js-Kamera ansteuert, um virtuelle Overlays zu erstellen.

## Funktionen {#functions}

| Funktion                            | Beschreibung                                                                                                                        |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| [pipelineModule](pipelinemodule.md) | Ein Pipeline-Modul, das mit der three.js Umgebung und dem Lebenszyklus zusammenarbeitet.                                            |
| [configure](configure.md)           | Konfiguriert den three.js Renderer.                                                                                                 |
| [xrScene](xrscene.md)               | Holen Sie sich ein Handle auf die xr-Szene, die Kamera, den Renderer, die (optionale) Kameratextur und die (optionale) layerScenes. |
