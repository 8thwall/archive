# LandingPage

## Beschreibung {#description}

Stellt ein Modul zur Verfügung, das eine benutzerdefinierte Landing Page für Ihr WebAR-Erlebnis erzeugt.

## Funktionen {#functions}

| Funktion                            | Beschreibung                                                                                                                   |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| [configure](configure.md)           | Konfiguriert die Einstellungen der LandingPage.                                                                                |
| [pipelineModule](pipelinemodule.md) | Erstellt ein Kamera-Pipeline-Modul, das, wenn es installiert wird, die Landing Page-Funktionalität zu Ihrem Projekt hinzufügt. |
