# XR8.LayersController.getLayerNames()

`XR8.LayersController.getLayerNames()`

## Beschreibung {#description}

Gibt die Ebenen zurück, die durch den `LayersController` konfiguriert wurden.

## Parameter {#parameters}

Keine

## Returns {#returns}

Eine Liste von Strings, von denen jeder ein Ebenenname ist.
