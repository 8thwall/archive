# PlayCanvas-Ereignisbehandlungsroutinen

Dieser Abschnitt beschreibt die Ereignisse, die von 8th Wall Web in einer PlayCanvas-Umgebung abgehört werden.

Sie können diese Ereignisse in Ihrer Webanwendung auslösen, um verschiedene Aktionen durchzuführen:

| Ereignis-Listener                              | Beschreibung                                                                                                                                                                                                                                                                                                                     |
| ---------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [xr:hidecamerafeed](xrhidecamerafeed.md)       | Blendet die Kamerafeed aus. Die Verfolgung hört nicht auf.                                                                                                                                                                                                                                                                       |
| [xr:recenter](xrrecenter.md)                   | Bringt die Kamerafeed zu ihrem Ursprung zurück. Wenn ein neuer Ursprung als Argument angegeben wird, wird der Ursprung der Kamera auf diesen zurückgesetzt und dann neu zentriert.                                                                                                                                               |
| [xr:screenshotrequest](xrscreenshotrequest.md) | Sendet eine Anfrage an die Engine, um einen Screenshot der PlayCanvas-Leinwand aufzunehmen. Die Engine sendet ein [`xr:screenshotready`](/api/playcanvasevents/xrscreenshotready) Ereignis mit dem JPEG-komprimierten Bild oder [`xr:screenshoterror`](/api/playcanvasevents/xrscreenshoterror) wenn ein Fehler aufgetreten ist. |
| [xr:showcamerafeed](xrshowcamerafeed.md)       | Zeigt die Kamerafeed an.                                                                                                                                                                                                                                                                                                         |
| [xr:stopxr](xrstopxr.md)                       | Beendet die aktuelle XR-Sitzung. Wenn die Kamera angehalten ist, wird die Übertragung gestoppt und die Bewegung des Geräts wird nicht verfolgt.                                                                                                                                                                                  |
