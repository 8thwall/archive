# XR8.PlayCanvas

PlayCanvas (<https://www.playcanvas.com/>) ist eine Open-Source-3D-Spiel-Engine/interaktive 3D-Anwendungs-Engine neben einer proprietären, in der Cloud gehosteten Erstellungsplattform, die eine gleichzeitige Bearbeitung von mehreren Computern aus über eine browserbasierte Schnittstelle ermöglicht.

## Beschreibung {#description}

Bietet eine Integration, die mit der PlayCanvas-Umgebung und dem Lebenszyklus zusammenarbeitet, um die PlayCanvas-Kamera für virtuelle Überlagerungen zu steuern.

## Funktionen {#functions}

| Funktion                                         | Beschreibung                                                                                                 |
| ------------------------------------------------ | ------------------------------------------------------------------------------------------------------------ |
| [laufen](run.md)                                 | Öffnet die Kamera mit den angegebenen Pipeline-Modulen und startet die Ausführung in einer PlayCanvas-Szene. |
| [runXr (veraltet)](runxr.md)                     | Öffnet die Kamera und startet das World Tracking und/oder Image Tracking in einer PlayCanvas-Szene.          |
| [runFaceEffects (veraltet)](runfaceeffects.md)   | Öffnet die Kamera und startet die Face Effects in einer PlayCanvas-Szene.                                    |
| [stop](stop.md)                                  | Entfernen Sie die in [hinzugefügten Module, führen Sie](run.md) aus und stoppen Sie die Kamera.              |
| [stopXr (veraltet)](stopxr.md)                   | Entfernen Sie die in [runXr](runxr.md) hinzugefügten Module und stoppen Sie die Kamera.                      |
| [stopFaceEffects (veraltet)](stopfaceeffects.md) | Entfernen Sie die in [runFaceEffects](runfaceeffects.md) hinzugefügten Module und stoppen Sie die Kamera.    |
