# XR8.XrConfig.camera()

Aufzählung

## Beschreibung {#description}

Zu verwendende Kamera.

## Eigenschaften {#properties}

| Eigentum | Wert            | Beschreibung                                 |
| -------- | --------------- | -------------------------------------------- |
| FRONT    | `'Vorderseite'` | Verwenden Sie die Frontkamera/Selfie-Kamera. |
| ZURÜCK   | `'zurück'`      | Verwenden Sie die Kamera auf der Rückseite.  |
