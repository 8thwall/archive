# XR8.FaceController.MeshGeometry

Aufzählung

## Beschreibung {#description}

Optionen, um festzulegen, für welche Teile der Fläche Mesh-Dreiecke zurückgegeben werden sollen.

## Eigenschaften {#properties}

| Eigentum | Wert        | Beschreibung                                    |
| -------- | ----------- | ----------------------------------------------- |
| FACE     | `'Gesicht'` | Geben Sie die Geometrie für das Gesicht zurück. |
| MOUTH    | `'Mund'`    | Return-Geometrie für den Mund.                  |
| AUGEN    | `'Augen'`   | Rückkehrgeometrie für die Augen.                |
| IRIS     | `'Iris'`    | Rückkehrgeometrie für die Iris.                 |
