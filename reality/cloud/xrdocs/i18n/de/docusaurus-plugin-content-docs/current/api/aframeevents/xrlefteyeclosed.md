# xrlefteyeclosed

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/api/aframe/#face-effects) ausgelöst, wenn sich das linke Auge eines verfolgten Gesichts schließt.

`xrlefteyeclosed.detail : {id}`

| Eigentum | Typ      | Beschreibung                              |
| -------- | -------- | ----------------------------------------- |
| id       | `Nummer` | Eine numerische ID der gefundenen Fläche. |
