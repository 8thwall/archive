# XR8.stop()

`XR8.stop()`

## Beschreibung {#description}

Wenn die Kamera angehalten ist, ist die Kamerafeed geschlossen und die Bewegung des Geräts wird nicht verfolgt. Muss [`XR8.run()`](run.md) aufrufen, um nach dem Stoppen des Motors neu zu starten.

## Parameter {#parameters}

Keine

## Returns {#returns}

Keine

## Beispiel {#example}

```javascript
XR8.stop()
```
