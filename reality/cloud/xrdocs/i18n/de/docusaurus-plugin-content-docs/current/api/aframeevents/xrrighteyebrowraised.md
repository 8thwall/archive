# xrrighteyebrowraised

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/api/aframe/#face-effects) ausgelöst, wenn die rechte Augenbraue eines verfolgten Gesichts aus der Position angehoben wird, in der das Gesicht gefunden wurde.

`xrrighteyebrowraised.detail : {id}`

| Eigentum | Typ      | Beschreibung                              |
| -------- | -------- | ----------------------------------------- |
| id       | `Nummer` | Eine numerische ID der gefundenen Fläche. |
