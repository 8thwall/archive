# XR8.MediaRecorder

## Beschreibung {#description}

Bietet ein Kamera-Pipeline-Modul, mit dem Sie ein Video im MP4-Format aufnehmen können.

## Funktionen {#functions}

| Funktion                                  | Beschreibung                                                                                                    |
| ----------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)                 | Konfigurieren Sie die Einstellungen für die Videoaufnahme.                                                      |
| [pipelineModule](pipelinemodule.md)       | Erzeugt ein Pipeline-Modul, das Videos im MP4-Format aufnimmt.                                                  |
| [recordVideo](recordvideo.md)             | Starten Sie die Aufnahme.                                                                                       |
| [requestMicrophone](requestmicrophone.md) | Ermöglicht die Aufnahme von Audio (falls nicht automatisch aktiviert) und fordert bei Bedarf Berechtigungen an. |
| [stopRecording](stoprecording.md)         | Stoppen Sie die Aufnahme.                                                                                       |
| [RequestMicOptions](requestmicoptions.md) | Enum, um festzulegen, ob automatisch Mikrofonberechtigungen angefordert werden sollen oder nicht.               |
