## XR8.XrConfig {#xr8xrconfig}

## Beschreibung {#description}

Hilfsprogramme zum Festlegen der Geräte- und Kameraklassen, auf denen Pipeline-Module laufen sollen.

## Eigenschaften {#properties}

| Eigentum              | Typ        | Beschreibung                                                            |
| --------------------- | ---------- | ----------------------------------------------------------------------- |
| [camera()](camera.md) | Aufzählung | Zu verwendende Kamera.                                                  |
| [device()](device.md) | Aufzählung | Geben Sie die Klasse der Geräte an, auf denen die Pipeline laufen soll. |
