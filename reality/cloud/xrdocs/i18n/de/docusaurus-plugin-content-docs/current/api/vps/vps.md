# XR8.Vps

## Beschreibung {#description}

Dienstprogramme, um mit VPS-Diensten zu sprechen.

## Funktionen {#functions}

| Funktion                                    | Beschreibung                                                                                                              |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| [makeWayspotWatcher](makewayspotwatcher.md) | Erstellen Sie einen Watcher, der nach allen Standorten sucht, nicht nur nach Projektstandorten.                           |
| [projectWayspots](projectwayspots.md)       | Gibt ein Versprechen mit einem Array von `ClientWayspotInfo` zurück, das Daten über jeden Ihrer Projektstandorte enthält. |
