# xrlefteyebrowlowered

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/api/aframe/#face-effects) ausgelöst, wenn die linke Augenbraue eines verfolgten Gesichts in ihre ursprüngliche Position gesenkt wird, als das Gesicht gefunden wurde.

`xrlefteyebrowlowered.detail : {id}`

| Eigentum | Typ      | Beschreibung                              |
| -------- | -------- | ----------------------------------------- |
| id       | `Nummer` | Eine numerische ID der gefundenen Fläche. |
