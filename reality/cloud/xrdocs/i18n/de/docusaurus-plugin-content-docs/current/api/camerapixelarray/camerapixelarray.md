# XR8.CameraPixelArray

## Beschreibung {#description}

Bietet ein Kamera-Pipeline-Modul, das den Zugriff auf Kameradaten als Graustufen- oder Farb-uint8-Array ermöglicht.

## Funktionen {#functions}

| Funktion                            | Beschreibung                                                                                                                                                     |
| ----------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [pipelineModule](pipelinemodule.md) | Ein Pipeline-Modul, das die Kameratextur als Array von RGBA- oder Graustufen-Pixelwerten bereitstellt, die für die CPU-Bildverarbeitung verwendet werden können. |
