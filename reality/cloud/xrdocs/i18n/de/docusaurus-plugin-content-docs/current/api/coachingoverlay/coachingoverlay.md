# CoachingOverlay

## Beschreibung {#description}

Stellt ein Modul zur Verfügung, das ein Coaching-Overlay für Ihr Absolute Scale WebAR-Erlebnis erzeugt.

Informationen über das Lightship VPS Coaching Overlay finden Sie unter [hier](/api/vpscoachingoverlay).

## Funktionen {#functions}

| Funktion                            | Beschreibung                                                                                                                      |
| ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| [configure](configure.md)           | Konfiguriert die Einstellungen für das Coaching-Overlay.                                                                          |
| [pipelineModule](pipelinemodule.md) | Erstellt ein Kamera-Pipeline-Modul, das, wenn es installiert ist, die Coaching-Overlay-Funktionalität zu Ihrem Projekt hinzufügt. |
