# xrrighteyebrowlowered

## Beschreibung {#description}

Dieses Ereignis wird von [`xrface`](/api/aframe/#face-effects) ausgelöst, wenn die rechte Augenbraue eines verfolgten Gesichts in ihre Ausgangsposition gesenkt wird, als das Gesicht gefunden wurde

`xrrighteyebrowlowered.detail : {id}`

| Eigentum | Typ      | Beschreibung                              |
| -------- | -------- | ----------------------------------------- |
| id       | `Nummer` | Eine numerische ID der gefundenen Fläche. |
