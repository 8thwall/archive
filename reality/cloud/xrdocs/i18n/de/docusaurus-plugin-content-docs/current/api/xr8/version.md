# XR8.version()

`XR8.version()`

## Beschreibung {#description}

Holen Sie sich die Version der 8th Wall Web Engine.

## Parameter {#parameters}

Keine

## Returns {#returns}

Eine String, der die Version der 8th Wall Web Engine angibt.

## Beispiel {#example}

```javascript
console.log(XR8.version())
```
