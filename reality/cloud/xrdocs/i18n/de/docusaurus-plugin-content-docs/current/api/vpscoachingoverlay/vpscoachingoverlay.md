# VpsCoachingOverlay

## Beschreibung {#description}

Stellt ein Modul zur Verfügung, das ein Coaching-Overlay für Ihr Lightship VPS-fähiges WebAR-Erlebnis erzeugt.

Informationen über das Absolute Scale Coaching Overlay finden Sie unter [hier](/api/coachingoverlay).

## Funktionen {#functions}

| Funktion                                                | Beschreibung                                                                                                                      |
| ------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| [configure](vps-coachingoverlay-configure.md)           | Konfiguriert die Einstellungen für das Coaching-Overlay.                                                                          |
| [pipelineModule](vps-coachingoverlay-pipelinemodule.md) | Erstellt ein Kamera-Pipeline-Modul, das, wenn es installiert ist, die Coaching-Overlay-Funktionalität zu Ihrem Projekt hinzufügt. |
