# XR8.isInitialized()

`XR8.isInitialized()`

## Beschreibung {#description}

Zeigt an, ob die WebAssembly der AR Engine initialisiert ist oder nicht.

## Parameter {#parameters}

Keine

## Returns {#returns}

Ein Boolescher Wert, der angibt, ob die WebAssembly der AR Engine initialisiert ist oder nicht.

## Beispiel {#example}

```javascript
if (XR8.isInitialized()) {
  console.log(XR8.version())
}
```
