# XR8.FaceController.AttachmentPoints

Aufzählung

## Beschreibung {#description}

Punkte des Gesichts, an denen Sie Inhalte verankern können.

## Eigenschaften {#properties}

| Eigenschaft              | Wert                          | Beschreibung                       |
| ------------------------ | ----------------------------- | ---------------------------------- |
| STIRN                    | `‚Stirn‘`                     | Stirn                              |
| RIGHT_EYEBROW_INNER    | `‚rightEyebrowInner‘`         | Innenseite der rechten Augenbraue  |

| RIGHT_EYEBROW_OUTER    | `‚rightEyebrowOuter‘`         | Außenseite der rechten Augenbraue  |

| LEFT_EYEBROW_MIDDLE    | `‚leftEyebrowMiddle‘`         | Mitte der linken Augenbraue        |
| LEFT_EYEBROW_OUTER     | `‚leftEyebrowOuter‘`          | Außenseite der linken Augenbraue   |
| LEFT_EAR                 | `‚leftEar‘`                   | Linkes Ohr                         |
| RIGHT_EAR                | `‚rightEar‘`                  | Rechtes Ohr                        |
| LEFT_CHEEK               | `‚leftCheek‘`                 | Linke Wange                        |
