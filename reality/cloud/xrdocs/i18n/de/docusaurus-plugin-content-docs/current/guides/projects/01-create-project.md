# Projekt erstellen

1. Klicken Sie auf der Startseite (eingeloggt) oder im Workspace Dashboard auf "Neues Projekt starten"

2. Wählen Sie den Arbeitsbereich für dieses Projekt.

3. Geben Sie Basisinformationen für das Projekt ein: Bitte geben Sie an: Titel, URL, Beschreibung (optional) und Titelbild (optional).  Alle diese Felder, **außer URL**, können später auf der Seite Projekteinstellungen bearbeitet werden.

4. Wählen Sie einen Projekttyp

## Projekttyp {#project-types}

**Alle Projekte** müssen das [Powered by 8th Wall](https://drive.google.com/drive/folders/1c9d23c5hS_HspHTUD7VceV6ocqdbPN7J?usp=sharing) Abzeichen auf der Ladeseite anzeigen. Es ist standardmäßig im `Lademodul` enthalten und kann nicht entfernt werden. Anweisungen zum Anpassen des Ladebildschirms finden Sie unter [hier](/guides/advanced-topics/load-screen/) .

* **Kommerziell**: Kommerzielle Projekte sind für kommerzielle Nutzung bestimmt. Wenn Sie bereit sind, ein kommerzielles Projekt zu starten, damit die Welt es sehen kann, müssen Sie eine kommerzielle Lizenz erwerben. Informationen zu den kommerziellen Lizenzoptionen finden Sie auf der [Preisseite](https://www.8thwall.com/pricing#commercial-license) unter . HINWEIS: Um eine kommerzielle Lizenz zu erwerben, muss der Arbeitsbereich zunächst über einen aktiven Pro-Plan verfügen.

* **Nicht-kommerziell**: Mit Ihrem bezahlten Abonnement können Sie unbegrenzt nicht-kommerzielle Projekte entwickeln und veröffentlichen. Auf dem Ladebildschirm erscheint der Hinweis „Nicht-kommerziell“. Wenn Sie sich zu irgendeinem Zeitpunkt entscheiden, Ihr Projekt kommerziell zu nutzen, wechseln Sie im Projekt-Dashboard zu „Kommerziell“.

* **Demo-Nutzung**: Sie können eine unbegrenzte Anzahl von Demo-Projekten erstellen, die öffentlich einsehbar sind und ausschließlich für die Präsentation potenzieller Arbeiten dienen. Auf dem Ladebildschirm erscheint der Hinweis „Nur für Demozwecke“. Wenn Sie sich zu irgendeinemZeitpunkt entscheiden, Ihr Projekt kommerziell zu nutzen, wechseln Sie im Projekt-Dashboard zu „Kommerziell“.

* **Verwendung für Bildungszwecke**: Bildungsprojekte sind nur für Bildungszwecke gedacht, z.B. in einem Klassenzimmer. Sie können unbegrenzt viele Bildungsprojekte entwickeln und veröffentlichen. Auf dem Ladebildschirm erscheint der Hinweis "Verwendung für Bildungszwecke". Wenn Ihr Projekt für die kommerzielle Nutzung bestimmt ist, müssen Sie „Kommerziell“ auswählen. Wenn Sie eine Bildungseinrichtung sind, wenden Sie sich bitte an 8th Wall, um unter Informationen über einen maßgeschneiderten Bildungsplan zu erhalten.

## Projekttyp ändern {#change-project-type}

1. Rufen Sie die Seite Projekt-Dashboard auf
2. Klicken Sie auf "Projekttyp ändern"
3. Wählen Sie einen geeigneten Projekttyp
4. Klicken Sie auf "Speichern"
