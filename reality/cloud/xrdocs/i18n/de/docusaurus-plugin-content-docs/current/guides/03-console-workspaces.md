# Arbeitsbereiche

Ein Arbeitsbereich ist eine logische Gruppierung von Projekten, Benutzern und Rechnungsstellung. Arbeitsbereiche
können einen oder mehrere Benutzer enthalten, die jeweils unterschiedliche Berechtigungen haben. Benutzer können
zu mehreren Arbeitsbereichen gehören.

Das Arbeitsbereich-Dashboard ermöglicht Ihnen:

- Neue Projekte zu erstellen.
- Bestehende Projekte zu verwalten.
- Teammitglieder und Berechtigungen für den Arbeitsbereich zu verwalten.
- Abonnements und kommerzielle Lizenzen zu verwalten.

## Ursprünglicher Arbeitsbereich {#initial-workspace}

Wenn Sie einen neuen 8th Wall Account direkt auf 8thwall.com erstellen, starten Sie
mit einem Arbeitsbereich im Basic (free) Plan.

Wenn Sie sich über eine Einladung eines anderen 8th Wall-Nutzers anmelden, werden Sie als
ein Teammitglied von dessen bestehendem Arbeitsbereich hinzugefügt.

**Wichtig: Wählen Sie den Namen Ihres Arbeitsbereichs sorgfältig aus, da dieser später nicht mehr geändert werden kann.** Standardmäßig wird die URL
Ihres veröffentlichten Web AR-Erlebnisses im Format von:

"arbeitsbereich-name.8thwall.app/projekt-name"

Um die URL des Arbeitsbereichs zu ändern, müssen Sie den Support kontaktieren und eine Anfrage erstellen.
