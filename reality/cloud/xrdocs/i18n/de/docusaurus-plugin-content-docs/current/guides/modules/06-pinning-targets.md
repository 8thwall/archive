# Ziele anheften {#pinning-targets}
Wenn Sie ein Modul aus Ihrem eigenen Arbeitsbereich importieren, sehen Sie mehrere Optionen für das Anheftungsziel:

**Version**: Wenn Sie ein "Version"-Anheftungsziel auswählen, können Sie Ihr importiertes Modul für Bugfix-Updates und kleinere Updates abonnieren oder automatische Modul-Updates deaktivieren. Sie sehen diese Option nur, wenn das Modul mindestens einmal bereitgestellt wurde.

**Commit**: ermöglicht Ihnen, das Modul an einen beliebigen Commit des Modulcodes zu binden.

**Entwicklung**: Ermöglicht Ihnen die Verwendung von Modulcode, den Sie in Ihrem lokalen Client haben und der noch nicht gelandet ist. Wird für Entwicklung und Tests verwendet.
