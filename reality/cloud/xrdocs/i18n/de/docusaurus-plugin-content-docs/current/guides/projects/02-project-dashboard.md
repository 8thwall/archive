# Projekt-Dashboard

Das Projekt-Dashboard ist Ihre Zentrale für die Verwaltung von 8th Wall-Projekten.  Auf der Projekt-Dashboard-Seite können Sie Projekteinstellungen verwalten, auf den 8th Wall Cloud Editor zugreifen, kommerzielle Lizenzen erwerben, Bildziele verwalten, benutzerdefinierte Domains einrichten und vieles mehr.

Die direkte URL zu Ihrem Project Dashboard hat das folgende Format: www.8thwall.com/workspacename/projectname

## Verwalten von Bildzielen {#managing-image-targets}

Um Bildziele für ein bestimmtes Projekt zu verwalten, klicken Sie entweder auf das Symbol Bildziel in der linken Navigation oder auf den Link "Bildziele verwalten" im Projekt-Dashboard.

![ManageImageTargets](/images/console-appkey-imagetargets.jpg)

Ausführliche Informationen zu Image Targets finden Sie in der Dokumentation [Image Target](/guides/image-targets).

## Lightship VPS-Standorte verwalten {#managing-lightship-vps-locations}

Um die Lightship VPS-Standorte für ein bestimmtes Projekt zu verwalten, klicken Sie auf das Kartensymbol in der linken Navigation.

![ManageWayspots](/images/console-wayspot-left-nav.jpg)

Ausführliche Informationen zu Lightship VPS für Web finden Sie in der Dokumentation [Lightship VPS für Web](/guides/lightship-vps).
