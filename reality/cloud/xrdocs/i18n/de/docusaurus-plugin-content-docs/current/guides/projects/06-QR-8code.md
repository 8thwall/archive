# QR 8Code

Zur Vereinfachung können für ein Projekt QR-Codes mit der Marke 8th Wall (auch "8 Codes" genannt) generiert werden, so dass einfach von einem mobilen Gerät aus gescannt werden kann, um auf Ihr WebAR-Projekt zuzugreifen.  Sie können jederzeit Ihre eigenen QR-Codes erstellen oder Websites oder Dienste von Drittanbietern zur Erstellung von QR-Codes verwenden.

Der QR-Code auf dem Projekt-Dashboard verweist auf einen eindeutigen "8th.io" Shortlink für Ihr Projekt. Dieser Shortlink leitet den Benutzer dann auf die URL Ihres WebAR-Erlebnisses weiter.

<u>Sowohl der QR-Code als auch der "8th.io"-Code für ein bestimmtes Projekt sind statisch und ändern sich nicht je nach Projekttyp oder Lizenz.</u>

### 8th Wall Hosted Projects (KEINE verbundene Domain) {#8th-wall-hosted-projects-no-connected-domain}

Wenn Ihr Projekt die Standard-URL von 8th Wall verwendet (im Format "**workspace-name**.8thwall.app/**project-name**"), werden der QR-Code und der 8th.io Shortlink immer auf die Standard-URL umgeleitet.  Es ist nicht möglich, die Ziel-URL zu ändern.

![ProjektDashboard8wHostedQR](/images/console-appkey-qrcode-8w-hosted.png)

### 8th Wall Gehostete Projekte (MIT verbundener Domain) {#8th-wall-hosted-projects-with-connected-domain}

Wenn Sie für Ihr von 8th Wall gehostetes Projekt eine [verbundene Domain](/guides/projects/connected-domains) konfiguriert haben, haben Sie die Möglichkeit, als Ziel für den QR-Code / Shortlink entweder die Standard-URL des Projekts oder die primäre verbundene Domain festzulegen.

Verwenden Sie die Optionsschaltfläche, um das Ziel Ihres QR-Codes / Shortlinks festzulegen:

![ProjektDashboard8wHostedQRConnectedDomain](/images/console-appkey-qrcode-8w-hosted-connected-domain.png)

### Selbst gehostete Projekte {#self-hosted-projects}

Um einen QR-Code und einen Shortlink zu erzeugen, geben Sie die vollständige URL zu Ihrem selbst gehosteten Projekt ein und klicken Sie auf **Speichern**:

![ProjektDashboardSelfHostedQR](/images/console-appkey-qrcode.png)

Der generierte QR-Code kann entweder im PNG- oder SVG-Format heruntergeladen werden, um ihn auf einer Website, auf oder an anderen Orten anzubringen, damit die Benutzer ihn einfach mit ihrem Smartphone scannen können, um die selbst gehostete URL zu besuchen.  Klicken Sie auf das Bleistiftsymbol, um das Ziel des Shortlinks zu bearbeiten, falls sich die selbst gehostete URL in Zukunft ändern sollte.

Beispiel:
