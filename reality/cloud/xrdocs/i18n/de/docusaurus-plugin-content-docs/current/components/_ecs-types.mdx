Die folgenden Datentypen sind nützlich für die Erstellung von Schemaeigenschaften für eine benutzerdefinierte Komponente oder Verweise auf einen bestimmten Typ.

| Typ         | Beschreibung                         |
| ----------- | ------------------------------------ |
| ecs.eid     | Referenz der Entität                 |
| ecs.f32     | 32-Bit-Gleitkommazahl                |
| ecs.f64     | 64-Bit-Gleitkommazahl                |
| ecs.i32     | 32-Bit-Ganzzahl                      |
| ecs.ui8     | 8-Bit-Ganzzahl ohne Vorzeichen       |
| ecs.ui32    | 32-Bit-Ganzzahl ohne Vorzeichen      |
| ecs.string  | Zeichenfolge                         |
| ecs.boolean | Boolesche                            |
