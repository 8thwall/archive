# Was gibt's Neues

8th Wall **Release 27.2** ist jetzt verfügbar! Diese Version bietet eine Reihe von Aktualisierungen und Verbesserungen:

#### Version 27.2: (2024-Dezember-04, v27.2.6.427 / 2024-November-04, v27.2.5.427 / 2024-Oktober-23, v27.2.4.427) {#whatsnew-release-27-2024-october-23-2724427}

- Neue Funktionen:
  - VPS-Kompatibilität für Niantic Studio-Projekte wurde hinzugefügt.

- Korrekturen und Erweiterungen:
  - Es wurde ein Problem behoben, das die Zuverlässigkeit des Simulators in VPS-Projekten beeinträchtigte. (27.2.5.427)
  - Verbesserte Zuverlässigkeit der Kamera-Pipeline-Initialisierung für verbesserte AR-Erlebnisse. (27.2.6.427)
