# XR8.initialize()

`XR8.initialize()`

## Beschreibung {#description}

Gibt ein Versprechen zurück, das erfüllt wird, wenn die WebAssembly der AR Engine initialisiert wird.

## Parameter {#parameters}

Keine

## Rückgabe {#returns}

Keine

## Beispiel {#example}

```javascript
XR8.initialize().then(() => console.log(XR8.version())
```
