# XR8.removeCameraPipelineModule()

XR8.removeCameraPipelineModule(Modulname)\\`

## Beschreibung {#description}

Entfernt ein Modul aus der Kamera-Pipeline.

## Parameter {#parameters}

| Parameter | Typ      | Beschreibung                           |
| --------- | -------- | -------------------------------------- |
| Modulname | `String` | Der Name eines Moduls. |

## Rückgabe {#returns}

Keine

## Beispiel {#example}

```javascript
XR8.removeCameraPipelineModule('Realität')
```
