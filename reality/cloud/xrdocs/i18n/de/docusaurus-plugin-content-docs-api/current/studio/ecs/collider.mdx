# Collider

## Beschreibung

Diese Komponente erstellt einen Physik-Collider für eine Entität.

## ColliderShape

| Typ         | Beschreibung   |
| ----------- | -------------- |
| Box         | Box            |
| Sphäre      | Sphäre         |
| Flugzeug    | Flugzeug       |
| Kapsel      | Kapsel         |
| Kegel       | Kegel          |
| Zylinder    | Zylinder       |

## ColliderType

| Typ         | Beschreibung                                                                                                                                                                                                                                                                           |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Statisch    | Gilt für ein Objekt, das sich bei der Simulation nicht bewegt und sich so verhält, als hätte es eine unendliche Masse                                                                                                                                                                  |
| Dynamisch   | Gilt für ein vollständig simuliertes, reales Objekt, das von Kräften, Schwerkraft und Kollisionen betroffen ist.                                                                                                                                                                       |
| Kinematisch | Gilt für ein Objekt, dessen Bewegung nicht von Kräften beeinflusst wird, sondern vollständig vom Benutzer programmatisch gesteuert wird. Im Gegensatz zu statischen Körpern können sich kinematische Körper bewegen und andere dynamische Körper durch Kollisionen beeinflussen.       |
