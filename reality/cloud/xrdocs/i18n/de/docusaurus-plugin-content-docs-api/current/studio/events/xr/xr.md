# XR-Ereignisse

XR-Ereignisse werden von der 8th Wall-Engine über Kamera-Pipeline-Module ausgesendet. Zu den Kamera-Pipeline-Modulen, die Ereignisse ausgeben, gehören: Realität" und "Facecontroller".

## Realität {#reality}

Ereignisse, die vom Modul "Realität" ausgegeben werden:

- Bildziel Ereignisse
- VPS-Ereignisse
- Welt-Effekt-Ereignisse

## facecontroller {#facecontroller}

Ereignisse, die vom Modul "Facecontroller" ausgegeben werden:

- Face Events
