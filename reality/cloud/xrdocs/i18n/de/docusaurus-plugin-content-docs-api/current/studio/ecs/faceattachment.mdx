# FaceAttachment

## Beschreibung

Diese Komponente bewirkt, dass die Transformation des Entitys der angegebenen, in XR verfolgten Flächenbefestigung folgt. Die Entität wird ein- und ausgeblendet, wenn das Gesicht gefunden oder verloren wird.
