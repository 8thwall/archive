# Das öffentliche Projekt ansehen

1. Gehen Sie zurück zum Projekt-Dashboard in der linken Navigation.  Im Abschnitt **QR 8.code** wird die öffentliche Projekt-URL
   zusammen mit einem **8th.io** Shortlink und einem zugehörigen QR-Code angezeigt.

2. Scannen Sie den QR-Code mit Ihrem Mobilgerät, um das öffentliche Projekt zu sehen.
