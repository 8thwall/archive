# Projekte exportieren

## Code-Ausfuhr

Der Code-Export ist jetzt sowohl für Studio- als auch für Cloud-Editor-Projekte verfügbar. So können Sie einen vollständigen Schnappschuss Ihres Projekts herunterladen, genau so, wie es heute auf der 8th Wall existiert. Mit dieser Version können Sie **Code, Assets und Bildziele mit einem Klick herunterladen**.

Code Export wurde entwickelt, um Ihnen bei der Umstellung auf Open Source sofortigen Zugriff und Backups Ihrer Arbeit zu ermöglichen oder Sie bei der Evaluierung anderer Plattformen zu unterstützen.

:::warning[Note]
Diese Version erzeugt kein baubares Projekt. Der exportierte Code kann nicht ausgeführt oder bereitgestellt werden. Ein bauwürdiger Export ist in Kürze möglich.
:::

### Studio

So laden Sie eine Kopie eines Studio-Projekts herunter:

1. Öffnen Sie das Studio-Projekt und klicken Sie auf **Veröffentlichen**.

![](/images/migration/studio-publish.png)

2. Wählen Sie **Code** unter dem Abschnitt **Exportieren**, und klicken Sie dann auf die Schaltfläche **Exportieren**.

![](/images/migration/studio-export-code.png)

3. Sie erhalten eine .zip-Datei mit Bildzielen, Code und Dateien.

![](/images/migration/studio-files.png)

### Cloud-Editor

Um eine Kopie eines Cloud Editor-Projekts herunterzuladen:

1. Öffnen Sie das Editor-Projekt und klicken Sie auf **Veröffentlichen**.

![](/images/migration/editor-publish.jpg)

2. Wählen Sie **Code** unter dem Abschnitt **Exportieren**, und klicken Sie dann auf die Schaltfläche **Exportieren**.

![](/images/migration/editor-export-code.png)

3. Sie erhalten eine .zip-Datei mit Bildzielen, Code und Dateien.

![](/images/migration/editor-files.png)

## Buildable Code Export

In ein paar Wochen werden wir Buildable Code Export veröffentlichen. Diese Version enthält alles, was zum Erstellen und Ausführen einer Offline-Version Ihres Projekts erforderlich ist, einschließlich:

* Projektcode und Vermögenswerte
* Bildziele
* Konfiguration erstellen
* XR-Motor
* XR-Extras
* Landing Page

Die Offline-Entwicklung unterstützt jedoch **keine** Cloud-abhängigen Funktionen oder solche, für die wir keine Lizenz haben, wie z. B.:

* VPS / Karten
* Aktiviert die Handverfolgung
* Module / Backends
