# ホワイトラベル購読

## 定期購入 {#purchase-subscription}

:::info
デフォルトでは、White Labelサブスクリプションは**無期限に実行され、お客様がキャンセルするか終了日をスケジュールするまで、請求期間ごとに**自動的に更新されます。 購入後、購読期間の設定はプロジェクトダッシュボードのページから管理できます。
:::

1. プロジェクト・ダッシュボード・ページへ
2. 今すぐ購入」をクリック

![PurchaseSubscription](/images/account/subscription-purchase.jpg)

3. ホワイトレーベル契約を確認し、同意する

![SubscriptionAgreement](/images/account/subscription-agreement.jpg)

2. 請求期間を選択します。 請求期間を選択します。 以下のオプションから選択できる：
   - **月刊**：購読は毎月自動的に更新されます。
   - **年間**：購読は12ヶ月ごとに自動更新されます。

![SubscriptionBillingPeriod](/images/account/subscription-billing-period.jpg)

3. お支払い方法を選択してください。

![SubscriptionPaymentMethod](/images/account/subscription-payment-method.jpg)

:::info
以下のエラーが表示された場合は、リンクをクリックしてアカウント設定に移動し、請求先住所を追加してください。

![SubscriptionBillingError](/images/account/subscription-billing-error.jpg)
:::

4. 今日の支払総額\*\*を確認し、次の自動チャージがいつ行われるかを理解し、**購入を完了する**をクリックする。

![SubscriptionSuccess](/images/account/subscription-success.jpg)

5. \*\*デフォルトのスプラッシュ画面を削除するには、プロジェクトを再パブリッシュしてください。

## サブスクリプションの再アクティブ化 {#reactivate-subscription}

デフォルトでは、White Labelサブスクリプションは**無期限に実行され、お客様がキャンセルするか終了日をスケジュールするまで、請求期間ごとに**自動的に更新されます。 購入後、購読期間の設定はプロジェクトダッシュボードのページから管理できます。

サブスクリプション期間の設定は、プロジェクトダッシュボードから管理できます。 以下のオプションがある：

- \*\*継続中プロジェクトは無期限に実行され、購読は毎月自動的に更新されます。 次回の更新日時が表示されます。
- **現在の請求サイクルの後に終了**：  サブスクリプションは、現在の請求期間を通じて実行され、その後終了します。
- **スケジュール終了日時**：サブスクリプションを終了するカスタム日時を選択します。
- \*\*今すぐプロジェクトを終了するすぐにサブスクリプションを終了します（**無制限ホワイトレーベルのカスタムプランのみ**）。

購読期間を変更するには

1. プロジェクトのダッシュボードに移動します。
2. White Label Subscription\*\*ボックスで、**Edit**をクリックします：
3. ご希望の購読終了日を設定し、**Update**をクリックして変更を保存します：

![SubscriptionDuration](/images/account/subscription-duration.jpg)

## サブスクリプション期間 {#subscription-duration}

White Labelサブスクリプションがキャンセルされた場合、または予定された終了日に達した場合、課金は停止され、プロジェクトはプロジェクトダッシュボードの**Completed Projects**タブの下に再配置されます。

プロジェクトを再始動し、新しいWhite Labelサブスクリプションを購入するには、以下のいずれかのオプションに従ってください：

\*\*ワークスペースのページより：

1. ワークスペース・ページから、**Completed Projects** タブを選択します。
2. 再アクティブ化するプロジェクトを探す。
3. ... "メニューをクリックし、ドロップダウンメニューから**Reactivate White Label**を選択します。
4. [Purchase White Label](#purchase-subscription)ウィザードを完了し、White Labelサブスクリプションを再アクティブ化します。

**プロジェクト・ダッシュボードより**：

1. ワークスペース・ページから、**Completed Projects** タブを選択します。
2. 目的のプロジェクトを選択し、プロジェクト・ダッシュボード・ページに移動する。
3. 今すぐ起動」をクリックする。
4. [Purchase White Label](#purchase-subscription)ウィザードを完了し、White Labelサブスクリプションを再アクティブ化します。

## プロジェクト・リダイレクトURL {#project-redirect-url}

立ち上げたプロジェクトがキャンセルまたは完了した場合、ユーザーを別のURLにリダイレクトしたい場合があります。

プロジェクトリダイレクトURLを指定すると、
ホワイトレーベルのサブスクリプションが終了したときに、ユーザーを自動的に別のサイトにリダイレクトできます。

1. プロジェクト・ダッシュボードから、**プロジェクト・リダイレクトURL**セクションを見つけてください。
2. URLを設定する\*\*」をクリックする。
3. ご希望のリダイレクトURLを入力してください。
