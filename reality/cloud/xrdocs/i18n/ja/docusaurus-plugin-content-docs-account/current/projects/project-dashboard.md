# プロジェクト・ダッシュボード

プロジェクトダッシュボードは、8th Wallプロジェクトを管理するためのハブです。  プロジェクトのダッシュボードページ
から、プロジェクト設定の管理、8th Wall Cloud Editorへのアクセス、ホワイトレーベルのサブスクリプションの購入、
イメージターゲットの管理、カスタムドメインの設定などを行うことができます。

プロジェクト・ダッシュボードへの直接のURLは次の形式です： www.8thwall.com/workspacename/projectname
