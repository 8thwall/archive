# 使用状況と最近の傾向

## ビューと滞空時間 {#views-and-dwell-time}

以下の利用分析は、プロジェクトごとに提供されます：

- ビュー
- 滞留時間

Views\`タブでは、プロジェクトの有効期間中に何回閲覧されたかを見ることができます。
「滞留時間」タブでは、ユーザーがあなたのエクスペリエンスに費やした平均時間を見ることができます。 平均は
、データがゼロの日は考慮していない。

チャート内の時刻はすべて現地時間で表示されるが、8th WallはUTCで1時間ごとに
集計されたデータのみを収集する。 時差のないタイムゾーンにいるユーザーは、
、その日に最も近いUTC時間に収集されたデータを見ることができる。 滞空時間は2023年1月1日以降のみ利用可能。
直近の日は、現在時刻までの部分的なデータのみで構成される。

グラフ上のツールチップの時刻はデータ収集期間の開始時刻を表示し、ポイント値
はその日付からのデータの集計値である。

## CSVエクスポート {#csv-export}

より高度な分析のためにCSV形式のデータも利用できる。 グラフの上にあるダウンロードアイコンをクリックすると、
。 CSVフィールドは以下の通り：

| フィールド           | 説明                       |
| --------------- | ------------------------ |
| dt              | ISO8601形式のUTC時刻日付文字列。    |
| meanDwellTimeMs | 指定された日の平均セッション時間（ミリ秒単位）。 |
| views           | その日の閲覧数。                 |
