# ワークスペース

:::info
他の8th Wallユーザーからの招待でサインアップした場合、そのユーザーの既存のワークスペースのチームメンバーとして追加されます。 [詳細はこちら](/account/teams)。
:::

ワークスペースは、プロジェクト、ユーザー、請求書の論理的なグループです。 ワークスペース
には、それぞれ異なる権限を持つ 1 つまたは複数のユーザーを含めることができます。 ユーザーは、
複数のワークスペースに所属できます。

ワークスペース・ダッシュボードでは、以下のことができます：

- 新しいプロジェクトを作成する。
- 既存プロジェクトの管理
- アクセスモジュール。
- ワークスペースのチームメンバーと権限を管理します。
- 公開プロフィールの管理
- サブスクリプションの管理
