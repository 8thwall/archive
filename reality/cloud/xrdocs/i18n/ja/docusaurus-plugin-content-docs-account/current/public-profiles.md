# 公開プロフィール

:::info
ワークスペースの公開プロフィールを変更できるのは `Owner` と `Admin` ユーザーロールだけです。
:::

公開プロフィールは8thwall.comのあなた自身のページで、あなたの作品を紹介したり、
の経験をデモしたり、他の開発者とコードを共有したりすることができます。

公開プロフィールはアカウント作成時に設定され、以下の情報が表示されます：

- プロフィール写真（最低400x400pxのPNGまたはJPEG）
- 名前（3～30文字）
- ウェブサイトURL (https://www.mycompany.com)
- 主な所在地（都市）
- 略歴（240字以内）
- 連絡先Eメール（例：contact@mycompany.com）

## 公開プロフィールの更新

公開プロフィールの詳細は、**公開プロフィール**タブからいつでも更新できます：

1. 左のナビゲーションでワークスペースを展開します。

![ExpandWorkspace](/images/console-home-my-workspaces.jpg)

2. 公開プロフィール\*\*をクリック

![SelectPublicProfile](/images/console-workspace-nav-public-profile.jpg)

3. 公開プロフィールの詳細を更新する

4. 入力した情報を保存するには、**保存**をクリックします。

5. ワークスペースのショートネームを https://8thwall.com/ (例: https://8thwall.com/mycompany) に追加して、公開プロフィールにアクセスしてください。
