# サブスクリプションの管理

## ホワイトラベル購読

ホワイトレーベルのサブスクリプションとその支払い方法は、ワークスペースのアカウントページから管理できます。 このウィジェットは、アクティブな購読がある場合にのみ表示されます。 購読の支払い方法を変更したり、すぐにキャンセルしたりすることができます。

1. 8th Wallアカウントにログインし、"マイプロジェクトへ "をクリックします：

<!-- ![GoToMyProjects](/images/go-to-my-projects.jpg) -->

2. まだ開いていなければ、ワークスペースを広げる。

<!-- ![SelectWorkspace](/images/console-home-my-workspaces.jpg) -->

3. 左ナビゲーションの**アカウント**をクリックします。

<!-- ![SelectAccountNav](/images/console-workspace-nav.jpg) -->

## クレジット契約

クレジットは、**AIアセット生成**や**ネイティブアプリエクスポート**などの高度な8th Wall機能を使用する際に必要となります。

### クレジット定期購入

1. アカウントにログインし、左のナビゲーションで**アカウント**をクリックします。
2. サブスクリプションプラン**セクションで、ご希望のプランの下にある**アップグレード\*\*をクリックします。
3. Stripe経由でのアップグレードを確認するダイアログが開きます。

> 一部月または未使用クレジットの払い戻しはありません。

![クレジット購入](/images/account/purchase-credit-subscription.jpg)

### クレジット契約をキャンセルする

購読の中止・解約はいつでも可能です。

クレジット契約をキャンセルするには

1. アカウントにログインし、左ナビゲーションの**アカウント**をクリックします。
2. プランのキャンセル\*\*をクリックします。
3. Stripe経由でのキャンセルを確認するダイアログが開きます。

### クレジットの使用

クレジットは、クレジットを必要とするアクションを実行したときに使用されます。 使用量はアクションやモデルの選択によって異なる。

- アクションを実行する前に、そのアクションで消費されるクレジット数が表示されます。
- クレジットの使用量が多いか少ないかで、高出力または低出力のモデルを選ぶことができる。

![クレジットを使う](/images/account/using-credits.jpg)

> エラーが発生した場合、クレジットは消費されない。

### クレジット残高

クレジット残高を確認できます：

- アカウント\*\*ページ
- アセット・ラボ\*\*内
- Native App Export\*\*\* のパブリッシュ・モダルで

これらのインターフェイスのトップバーに表示されているクレジット番号をクリックします。

![クレジット残高](/images/account/credit-balance.jpg)

### クレジットのリフレッシュ方法

| 口座種別          | リフレッシュ動作          |
| ------------- | ----------------- |
| 無料ワークスペース     | クレジットは契約日に毎月更新される |
| サブスクリプション     | クレジットは加入日に毎月更新される |
| 1回限りのトップアップ   | クレジットの有効期限は購入後30日 |
| プロモーション・クレジット | 月末まで              |

カスタムアカウント\*\*については、[営業担当までお問い合わせください](https://www.8thwall.com/custom)。

## サービス信用条件

当社は、クレジットの購入を通じて特定のサービスの料金を前払いするオプション（以下「プリペイドサービスクレジット」）を提供する場合があります。 プリペイド サービス クレジットは、各サービスの利用に対してお客様が事前に支払った金額を表します。 毎月更新されるサブスクリプションまたは1回限りの購入として提供されます。 特定のサービスには追加条件が適用される場合があります。

また、サービスクレジットの購入ではなく、プロモーションプログラムの一環としてサービスクレジットを無料で提供する場合があります（「プロモサービスクレジット」）。 プロモサービスクレジットは、売上税、使用税、総収入税、または類似の取引に基づく税金には適用されません。

サービスクレジット

- 法定通貨ではない
- 換金、払い戻し、または金銭との交換はできない。
- 不換紙幣に相当する価値がない
- 譲渡はできず、適用されるサービスでのみ使用可能です。

前払いクレジットを含む本サービスの販売はすべて最終的なものとなります。 サービスクレジットの払い戻しはできず、有効期限は**購入または発行から30日**です。

クレジット残高は8th Wallアカウントで確認することができます。 正しい加減算を確認する責任はあなたにあります。 クレジット残高は銀行口座や決済手段ではありません。

当社は、サービスクレジットの譲渡、販売、または取引を禁止します。 そのような試みは、結果として

- クレジットの取り消しまたは終了
- アカウントの停止または終了
- 払い戻しは行わない

当社は、当社の利用規約に従い、お客様のアカウントを停止または終了する権利を留保します。
