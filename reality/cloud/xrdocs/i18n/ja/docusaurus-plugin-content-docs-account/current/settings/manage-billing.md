# 請求と支払い方法の管理

8th Wallは、Stripe Billing Portalを使用して支払い方法と請求情報を管理しています。 このページでは、以下のことができる：

- 新しい支払い方法を追加する
- 支払い方法を削除する
- デフォルトのお支払い方法を変更する
- 請求情報の更新

1. 8th Wallアカウントにログインし、"マイプロジェクトへ "をクリックします：

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. まだ開いていなければ、ワークスペースを広げる。

![SelectWorkspace](/images/account/select-workspace.jpg)

3. 左ナビゲーションの**アカウント**をクリックします。

![SelectAccount](/images/account/select-account.jpg)

4. 請求と支払い方法の管理\*\*」をクリックします。

![ManageBilling](/images/account/manage-billing.jpg)

:::info
更新された請求情報は、**今後の請求書/領収書**にのみ反映されます。
:::
