# クレジット管理

クレジットは、**AIアセット生成**や**ネイティブアプリエクスポート**などの高度な8th Wall機能を使用する際に必要となります。

## クレジット定期購入

1. アカウントにログインし、左のナビゲーションで**アカウント**をクリックします。
2. サブスクリプションプラン**セクションで、ご希望のプランの下にある**アップグレード\*\*をクリックします。
   ![Purchase credit subscription](/images/account/purchase-credit-subscription.jpg)
3. Stripe Checkoutでアップグレードを完了します。
   ![Stripe checkout](/images/account/stripe-checkout.jpg)

## クレジット契約をキャンセルする

購読の中止・解約はいつでも可能です。 一部月または未使用クレジットの払い戻しはありません。

クレジット契約をキャンセルするには

1. アカウントにログインし、左のナビゲーションで**アカウント**をクリックします。
2. プランのキャンセル\*\*をクリックします。
   ![Cancel credit subscription](/images/account/cancel-credit-subscription.jpg)
3. Stripe経由でのキャンセルを確認するダイアログが開きます。

## クレジット・トップアップの購入

1. アカウントにログインし、左のナビゲーションで**アカウント**をクリックします。
2. サブスクリプション・プラン**セクションで、クレジット・トップアップの下にある**今すぐ購入\*\*をクリックします。
   ![Purchase credit top-up](/images/account/purchase-credit-top-up.jpg)
3. クレジットの上乗せ額を選択します。
   ![Choose credit amount](/images/account/credit-top-up-amount.jpg)
4. Stripe Checkoutでアップグレードを完了します。

## クレジットの使用

:::info
エラーが発生した場合、クレジットは消費されない。
:::

クレジットは、クレジットを必要とするアクションを実行したときに使用されます。 使用量はアクションやモデルの選択によって異なる。 使用量はアクションやモデルの選択によって異なる。

- アクションを実行する前に、そのアクションで消費されるクレジット数が表示されます。
- クレジットの使用量が多いか少ないかで、高出力または低出力のモデルを選ぶことができる。

## クレジットのリフレッシュ方法

| 口座種別          | リフレッシュ動作          |
| ------------- | ----------------- |
| 無料ワークスペース     | クレジットは契約日に毎月更新される |
| サブスクリプション     | クレジットは加入日に毎月更新される |
| 1回限りのトップアップ   | クレジットの有効期限は購入後30日 |
| プロモーション・クレジット | 月末まで              |

カスタムアカウント\*\*については、[営業担当までお問い合わせください](https://www.8thwall.com/custom)。
