# プロジェクト・ダッシュボード

プロジェクト・ダッシュボードへの直接のURLは次の形式です： www.8thwall.com/workspacename/projectname

## イメージターゲットの管理 {#managing-image-targets}

指定したプロジェクトのイメージターゲットを管理するには、左の
ナビゲーションにあるイメージターゲットアイコン、またはプロジェクトダッシュボードの「イメージターゲットを管理」リンクをクリックします。

![ManageImageTargets](/images/console-appkey-imagetargets.jpg)

イメージターゲットの詳細については、
[イメージターゲットドキュメント](/legacy/guides/image-targets)を参照してください。

## Lightship VPSロケーションの管理 {#managing-lightship-vps-locations}

特定のプロジェクトのライトシップVPSロケーションを管理するには、左ナビゲーションの地図アイコンをクリックします。

![ManageWayspots](/images/console-wayspot-left-nav.jpg)

Lightship VPS for Web の詳細については、
[Lightship VPS for Web ドキュメント](/legacy/guides/lightship-vps)をご参照ください。
