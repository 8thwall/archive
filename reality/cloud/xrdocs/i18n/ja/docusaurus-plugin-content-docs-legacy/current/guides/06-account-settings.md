# アカウント設定

アカウント・ページでは以下のことができる：

- [購入](#purchase-plan) プラン加入。
- 有効なプラン契約がある場合は、[プランのキャンセル](#cancel-plan)。
- [課金期間変更](#change-billing-period)で月課金と年課金を切り替えることができます。
- 異なるプランへのアップグレードまたはダウングレードが必要な場合は、[プラン変更](#change-plan)をご利用ください。
- [商用ライセンスの管理](#manage-commercial-licenses)
- [支払い方法の管理](#manage-payment-methods)
- [請求書管理](#invoices) (閲覧、支払い、ダウンロード)
- [請求情報の更新](#update-billing-information)
- [Setup Payments API](./advanced-topics/08-payments-module.md)を使って、WebARやWebVR
  の体験を8th Wall Paymentsで収益化することができます。

5. プランの管理\*\*」をクリックする。

![ManagePlanBillingPeriod](/images/console-workspace-manage-plan.jpg)

6. 現在の請求間隔が表示されます。  現在の請求間隔が表示されます。  新しい請求間隔を選択し、プロモーションコード（該当する場合）を追加し、支払い方法を選択し、**Submit**をクリックして、次の請求サイクルの開始に合わせて請求期間の変更をスケジュールします。
