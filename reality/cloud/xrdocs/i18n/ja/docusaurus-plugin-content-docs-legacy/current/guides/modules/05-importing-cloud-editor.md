## モジュールをクラウド エディタ プロジェクトにインポートする {#importing-modules-into-a-cloud-editor-project}

**モジュールをクラウド・エディター・プロジェクトにインポートするには**：

1. クラウド エディタ内で、モジュールの隣にある「+」ボタンを押します。

   、プロジェクト内に同じエイリアスを持つモジュールがある場合、モジュール名を変更する必要があるかもしれません。

4. これで、プロジェクト内の "Modules "セクションにモジュールが表示されます。

![modules-step4-press-import](/images/modules-added-to-project.jpg)

5. インポートされたモジュールを選択すると、モジュール設定ページに移動します。 このページを使用して、さまざまなモジュール・パラメーターを設定することができます。 このページを使用して、さまざまなモジュール・パラメーターを設定することができます。
