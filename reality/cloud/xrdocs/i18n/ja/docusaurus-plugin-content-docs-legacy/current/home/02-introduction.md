## フレームワーク互換性 {#framework-compatibility}

8th Wall Cloud Editorは、次のような3D JavaScriptフレームワークに簡単に統合できます：

- [Aフレーム](https://aframe.io/)
- [three.js](https://threejs.org/)
- [プレイキャンバス](https://www.playcanvas.com)
- [Babylon.js](https://www.babylonjs.com/)

注：UnityでアプリベースのARを開発するには、[Niantic Lightship ARDK](https://lightship.dev/)を使用してください。
