# XR8.Xrパーミッション

## 説明 {#description}

パイプラインモジュールに必要なパーミッションを指定するユーティリティ。

モジュールは、許可要求が必要なブラウザの機能を示すことができます。 これらは、もし適切なパーミッションがない場合、フレームワークが要求するために使用されるか、XRを実行する前に適切なパーミッションを要求するコンポーネントを作成するために使用される。 これらは、もし適切なパーミッションがない場合、フレームワークが要求するために使用されるか、XRを実行する前に適切なパーミッションを要求するコンポーネントを作成するために使用される。

## 物件 {#properties}

| プロパティ                                          | タイプ | 説明                          |
| ---------------------------------------------- | --- | --------------------------- |
| [パーミッション()](permissions.md) | 列挙  | パイプラインモジュールで必要なパーミッションのリスト。 |

## 例 {#example}

```javascript
XR8.addCameraPipelineModule({
  name: 'request-gyro',
  requiredPermissions: () => ([XR8.XrPermissions.permissions().DEVICE_ORIENTATION]),
})
```
