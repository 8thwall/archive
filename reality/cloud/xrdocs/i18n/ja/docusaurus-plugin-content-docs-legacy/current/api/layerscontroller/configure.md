# XR8.LayersController.configure()

`XR8.LayersController.configure({ nearClip, farClip, coordinates, layers })`.

## 説明 {#description}

LayersController\\` が行う処理を設定する。

## パラメータ {#parameters}

| パラメータ                                      | タイプ                             | デフォルト  | 説明                                                                                                                                                                |
| ------------------------------------------ | ------------------------------- | ------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| nearClip [オプション］ | 番号                              | `0.01` | ニアクリッププレーンのカメラからの距離、つまりシーンオブジェクトが見えるカメラからの最も近い距離。                                                                                                                 |
| farClip [オプション］  | 番号                              | `1000` | 遠いクリップ面のカメラからの距離。つまり、シーンオブジェクトが見えるカメラからの最も遠い距離。                                                                                                                   |
| 座標 [オプション］       | 座標                              |        | カメラの設定。                                                                                                                                                           |
| レイヤー [オプション］     | `Record<String, LayerOptions?>` | `{}`   | 検出する意味層。 キーはレイヤー名である。 レイヤーの端を滑らかにする量。 有効な値は[0-1]の間である。 デフォルト値に戻すには `null` を渡す。 現在サポートされているレイヤー名は`'sky'`のみである。 |

Coordinates\\`オブジェクトは以下のプロパティを持つ：

| パラメータ                                    | タイプ                                                     | デフォルト                                                     | 説明                                                                                   |
| ---------------------------------------- | ------------------------------------------------------- | --------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| origin [オプション］ | `{position： {x, y, z}回転：{w, x, y, z}}`. | `{position： {x: 0, y: 2, z: 0}回転：{w:1, x: 0, y：0, z: 0}}` | カメラの位置と回転。                                                                           |
| scale [オプション］  | 番号                                                      | `2`                                                       | 現場の規模。                                                                               |
| axes [オプション］   | 文字列                                                     | 右利き                                                       | LEFT_HANDED'`または`'RIGHT_HANDED'\\` を指定する。 |
| mirroredDisplay［オプション］                   | ブーリアン                                                   | false\\`                                                 | trueの場合、出力の左右を反転する。                                                                  |

LayerOptions\\`オブジェクトは以下のプロパティを持つ：

| パラメータ                                                                        | タイプ   | デフォルト     | 説明                                                                                                                                                                                                                                                                                                                       |
| ---------------------------------------------------------------------------- | ----- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| invertLayerMask [オプション]。 | ブーリアン | false\\` | true`の場合、シーンに配置したコンテンツは空以外のエリアでも見えるようになります。 もし`false`なら、シーンに配置したコンテンツはスカイエリアに表示されます。 デフォルト値に戻すには `null\` を渡す。 true`の場合、シーンに配置したコンテンツは空以外のエリアでも見えるようになります。 もし`false`なら、シーンに配置したコンテンツはスカイエリアに表示されます。 デフォルト値に戻すには `null\` を渡す。 もし`false`なら、シーンに配置したコンテンツはスカイエリアに表示されます。 デフォルト値に戻すには `null\` を渡す。 デフォルト値に戻すには `null` を渡す。 |
| edgeSmoothness［オプション］                                                        | 番号    | `0`       | レイヤーの端を滑らかにする量。 有効な値は[0-1]の間である。 デフォルト値に戻すには `null` を渡す。                                                                                                                                                                                                             |

\*\* 重要:\*\* [`XR8.LayersController`](./layerscontroller.md) と [`XR8.FaceController`](../facecontroller/facecontroller.md) を同時に使用することはできません。

## {#returns}を返す。

なし

## 例 {#example}

```javascript
XR8.LayersController.configure({layers: {sky: {invertLayerMask: true, edgeSmoothness: 0.8}}})
```
