# PlayCanvas イベントリスナー

このセクションでは、PlayCanvas環境で8th Wall Webがリッスンするイベントについて説明します。

Webアプリケーションでこれらのイベントを発生させ、様々なアクションを実行することができます：

| イベント・リスナー                                                      | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| -------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [xr:hidecamerafeed](xrhidecamerafeed.md)       | カメラの映像を隠す。 追跡は止まらない。                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [xr:recenter](xrrecenter.md)                   | カメラの映像を原点に戻す。 新しい原点が引数として与えられると、カメラの原点はその原点にリセットされ、リセンターされる。                                                                                                                                                                                                                                                                                                                                                                                  |
| [xr:screenshotrequest](xrscreenshotrequest.md) | PlayCanvas キャンバスのスクリーンショットをキャプチャするリクエストをエンジンに発行します。 エンジンはJPEG圧縮画像とともに[`xr:screenshotready`](/legacy/api/playcanvasevents/xrscreenshotready)イベントを発行し、エラーが発生した場合は[`xr:screenshoterror`](/legacy/api/playcanvasevents/xrscreenshoterror)イベントを発行します。 エンジンはJPEG圧縮画像とともに[`xr:screenshotready`](/legacy/api/playcanvasevents/xrscreenshotready)イベントを発行し、エラーが発生した場合は[`xr:screenshoterror`](/legacy/api/playcanvasevents/xrscreenshoterror)イベントを発行します。 |
| [xr:showcamerafeed](xrshowcamerafeed.md)       | カメラのフィードを表示します。                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [xr:stopxr](xrstopxr.md)                       | 現在の XR セッションを停止する。 現在の XR セッションを停止する。 停止中、カメラフィードは停止し、デバイスの動きは追跡されません。 現在の XR セッションを停止する。 停止中、カメラフィードは停止し、デバイスの動きは追跡されません。                                                                                                                                                                                                                                                                                                                    |
