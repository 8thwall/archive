# XR8.isInitialized()

`XR8.isInitialized()`

## 説明 {#description}

ARエンジンのWebAssemblyが初期化されているかどうかを示します。

## パラメータ {#parameters}

なし

## {#returns}を返す。

ARエンジンのWebAssemblyが初期化されているかどうかを示すブール値。

## 例 {#example}

```javascript
if (XR8.isInitialized()) {
  console.log(XR8.version())
}.
```
