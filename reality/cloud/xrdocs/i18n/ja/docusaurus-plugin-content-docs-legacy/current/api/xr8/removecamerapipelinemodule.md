# XR8.removeCameraPipelineModule()

XR8.removeCameraPipelineModule(moduleName)\\`を実行します。

## 説明 {#description}

カメラパイプラインからモジュールを削除する。

## パラメータ {#parameters}

| パラメータ  | タイプ | 説明        |
| ------ | --- | --------- |
| モジュール名 | 文字列 | モジュールの名前。 |

## {#returns}を返す。

なし

## 例 {#example}

```javascript
XR8.removeCameraPipelineModule('現実')
```
