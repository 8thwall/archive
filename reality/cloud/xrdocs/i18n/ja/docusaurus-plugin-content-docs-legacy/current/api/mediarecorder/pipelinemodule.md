# XR8.MediaRecorder.pipelineModule()

`XR8.MediaRecorder.pipelineModule()`

## 説明 {#description}

MP4形式でビデオを録画できるカメラパイプラインモジュールを提供します。

## パラメータ {#parameters}

なし

## {#returns}を返す。

MediaRecorder\\`パイプラインモジュールはビデオを録画することができる。

## 例 {#example}

```javascript
XR8.addCameraPipelineModule(XR8.MediaRecorder.pipelineModule())
```
