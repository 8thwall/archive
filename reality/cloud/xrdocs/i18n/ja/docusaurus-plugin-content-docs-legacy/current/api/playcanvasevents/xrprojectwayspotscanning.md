# xr:プロジェクトウェイスポットスキャンニング

## 説明 {#description}

このイベントは、すべてのプロジェクトロケーションがスキャンのためにロードされたときに発行されます。

`xr:projectwayspotfound.detail : { ウェイスポット：[] }`

| プロパティ                                                         | 説明               |
| ------------------------------------------------------------- | ---------------- |
| ウェイスポット[] | 位置情報を含む配列オブジェクト。 |

wayspots\\`は以下のプロパティを持つオブジェクトの配列である：

| プロパティ   | 説明                            |
| ------- | ----------------------------- |
| アイドル    | セッション内で安定している、このプロジェクトロケ地のID。 |
| 名称      | プロジェクトの場所名。                   |
| イメージURL | このプロジェクトの場所の代表的な画像のURL。       |
| タイトル    | ロケのタイトル                       |
| ラット     | このプロジェクトの場所の緯度                |
| lng     | このプロジェクトの場所の経度。               |
