# xrinterpupillarydistance

## 説明 {#description}

このイベントは[`xrface`](/legacy/api/aframe/#face-effects)によって、トラッキングされた顔の各瞳孔の中心間のミリメートル単位の距離が最初に検出されたときに発行される。

`xrinterpupillarydistance.detail : {id}`.

| プロパティ | タイプ | 説明                       |
| ----- | --- | ------------------------ |
| アイドル  | 番号  | 位置する面の数値ID。              |
| 瞳孔間距離 | 番号  | 各瞳孔の中心間のおおよその距離（ミリメートル）。 |
