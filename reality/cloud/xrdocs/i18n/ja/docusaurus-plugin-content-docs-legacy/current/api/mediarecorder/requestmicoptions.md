# XR8.MediaRecorder.RequestMicOptions

列挙

## 説明 {#description}

オーディオパーミッションが要求されるタイミングを決定します。

## 物件 {#properties}

| プロパティ | 価値    | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ----- | ----- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| オート   | オート   | onAttach()\\`](/legacy/api/camerapipelinemodule/onattach)で自動的にマイクのパーミッションを要求する。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| マニュアル | マニュアル | onAttach()`](/legacy/api/camerapipelinemodule/onattach)では、マイクのパーミッションは要求されません。 アプリに追加された他の音声は、AudioContextに追加され、[`XR8.MediaRecorder.configure()`](configure.md)に渡されたユーザーの[`configureAudioOutput`](/legacy/api/mediarecorder/configure/#parameters)関数に指定されたaudioProcessorに接続された場合でも録音されます。 XR8.MediaRecorder.requestMicrophone()`](requestmicrophone.md)を呼び出すことで、マイクの許可を手動で要求することができます。 アプリに追加された他の音声は、AudioContextに追加され、[`XR8.MediaRecorder.configure()`](configure.md)に渡されたユーザーの[`configureAudioOutput`](/legacy/api/mediarecorder/configure/#parameters)関数に指定されたaudioProcessorに接続された場合でも録音されます。 XR8.MediaRecorder.requestMicrophone()\`](requestmicrophone.md)を呼び出すことで、マイクの許可を手動で要求することができます。 |
