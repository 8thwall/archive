# XR8.XrController

## 説明 {#description}

XrController\\`は6DoFカメラ・トラッキングとトラッキング設定用インターフェースを提供する。

| [configure](configure.md)                                       | XrController\\`がどのような処理を行うかを設定する（パフォーマンスに影響する場合がある）。                                                                                                                                                                                 |
| [ヒットテスト](hittest.md)                                            | カメラフィード上の点の3D位置を推定する。                                                                                                                                                                                                                 |

| [recenter](recenter.md)                                         | updateCameraProjectionMatrixで指定された原点/向いている方向にカメラを再配置し、トラッキングを再開します。                                                                                                                                                                   |
