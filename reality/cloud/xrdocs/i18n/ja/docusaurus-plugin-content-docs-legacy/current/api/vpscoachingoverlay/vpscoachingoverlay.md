# VpsCoachingOverlay

## 説明 {#description}

Lightship VPS対応のWeb AR体験にコーチングオーバーレイを生成するモジュールを提供します。

Absolute Scale Coaching Overlayについては、[こちら](/legacy/api/coachingoverlay)をご参照ください。

## 機能 {#functions}

| 機能                                                      | 説明                                                       |
| ------------------------------------------------------- | -------------------------------------------------------- |
| [configure](vps-coachingoverlay-configure.md)           | コーチングオーバーレイの設定を行います。                                     |
| [pipelineModule](vps-coachingoverlay-pipelinemodule.md) | インストールすると、プロジェクトにコーチングオーバーレイ機能を追加するカメラパイプラインモジュールを作成します。 |
