# XR8.XrDevice.IncompatibilityReasons

列挙

## 説明 {#description}

デバイスとブラウザが第8ウォールウェブと互換性がない可能性のある理由。

## 物件 {#properties}

| プロパティ                                      | 価値  | 説明                                |
| ------------------------------------------ | --- | --------------------------------- |
| 不明                                         | `0` | 互換性のない理由は明記されていない。                |
| UNSUPPORTED_OS        | `1` | 推定されるオペレーティングシステムはサポートされていません。    |
| 非対応ブラウザ                                    | `2` | 推定ブラウザはサポートされていません。               |
| device_orientationの欠落 | `3` | このブラウザはデバイスの向きのイベントをサポートしていません。   |
| ユーザーメディア不足                                 | `4` | このブラウザはユーザーメディアへのアクセスをサポートしていません。 |
| 欠落しているウェブ・アセンブリ                            | `5` | ブラウザがウェブアセンブリをサポートしていません。         |
