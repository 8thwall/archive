# XR8.スリージェイエス

## 説明 {#description}

仮想オーバーレイを行うためにthree.jsカメラを駆動するカメラパイプラインモジュールを提供します。

## 機能 {#functions}

| 機能                                  | 説明                                                                       |
| ----------------------------------- | ------------------------------------------------------------------------ |
| [pipelineModule](pipelinemodule.md) | three.jsの環境とライフサイクルとのインターフェイスとなるパイプラインモジュール。             |
| [configure](configure.md)           | three.jsのレンダラーを設定します。                                    |
| [xrScene](xrscene.md)               | xrシーン、カメラ、レンダラー、（オプションの）カメラフィードテクスチャ、および（オプションの）layerScenesへのハンドルを取得します。 |
