# ランディングページ

## 説明 {#description}

ウェブAR体験のためのカスタムランディングページを生成するモジュールを提供します。

## 機能 {#functions}

| 機能                                  | 説明                                                     |
| ----------------------------------- | ------------------------------------------------------ |
| [configure](configure.md)           | LandingPageの設定を行います。                                   |
| [pipelineModule](pipelinemodule.md) | インストールすると、プロジェクトにランディングページ機能を追加するカメラパイプラインモジュールを作成します。 |
