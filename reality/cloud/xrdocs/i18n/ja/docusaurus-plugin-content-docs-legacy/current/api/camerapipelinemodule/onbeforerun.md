# onBeforeRun()

OnBeforeRun: ({ config })\\`。

## 説明 {#description}

onBeforeRun\\`は[XR8.run()](/legacy/api/xr8/run)の直後に呼び出される。 何らかの約束が返された場合、XRはすべての約束を待ってから続行する。 何らかの約束が返された場合、XRはすべての約束を待ってから続行する。

## パラメータ {#parameters}

| パラメータ | 説明                                                                                                                            |
| ----- | ----------------------------------------------------------------------------------------------------------------------------- |
| コンフィグ | XR8.run()](/legacy/api/xr8/run) に渡された設定パラメータ。 |
