# コーチングオーバーレイ

## 説明 {#description}

Absolute Scale Web AR体験のためのCoaching Overlayを生成するモジュールを提供します。

ライトシップVPSコーチング・オーバーレイについては、[こちら](/legacy/api/vpscoachingoverlay)をご覧ください。

## 機能 {#functions}

| 機能                                  | 説明                                                       |
| ----------------------------------- | -------------------------------------------------------- |
| [configure](configure.md)           | コーチングオーバーレイの設定を行います。                                     |
| [pipelineModule](pipelinemodule.md) | インストールすると、プロジェクトにコーチングオーバーレイ機能を追加するカメラパイプラインモジュールを作成します。 |
