## XR8.XrConfig {#xr8xrconfig}

## 説明 {#description}

パイプライン・モジュールが実行されるべきデバイスやカメラのクラスを指定するためのユーティリティ。

## 物件 {#properties}

| プロパティ                                  | タイプ | 説明                        |
| -------------------------------------- | --- | ------------------------- |
| [カメラ()](camera.md)  | 列挙  | 使用したいカメラ                  |
| [デバイス()](device.md) | 列挙  | パイプラインを実行するデバイスのクラスを指定する。 |
