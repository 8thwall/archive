## 説明 {#description}

バーチャル・オーバーレイを行うためにBabylon.jsカメラを駆動するために、BabylonJS環境とライフ・スタイルとのインターフェイスを提供するインテグレーション。

## 機能 {#functions}

| 機能                                          | 説明                                                             |
| ------------------------------------------- | -------------------------------------------------------------- |
| [xrCameraBehavior](xrcamerabehavior.md)     | ワールドトラッキングおよび/またはイメージターゲットを実行するために、バビロンカメラにアタッチできるビヘイビアを取得します。 |
| [faceCameraBehavior](facecamerabehavior.md) | バビロンカメラに取り付けてフェイス・エフェクトを実行できるビヘイビアを入手する。                       |
