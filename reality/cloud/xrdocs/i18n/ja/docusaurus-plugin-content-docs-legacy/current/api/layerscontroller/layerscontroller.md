# XR8.LayersController

## 説明 {#description}

LayersController\\` はセマンティックレイヤーの検出と、レイヤーのレンダリングを設定するためのインターフェースを提供する。

## 機能 {#functions}

| 機能                                  | 説明                                              |
| ----------------------------------- | ----------------------------------------------- |
| [configure](configure.md)           | LayersController\\` がどのような処理を行うかを設定する。         |
| [getLayerNames](getlayernames.md)   | LayersController\\` で設定されているレイヤーを返す。           |
| [pipelineModule](pipelinemodule.md) | カメラパイプラインモジュールを作成し、インストールするとセマンティックレイヤー検出を行います。 |
| [recenter](recenter.md)             | カメラの向きを原点に戻します。                                 |
