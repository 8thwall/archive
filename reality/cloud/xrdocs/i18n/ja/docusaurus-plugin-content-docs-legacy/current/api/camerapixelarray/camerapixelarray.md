# XR8.カメラピクセル配列

## 説明 {#description}

グレースケールまたはカラーの uint8 配列としてカメラデータにアクセスできるカメラパイプラインモジュールを提供します。

## 機能 {#functions}

| 機能                                  | 説明                                                                 |
| ----------------------------------- | ------------------------------------------------------------------ |
| [pipelineModule](pipelinemodule.md) | CPUの画像処理に使用できるRGBAまたはグレースケールのピクセル値の配列としてカメラのテクスチャを提供するパイプラインモジュール。 |
