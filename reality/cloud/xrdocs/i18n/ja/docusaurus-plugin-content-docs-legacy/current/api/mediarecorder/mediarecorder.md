# XR8.メディアレコーダー

## 説明 {#description}

MP4形式でビデオを録画できるカメラパイプラインモジュールを提供します。

## 機能 {#functions}

| 機能                                        | 説明                                         |
| ----------------------------------------- | ------------------------------------------ |
| [configure](configure.md)                 | ビデオ録画設定を行う。                                |
| [pipelineModule](pipelinemodule.md)       | MP4形式でビデオを録画するパイプラインモジュールを作成します。           |
| [録画ビデオ](recordvideo.md)                   | 録音を開始する。                                   |
| [requestMicrophone](requestmicrophone.md) | 音声の録音を有効にし（自動的に有効になっていない場合）、必要に応じて許可を要求する。 |
| [stopRecording](stoprecording.md)         | 録音を停止する。                                   |
| [RequestMicOptions](requestmicoptions.md) | マイクの許可を自動的に要求するかどうかのEnum。                  |
