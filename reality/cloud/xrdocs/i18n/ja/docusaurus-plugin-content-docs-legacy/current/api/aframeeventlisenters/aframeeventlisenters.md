# AFrameイベントリスナー

このセクションでは、"xrweb" A-Frame コンポーネントがリッスンするイベントについて説明します。

Webアプリケーションでこれらのイベントを発生させ、様々なアクションを実行することができます：

| イベント・リスナー                              | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| -------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [hidecamerafeed](hidecamerafeed.md)    | カメラの映像を隠す。 追跡は止まらない。                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [recenter](recenter.md)                | カメラの映像を原点に戻す。 新しい原点が引数として与えられると、カメラの原点はその原点にリセットされ、リセンターされる。                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [スクリーンショットリクエスト](screenshotrequest.md) | AFrameキャンバスのスクリーンショットをキャプチャするリクエストをエンジンに送ります。 AFrameキャンバスのスクリーンショットをキャプチャするリクエストをエンジンに送ります。 エンジンはJPEG圧縮画像とともに[`screenshotready`](/legacy/api/aframeevents/screenshotready)イベントを発行し、エラーが発生した場合は[`screenshoterror`](/legacy/api/aframeevents/screenshoterror)イベントを発行します。 AFrameキャンバスのスクリーンショットをキャプチャするリクエストをエンジンに送ります。 エンジンはJPEG圧縮画像とともに[`screenshotready`](/legacy/api/aframeevents/screenshotready)イベントを発行し、エラーが発生した場合は[`screenshoterror`](/legacy/api/aframeevents/screenshoterror)イベントを発行します。 |
| [showcamerafeed](showcamerafeed.md)    | カメラのフィードを表示します。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [stopxr](stopxr.md)                    | 現在の XR セッションを停止する。 現在の XR セッションを停止する。 停止中、カメラフィードは停止し、デバイスの動きは追跡されません。 現在の XR セッションを停止する。 停止中、カメラフィードは停止し、デバイスの動きは追跡されません。                                                                                                                                                                                                                                                                                                                                                                      |
