# XR8.FaceController.MeshGeometry

列挙

## 説明 {#description}

面のどの部分にメッシュ三角形を返すかを定義するためのオプション。

## 物件 {#properties}

| プロパティ | 価値   | 説明             |
| ----- | ---- | -------------- |
| 顔     | 顔    | 顔のジオメトリを返す。    |
| 口     | 口    | 口のジオメトリーを返す。   |
| 瞳     | 目    | 目にジオメトリーを返す。   |
| アイリス  | アイリス | アヤメのジオメトリーを返す。 |
