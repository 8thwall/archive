# 変更が表示されない

## はじめに

{frontMatter.description}

## なぜこのようなことが起こるのか？

もしかしたら、変更を忘れているかもしれない。 ランディングは、他の人があなたの変更を同期したり、変更を公開したりするための前提条件です。 パブリッシュは、変更をステージングまたはプロダクションに反映させる唯一の方法です。

## どうすれば直りますか？

1. ビルド
2. 土地
3. ステージングまたはプロダクションにパブリッシュする。
