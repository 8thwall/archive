# アセット・ラボ

Asset Labは、8th Wall Studioでアセットを生成するためのAIを搭載した機能群です。 Studioプロジェクトで使用する画像、3Dモデル、アニメーションキャラクターを生成するワークフローを効率化するように設計されています。

### 主な特徴

- さまざまなアセットタイプを生成するためのユースケースに最適化されたワークフロー
- 業界をリードするAIモデルとの統合
- 簡単なプロジェクトのインポート
- ワークスペース・レベルで生成された資産ライブラリとクレジット残高

## 資産を生み出す

1. Studio の左パネルにある **Asset Lab** タブをクリックします。
2. `Generate…`ボタンをクリックしてワークフローを選択する。
3. 選択したワークフローに基づいた手順に従ってください。

選択したワークフローのステップが上部に表示されます。
