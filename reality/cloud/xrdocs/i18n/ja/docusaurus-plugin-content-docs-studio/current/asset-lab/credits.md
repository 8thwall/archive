# クレジット

StudioでAIによるアセット生成を使用するにはクレジットが必要です。 各アクションは、使用するモデルやサービスに応じて特定のクレジット数を消費する。 すべてのアカウントは、サインアップ時に無料クレジットのプールから始まり、追加クレジットはサブスクリプションで購入することができます。

アカウント設定](/account/settings/manage-credits/)から、クレジットの購入やトップオフが可能です。

## テキストから画像へ

| モデル                              | クレジット価格                      |
| -------------------------------- | ---------------------------- |
| FLUX.1 シュネル      | 0.25クレジット/画像 |
| FLUX1.1プロ        | 2クレジット／画像                    |
| FLUX.1 コンテキスト・プロ | 2クレジット／画像                    |
| 安定したイメージコア                       | 2クレジット／画像                    |
| イメージグプト1                         | 2.5クレジット／画像  |
