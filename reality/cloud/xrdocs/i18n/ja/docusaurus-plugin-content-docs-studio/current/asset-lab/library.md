# 資産ライブラリ

アセット・ラボ・ライブラリー

- ワークスペースで生成されたすべてのアセットにアクセスする
- 右上に表示されるクレジット残高の確認
- タイプで絞り込む画像, 3D モデル, アニメーションキャラクター
- 検索と並べ替えツールを使って整理する

![](/images/studio/asset-lab/library1.png)

アセットをクリックするとメタデータが表示されます：

- 使用されるプロンプト
- 提出したユーザー
- 使用モデル
- 信用コスト

利用可能なアクション

- **ワークフローに送る** - 資産を入力として再利用する
- **ダウンロード** - ローカルに保存
- **リミックス** - 入力設定の調整
- **インポート** - プロジェクトのアセット・フォルダーに追加する。
