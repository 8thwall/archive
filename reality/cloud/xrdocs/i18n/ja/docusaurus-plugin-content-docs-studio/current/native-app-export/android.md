# Android

## アンドロイドへのエクスポート

1. \*\*スタジオプロジェクトを開いてください。 プロジェクトが[要求基準](/studio/native-app-export/#requirements)を満たしていることを確認する。

2. Publish**をクリックしてください。 エクスポート**で、**アンドロイド**を選択します。

3. \*\*アプリのビルドをカスタマイズする。
   - \*\*Androidのホーム画面に表示される名前です。
   - **Bundle Identifier:** `com.mycompany.myapp` などの一意な文字列。
   - \*\*(オプション)**アプリのアイコンをアップロードする** (1024x1024)

4. アプリの基本情報を入力したら、**Continue**をクリックしてビルド設定を確定します。

---

## ビルド設定の最終決定

次に、アプリをどのようにパッケージ化するかを定義する：

- **バージョン:** セマンティック・バージョニング（例えば `1.0.0`）を使用する。
- \*\*オリエンテーション
  - \*\*ポートレート:\*\*デバイスが回転しても、アプリを縦位置に固定します。
  - **ランドスケープ・レフト:** デバイスを左側が下になるように回転させ、アプリを水平に表示します。
  - \*\*アプリを水平に表示します。
  - \*\*自動回転：\*\*\*デバイスの物理的な回転にアプリが追従し、縦と横のビューを自動的に切り替えることができます。
  - **Auto Rotate (Landscape Only):** デバイスの回転に基づいてアプリの位置を調整しますが、水平表示のみに制限されます。
- \*\*ステータスバー
  - \*\*アプリケーションの上にデフォルトのシステムステータスバーを表示します。
  - \*\*デフォルトのシステム・ステータス・バーを非表示にします。
- **ビルド・タイプ:**
  - \*\*APK（Androidパッケージ）：\*\*テストまたはサイドローディングのための直接インストールファイル
  - **AAB（Android App Bundle）：** Google Playでの公開に必要です。
- \*\*ビルド・モード
  - \*\*ライブリロード：\*\*\*プロジェクトが更新されると、スタジオから更新をプルします。
  - **スタティック・バンドル:** 完全自己完結型ビルド
- **環境:** `Dev`、`Latest`、`Staging`、`Production` から選択する。

すべての設定が完了したら、**Build**をクリックしてアプリ・パッケージを生成します。

> ビルドが完了したら、ビルドの概要に記載されているダウンロードリンクを使って `.apk` または `.aab` ファイルをダウンロードしてください。

## Google Playストアへの公開

エクスポートが完了したら、\*\*AAB (Android App Bundle)\*\*を使ってアプリをPlayストアに公開する準備が整いました：

### なぜAABなのか？

Googleは2021年8月以降、すべての新規アプリにAABフォーマットを要求している。

### Google Play Consoleにアップロードする

1. Play Console](https://play.google.com/console)にログインし、必要に応じてPlay App Signingに登録する。
2. アプリの作成 "\*\*に移動→名前、言語、無料/有料ステータスを入力します。
3. テスト＆リリース\*\* → **プロダクション**（または内部/ベータトラック）に移動します。 **Create new release**をクリックし、.aabファイルを**Drop app bundles here to upload**セクションにドラッグしてアップロードしてください。
4. 完全な店舗リスト、プライバシーポリシー、コンテンツ評価、対象地域
5. リリースのレビューとロールアウト

🔗 [アップロードに関する完全なドキュメントはこちら: Play Consoleへのアプリのアップロード](https://developer.android.com/studio/publish)

## Androidデバイスに直接インストールする

### 物理的なAndroidデバイスへのインストール

1. ブラウザやファイルマネージャーで、\*\*「不明なアプリをインストールする」\*\*を有効にする。
2. USB、Eメール、クラウドストレージ経由でAPKを転送する。
3. お使いのデバイスからAPKを開き、**インストール**をタップします。

**コマンドライン方式の場合：**\*。

```bash
adb install path/to/app.apk
```

### Androidエミュレータへのインストール

1. Android StudioのAVD Managerでエミュレーターをセットアップする。
2. エミュレータを実行する。
3. コンピュータからAPKをエミュレータにドラッグ＆ドロップしてインストールしてください。

ターミナルで

```bash
adb install path/to/app.apk
```
