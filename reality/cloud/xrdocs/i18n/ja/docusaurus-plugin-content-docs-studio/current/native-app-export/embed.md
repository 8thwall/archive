# 埋め込み

## iFrameに8th Wallプロジェクトを埋め込む

1. \*\*スタジオプロジェクトを開いてください。

2. Publish**をクリックしてください。 左上の**Publish\*\*セクションで、**Embed**を選択します。

3. \*\*埋め込みをカスタマイズする。
   - \*\*単純なiFrameまたは完全なHTMLページのいずれかを選択することができます。

4. 埋め込みコードをクリップボードにコピーするには、**コピー**をクリックしてください。

5. 埋め込みコードをあなたのウェブサイトのソースコードに貼り付けます。 また、必要に応じて埋め込みコードをカスタマイズすることもできます。

## ゲーム・プラットフォームへの8th Wallプロジェクトの公開

8th WallのiFrameは、多くのゲームプラットフォームと互換性があり、公開することができます。

### イッチ・ドット・アイオ

1. フルHTML\*\*埋め込みコードをダウンロードしてください。
2. [Itch.io](https://itch.io)にログインし、[新規プロジェクトを作成](https://itch.io/game/new)してください。
3. プロジェクトの詳細を記入してください：
   - プロジェクトの種類\*\*で、**HTML**を選択します。
   - アップロード**で、**ファイルのアップロード**を選択します。 ステップ1でダウンロードした `index.html` ファイルをアップロードする。 このファイルはブラウザで再生されます**チェックボックスをオンにします。
   - 埋め込みオプション\*\*で、プロジェクトに適したサイズを選択します。
4. ゲームの設定を完了し、公開します。

### 多様性

1. フルHTML\*\*埋め込みコードをダウンロードしてください。 この `index.html` ファイルを `.zip` ファイルにする。
2. [Viverse](https://viverse.com)にサインインし、[Viverse Studio](https://studio.viverse.com)にアクセスしてください。
3. Upload Your Own Build**の下にある**Upload\*\*をクリックします。
4. 新しい世界を作る\*\*をクリックします。
5. プロジェクトの**名前**と**説明**を入力し、**作成**をクリックします。
6. コンテンツバージョン\*\*をクリックします。
7. 新しいバージョン\*\*」の下にある「**ファイルを選択**」をクリックします。 ステップ1で作成した`.zip`ファイルをアップロードし、**アップロード**をクリックします。
8. プレビューの**iframeサポート**で、**iframe設定を適用**をクリックし、プロジェクトが必要とするすべてのパーミッションを有効にします。
   - Viverseは、あなたが8th WallからダウンロードしたiFrameを、それ自身のiFrameに入れます。
9. ゲームの設定を完了し、公開します。

### ゲームジョルト

1. フルHTML\*\*埋め込みコードをダウンロードしてください。 この `index.html` ファイルを `.zip` ファイルにする。
2. [Game Jolt](https://gamejolt.com)にサインインし、[Game Jolt Store](https://gamejolt.com/games)にアクセスしてください。
3. Add Your Game\*\*をクリックしてください。
4. プロジェクトの詳細を入力し、**保存して次へ**をクリックします。
5. ゲームダッシュボードの **パッケージ** の下にある **パッケージを追加** をクリックします。
6. パッケージの編集\*\*で、**新規リリース**をクリックします。
7. ブラウザビルドのアップロード\*\*をクリックします。 ステップ1で作成した `.zip` ファイルをアップロードします。
8. ゲームの寸法を設定するか、またはゲームを画面に合わせたい場合は、\*\*Fit to screen?\*\*を選択します。
9. ゲームの設定を完了し、公開します。

### ゲームピックス

:::info[Important]
GamePixでは、外部リンクのあるゲームを許可していません。 つまり、あなたのゲームをそこで公開するには、[ホワイトレーベル・サブスクリプション](https://8th.io/nae-white-label)を購入し、8番目の壁のブランドとデフォルトのスプラッシュ画面を削除する必要があります。
:::

1. フルHTML\*\*埋め込みコードをダウンロードしてください。 この `index.html` ファイルを `.zip` ファイルにする。
2. GamePix Developer Account](https://partners.gamepix.com/join-us?t=developer)にサインアップし、[GamePix Dashboard](https://my.gamepix.com/dashboard)にアクセスしてください。
3. 新しいゲームを作る\*\*をクリックします。
4. ゲームの詳細を入力し、**作成**をクリックします。
5. Info\*\*」の「**Game Engine**」から「**HTML5-JS**」を選択します。
6. Build**の下にある**Browse File\*\*をクリックします。 ステップ1で作成した `.zip` ファイルをアップロードします。
7. ゲームの設定を完了し、公開します。

### ニューグラウンズ

1. フルHTML\*\*埋め込みコードをダウンロードしてください。 この `index.html` ファイルを `.zip` ファイルにする。
2. Newgroundsアカウント](https://www.newgrounds.com)にサインアップします。
3. 右上の矢印をクリックし、\*\*ゲーム（swf、HTML5）\*\*\*を選択します。
4. Submission File(s)\*\*の下にある、**Upload File**をクリックします。 ステップ1で作成した `.zip` ファイルをアップロードします。
5. ゲームの寸法を設定し、**タッチスクリーンフレンドリー**をチェックする。
6. ゲームの設定を完了し、公開します。

### Y8

1. フルHTML\*\*埋め込みコードをダウンロードしてください。 この `index.html` ファイルを `.zip` ファイルにする。
2. Y8](https://www.y8.com/upload)にログインする。
3. Eメールを確認し、[無料のY8ストレージアカウントを作成](https://account.y8.com/storage_account)してください。
4. Game**で**Zip**を選択し、次に**HTML5\*\*を選択します。
5. ファイルを選択\*\*をクリックします。 ステップ1で作成した `.zip` ファイルをアップロードします。 ストレージアカウントを作成していない場合は失敗します。 その場合は、**Create Storage Account**をクリックしてアカウントを作成し、**Upload Your Content to Y8**ページをリフレッシュして再度お試しください。
6. ゲームの設定を完了し、公開します。

### ポキ

1. Poki Developer Portal](https://developers.poki.com/share)にアクセスする。
2. ゲームのリンク\*\*の下にある8th Wallの経験（例：https://8w.8thwall.app/my-project）へのリンクを使用して、プロジェクトの詳細を記入してください。
3. ゲームを共有する\*\*をクリックしてください。

### コングレゲート

1. Kongregateチームにメールする [bd@kongregate.com](mailto:bd@kongregate.com). Eメールに、あなたの8th Wall体験へのリンク（例：https://8w.8thwall.app/my-project）を添付してください。

### アーマーゲーム

1. アーマーゲームズチームにメールする [mygame@armorgames.com](mailto:mygame@armorgames.com). Eメールに、あなたの8th Wall体験へのリンク（例：https://8w.8thwall.app/my-project）を添付してください。

### やみつきになるゲーム

1. フルHTML\*\*埋め込みコードをダウンロードしてください。 この `index.html` ファイルを `.zip` ファイルにする。
2. Addicting Games チームにメールする [games@addictinggames.com](mailto:games@addictinggames.com). ステップ1で作成した`.zip`ファイルをメールに添付し、[Addicting Games Developer Center](https://www.addictinggames.com/about/upload#Send)で要求されたその他の情報もすべて添付してください。

### 遅延

1. メールでのお問い合わせは、ラグジュアリーチーム[contact@lagged.com](mailto:contact@lagged.com)まで。 Eメールに、あなたの8th Wall体験へのリンク（例：https://8w.8thwall.app/my-project）を添付してください。
2. 承認されたら、**招待コード**を使って[Laggedアカウントにサインアップ](https://lagged.dev/signup)し、ゲームをアップロードします。
