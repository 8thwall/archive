# ネイティブアプリのエクスポート

:::info[Beta 特集］
Native App Exportは現在ベータ版で、**Android & iOS**ビルドに限定されています。 デスクトップとヘッドセットのサポートは間もなく。
:::

Native App Exportを使用すると、Studioプロジェクトをスタンドアロンのアプリケーションとしてパッケージ化できます。

## 要件 {#requirements}

iOS用であれAndroid用であれ、エクスポートを試みる前に、プロジェクトが少なくとも一度はウェブ用に正常にビルドされたことを確認してください。

### iOS

iOS用のネイティブエクスポートは、ARと3Dプロジェクトで利用可能です。 あなたのアプリケーションは**サポートしません**：

- プッシュ通知
- アプリ内課金

### Android

Android用のネイティブエクスポートは、非AR、3Dのみのプロジェクトでのみ利用可能です。 あなたのプロジェクトは、**使用してはならない**：

- カメラまたはAR機能
- GPS
- 仮想または物理キーボード
- プッシュ通知
- アプリ内課金
- ビデオテクスチャ
- MediaRecorder API
- CSS
