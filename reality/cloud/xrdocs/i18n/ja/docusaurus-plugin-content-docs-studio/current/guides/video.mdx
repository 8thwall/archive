# ビデオ

## はじめに

{frontMatter.description} デフォルトでは、すべてのビデオテクスチャとUIビデオのデフォルトは**ミュート自動再生**です。 ビデオの動作をカスタマイズするには、[`VideoControls`](/api/studio/ecs/video-controls) を使用します。

## ビデオの追加

エンティティにビデオテクスチャを適用するには、アセットから任意のテクスチャマップドロップダウンでビデオを選択します。

ビデオテクスチャは、プリミティブのスタンダードマテリアルとアンライトマテリアルにのみ適用できます。
GLBエンティティの場合、ビデオマテリアルを使ってビデオテクスチャを適用することができ、これはモデルのすべてのカラーマップを上書きします。
