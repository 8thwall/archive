## カメラの追加

カメラは新しいスタジオプロジェクトにデフォルトで用意されていますが、スペース階層の+Objectドロップダウンを使ってシーンにカメラを追加することもできます。 さらに、カメラはスタジオのAdd Componentインターフェイスまたはコードでエンティティに追加できます。

![camera-new-object.png](/images/studio/camera-new-object.png)

## アクティブカメラの変更

カメラのプロパティで［アクティブカメラに設定］ボタンをクリックします。

![camera-set-active.png](/images/studio/camera-set-active.png)

## カメラタイプの変更

スタジオには3種類のカメラがあります：3Dのみ、フェイス、ワールドです。 これらのカメラタイプはそれぞれ設定が異なる。 AR体験にはフェイスカメラやワールドカメラが必要だ。 \*\*AR体験の構築についての詳細は、XRガイドのセクションをご覧ください。
