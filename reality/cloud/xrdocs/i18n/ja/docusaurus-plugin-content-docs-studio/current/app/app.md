# 第8回ウォールデスクトップアプリ

:::info[Public ベータ]
8thウォールのデスクトップアプリはパブリックベータ版\*\*であり、将来のリリースで機能が変更される可能性があります。 皆様のフィードバックをお待ちしております。また、[デスクトップアプリベータユーザーのための専用サポートフォーラム](https://forum.8thwall.com/c/desktop-beta/17)もご用意しております。
:::

8th Wallデスクトップアプリは、8th Wallをブラウザを超えて、あなたのローカルワークフローにもたらします。 自分のマシンからプロジェクトを立ち上げることができ、しかも8th Wallですでに知っていること全てにつながることができる。 つまり、自分のコード・エディターでより速く動くことができ、変更を保存してすぐに結果を見ることができ、実験や反復をしながらフローにとどまることができる。
