# はじめに

:::warning[Important 更新情報]
8番目の壁は来年中に閉鎖される - [今すぐアップデートの全文を読む](/migration)。
:::

{frontMatter.description} Studioを使用すれば、魅力的なWebARエクスペリエンスやインタラクティブな3Dゲームなどを簡単にリアルタイムで作成し、モバイルデバイス、デスクトップ、高度なヘッドセットにシームレスに展開できます。

Studioは、WebAR、WebVR体験、Webブラウザで直接動作する3Dゲームを作成、コラボレーション、公開するための完全なソリューションを提供します。

![Intro-Image](/images/studio/introducing-studio.png)

Studioはブラウザベースのツールで、ソフトウェアのダウンロードは不要です。 スタジオを使用するには、[既存の8th Wallアカウントにログイン](https://www.8thwall.com/login)、またはまだアカウントをお持ちでない場合は、[新規アカウントを作成](https://www.8thwall.com/get-started)してください。
