# 公開プロジェクトを見る

1. プロジェクトダッシュボードに移動します。 スタジオから、左上のハンバーガーメニューをクリックできます。 プロジェクトダッシュボードの**QR 8.code**セクションに、公開プロジェクトのURLが**8th.io**ショートリンクと関連するQRコードとともに表示されます。

2. 携帯端末でQRコードをスキャンすると、公開プロジェクトが表示されます。
