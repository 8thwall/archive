# コンポーネント

## はじめに

{frontMatter.description} 各コンポーネントは再利用可能な機能ブロックであり、エンティティにアタッチすることで、特定の動作や特性を示すことができます。 コンポーネントは、視覚的な外観、物理的なプロパティ、入力処理、またはカスタムゲームロジックを定義することができます。 複数のコンポーネントを組み合わせることで、リッチな動作を持つ複雑なエンティティを作成できます。

## コンポーネント

コンポーネントは、エンティティに機能を与えるビルディング・ブロックである。 エンティティが空白のオブジェクトを表すのに対し、コンポーネントは、そのオブジェクトが世界とどのように相互作用し、どのように見え、どのように振る舞うかを定義する。 8th Wall Studioでは、ビルトインコンポーネントを使用したり、独自のカスタムコンポーネントを作成して、ゲームのユニークなビヘイビアを定義することができます。
