# エンティティ

## はじめに

{frontMatter.description} エンティティは、それ自体では動作や外観を持たず、単にコンポーネントをアタッチできるコンテナとして機能する。 これらのコンポーネントは、位置、視覚的な外観、物理システムとの相互作用など、エンティティの動作や特性を定義します。

## エンティティ

エンティティは、8th Wall Studioのあらゆるゲームやシミュレーションのバックボーンを形成します。 さまざまなコンポーネントを組み合わせることで、複雑なオブジェクトやシステム、インタラクションを作成できる。

**エンティティは、エンティティIDまたはeid.**と呼ばれる一意の64ビット整数で表されます。
