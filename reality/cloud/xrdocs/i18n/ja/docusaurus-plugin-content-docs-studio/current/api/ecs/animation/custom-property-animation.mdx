# カスタムプロパティアニメーション

## 商品説明

このコンポーネントは、エンティティにアニメーションを作成します。

## プロパティ一覧

| Property       | Type    | 初期値       | 商品説明                                                                                                            |
| -------------- | ------- | --------- | --------------------------------------------------------------------------------------------------------------- |
| 属性             | ストリング   | ''        | アニメートする属性を選択（位置、スケール、マテリアルなど）                                                                                   |
| プロパティ          | ストリング   | ''        | アニメートする個々のプロパティ（例: ベクトル属性のx、y、z）                                                                                 |
| ターゲット          | イード     | undefined       | アニメートする対象のオブジェクト。 指定されていない場合、アニメーションはコンポーネントがアタッチされているオブジェクト上で実行されます。                                           |
| より             | オブジェクト  | undefined       | アニメーションの開始状態                                                                                                    |
| への             | オブジェクト  | undefined       | アニメーションの終了状態                                                                                                    |
| オートフロム         | ブーリアン   | false     | 有効にすると、'from' プロパティを無視し、アニメーション開始時のオブジェクトの状態からアニメーションを行う。                                                       |
| 存続期間           | 番号      | 0         | アニメーションの実行時間（ミリ秒）。                                                                                              |
| ループ            | ブーリアン   | false     | 有効にすると、アニメーションを繰り返す。                                                                                            |
| リバース           | ブーリアン   | false     | ループが設定されている場合、逆再生を行うかどうか。                                                                                       |
| イージーイン         | ブーリアン   | false     | 有効にすると、直線補間の代わりにイージング機能が適用される。                                                                                  |
| イーズアウト         | ブーリアン   | false     | 有効にすると、直線補間の代わりにイージング機能が適用される。                                                                                  |
| イージング機能        | ストリング   | ''        | 二次（デフォルト）、三次、四次、五次、正弦波、指数、円形、弾性、バック、バウンスから選択                                                                    |

## 関数一覧

<ComponentFunctions
    name="CustomPropertyAnimation"
    properties={{
        attribute: "attribute",
        property: "property",
        from: { x: 0, y: 0, z: 0 },
        to: { x: 10, y: 10, z: 10 },
        autoFrom: false,
        duration: 1000,
        loop: false,
        reverse: false,
        easeIn: false,
        easeOut: false,
        easingFunction: "Cubic"
    }}
    mutate={`cursor.duration = 1000;
    cursor.loop = true;
    return false;`}
/>
