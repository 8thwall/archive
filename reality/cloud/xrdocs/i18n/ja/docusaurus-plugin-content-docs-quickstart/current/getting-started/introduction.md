# はじめに

8th Wallは、WebAR、WebVR体験（
）、Webブラウザで直接実行される3Dゲーム（アプリ不要）を作成、コラボレーション、公開するための完全なソリューションを提供します。

\*\*クレジットカードは必要ありません。

## 仕組み {#how-it-works}

標準に準拠したJavaScriptとWebGLを使用して構築された8th Wall Webは、8th WallのSLAM（Simultaneous Localization and Mapping）エンジンの完全な実装（
）であり、ブラウザ上のリアルタイムのウェブARのために超最適化（
）されている。 AR機能には、ワールドトラッキング、イメージターゲット、フェイスエフェクト、
、スカイセグメンテーションなどがある。

**Studio**は、リアルタイムのXRビジュアル・エディターとゲーム・エンジンであり、
Webのスピードとパワーを、超没入型3DとXR体験を作成するために必要なツールと組み合わせています。 Studioは、ビジュアル3Dエディター
インターフェイスを備えており、デバイスを問わずXRおよびウェブゲームを作成できます。 物理学のシステム、カメラ・ライティング＆オーディオのシステム（
）、アニメーション・マテリアル・パーティクル、プレイヤー・アクションのトラッキングなど、まったく新しいエンジンが搭載されている。
Studioを使用すれば、魅力的なWebAR体験や3Dゲームなどをリアルタイムで作成し、
をモバイルデバイス、デスクトップ、高度なヘッドセットに簡単に展開できます。 [スタジオについて詳しくはこちら](/studio)。

ビルトインホスティングにより、
パスワードで保護されたステージング環境を含む、8th Wallの信頼性が高く安全なグローバルネットワーク上でホストされている複数の
配備状態にプロジェクトを公開することができます。

8th Wallのデベロッパー・プラットフォームは、多くの便利な機能を提供しています。そこで、
、核となる機能のいくつかを簡単に見てみましょう：
