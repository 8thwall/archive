# サンプルプロジェクトを見る

8th Wallは、8th Wallチームと開発者コミュニティの両方によって作成された、すぐに使えるサンプルプロジェクトのコレクションを提供しています。 これらのプロジェクトは、8th Wallのプラットフォームを使って独自の3DやAR体験を素早く構築し、立ち上げるための参考資料や出発点として役立ちます。 あなたの開発プロセスを加速し、ベストプラクティスを学び、新しいアイデアを閃くために、これらの例を探求してください。

8thウォールプロジェクトライブラリーには、サンプルプロジェクトや再利用可能なモジュールが豊富に揃っています。

各サンプルプロジェクトでは、以下のことができます：

- ソースコードを見てベストプラクティスを学ぶ
- ブラウザで直接3DまたはWebARで体験をプレビューする
- プロジェクトをクローンしたり、自分のワークスペースにモジュールをフォークして、簡単にカスタマイズできます。

:::tip
ライブラリーの右上にある**All projects**ドロップダウンを使って、8th WallとCommunityのサンプルプロジェクトを絞り込んでください。
:::

## 第8回ウォール・サンプル・プロジェクト

8th Wallサンプルプロジェクトは、8th Wallチームによって開発・管理されています。 これらのプロジェクトは、プラットフォームのコア機能を実証し、推奨されるベストプラクティスに従っており、あなた自身の経験のための強固な基盤を提供します。

![8th Wall Sample Projects](/images/quickstart/samples-8w.jpg)

## コミュニティ・サンプル・プロジェクト

コミュニティサンプルプロジェクトは、グローバルな8th Wallコミュニティの開発者によって寄稿されています。 これらのプロジェクトは、さらなるインスピレーションと洞察を提供するが、8th Wallチームによって管理されているわけではない。 コミュニティ・プロジェクトについて何か問題が発生したり、質問がある場合は、プロジェクト作成者に直接お問い合わせください。

コミュニティサンプルプロジェクトを閲覧するには、プロジェクトライブラリの右上にあるドロップダウンを使い、**コミュニティプロジェクト**を選択してください。
