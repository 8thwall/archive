# 新しいプロジェクトを作成する

1. ログインしたホームページまたはワークスペースのダッシュボードから、"新規プロジェクト "をクリックします。

![StartNewProject](/images/console-workspace-start-project.jpg)

2. 複数のワークスペースに所属している場合は、このプロジェクトのワークスペースを選択します。

3. プロジェクトのURLを入力します。

:::warning
プロジェクトURLは、デフォルトのプロジェクトURLと注目のプロジェクトページのURL（有効になっている場合）を生成するために使用されます。  **変更することはできません**。 プロジェクトのURLを変更するには、プロジェクトをクローンし、新しいURLを指定する必要があります。
:::

![ProjectUrl](/images/account/project-create.jpg)

4. 作成\*\*をクリックします。
