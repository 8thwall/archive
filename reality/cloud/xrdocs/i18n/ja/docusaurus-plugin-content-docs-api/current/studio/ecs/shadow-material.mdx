# シャドウマテリアル

## 説明

このコンポーネントは、ジオメトリを持つエンティティに影のみをレンダリングするマテリアルを作成します。

## プロパティ

| プロパティ      | タイプ    | デフォルト   | 説明                                                                |
| ---------- | ------ | ------- | ----------------------------------------------------------------- |
| r          | 番号     | 0       | 素材の赤チャンネルの値 [0...255]                                             |
| g          | 番号     | 0       | 素材のグリーン・チャンネル値 [0...255]                                          |
| b          | 番号     | 0       | 素材の青チャンネル値 [0...255]                                              |
| 不透明性       | 番号     | 0       | 素材の全体的なアルファ値／透明度 [0...1]                                          |
| サイド        | ストリング  | フロント    | 面のどの面をレンダリングするか。 フロント、バック、ダブルからお選びください。                           |
| デプス試験      | ブーリアン  | 真の      | この素材をレンダリングする際に深さをテストするかどうか                                       |
| デプスライト     | ブーリアン  | 真の      | この素材のレンダリングが深度バッファに影響を与えるかどうか                                     |

## 機能

<ComponentFunctions
    name="ShadowMaterial"
    properties={{
        r: 0,
        g: 0,
        b: 0,
        opacity: 0.5,
    }}
    mutate={`カーソル.opacity = 0.5;
    カーソル.r = 128;
    カーソル.g = 128;
    カーソル.b = 128;
    return false;`}
/>
