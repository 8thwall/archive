# スペース

{frontMatter.description}

## 機能

### ロードスペース

IDまたは名前で指定されたスペースをロードする。 新しくロードされたスペースは、アクティブなスペースとして古いスペースに取って代わります。 古いスペース（とそれに含まれるスペース）のオブジェクトはデスポーンされ、新しいスペース（とそれに含まれるスペース）がスポーンされます。 指定されたIDまたは名前にマッチするものがない、または複数マッチする場合はエラーを投げる。

```ts
world.spaces.loadSpace(idOrName: string) // -> void
```

### リストスペース

すべてのSpaceのSpaceDataを返します。

```ts
world.spaces.listSpaces() // -> SpaceData[] | undefined
```

### アクティブスペース

アクティブなSpaceのSpaceDataを返します。

```ts
world.spaces.getActiveSpace() // -> SpaceData | undefined
```

## スペースデータ

| プロパティ      | タイプ    | 説明                                     |
| ---------- | ------ | -------------------------------------- |
| 名称         | ストリング  | スペース名。 ユニークでなければならない                   |
| アイドル       | ストリング  | スペースID                                 |
| 産卵         | ブーリアン  | スペースが現在スポーンされているかどうかを示す                |
