# シャドウマテリアル

## 概要

このコンポーネントは、ジオメトリを持つエンティティに影のみをレンダリングするマテリアルを作成します。

<ShadowMaterialTable />

## 機能

<ComponentFunctions
    name="ShadowMaterial"
    properties={{
        r: 0,
        g: 0,
        b: 0,
        opacity: 0.5,
    }}
    mutate={`cursor.opacity = 0.5;
    cursor.r = 128;
    cursor.g = 128;
    cursor.b = 128;
    return false;`}
/>
