# 資産

## 説明

このライブラリには、資産管理を行う関数が含まれています。

## 種類

### アセットリクエスト

| パラメータ | タイプ   | 説明       |
| ----- | ----- | -------- |
| アイドル  | イード   | 未使用      |
| url   | ストリング | アセットのURL |

### 資産統計

| パラメータ | タイプ | 説明               |
| ----- | --- | ---------------- |
| 保留    | 番号  | 保留中の資産の数         |
| 完全    | 番号  | ローディングを完了したアセット数 |
| 合計    | 番号  | ロードされる資産の総数      |

### 資産

| パラメータ   | タイプ   | 説明             |
| ------- | ----- | -------------- |
| データ     | 小塊    | 資産データ          |
| リモートURL | ストリング | データの取得元        |
| ローカルURL | ストリング | データから構築された url |

## 機能

### ロード

アセットをロードする

```ts
ecs.assets.load(assetRequest: AssetRequest) // -> プロミス<Asset>
```

### クリア

ロードされたアセットからリクエストアセットをクリアします。

```ts
ecs.assets.clear(assetRequest: AssetRequest) // -> プロミス<Asset>
```

### 統計

アセットの読み込みに関する統計情報を取得します。

```ts
ecs.assets.getStatistics() // -> AssetStatistics
```
