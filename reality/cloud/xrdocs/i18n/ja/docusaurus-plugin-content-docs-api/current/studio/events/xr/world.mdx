# ワールド・エフェクト・イベント

## イベント {#events}

### リアリティ追跡ステータス

このイベントは、エンジンが始動し、トラッキングの状態や理由が変化したときにWorld Effectsから発行される。

#### プロパティ {#properties}

| プロパティ | タイプ | 説明                                  |
| ----- | --- | ----------------------------------- |
| ステータス | 文字列 | `LIMITED`または`NORMAL`のいずれか。          |
| 理由    | 文字列 | `INITIALIZING`または`UNDEFINED` のいずれか。 |
