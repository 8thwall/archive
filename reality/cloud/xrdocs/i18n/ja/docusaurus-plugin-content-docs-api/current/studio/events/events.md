## イベントカテゴリー

- [XR Events](xr)：reality`や`facecontroller\\`のような8th Wallのカメラパイプラインモジュールによって発せられるイベントで、Image Target tracking、VPS location spawning、Face detectionのようなものをカバーする。

- [アセットイベント](assets)：アセットのロードや再生イベントなど、アセットに関連するイベント。

- [カメライベント](camera)：アクティブなカメラの切り替え、XR カメラ属性の編集、アクティブなカメラのエンティティの変更など、カメラの状態変更に関連するイベント。

- [一般イベント](general)：スタジオ体験内で発生するワールドレベルのコアイベント。

- [入力イベント](input)：タッチ、ジェスチャー、UIクリックイベントなど、ユーザーインタラクションによってトリガーされるイベント。 単純なタップと複雑なマルチタッチジェスチャーの両方をカバー。 単純なタップと複雑なマルチタッチジェスチャーの両方をカバー。

- [物理イベント](physics)：衝突の開始や終了など、エンティティ間で物理的相互作用が発生したときに発せられるイベント。
