# フライコントローラー

## 説明

このコンポーネントにより、オブジェクトをコントロールし、空間を自由に動き回ることができる。

## プロパティ

テストしたいサンプルシーンを **Scenes in Build** にドラッグします。 **Home** シーンがリストの一番上にあることを確認します。

<ComponentFunctions
