# XR8.XrController.updateCameraProjectionMatrix()

\`XR8.XrController.updateCameraProjectionMatrix({ cam, origin, facing })\`\\`.

## 説明 {#description}

シーンの表示ジオメトリとシーン内のカメラの開始位置をリセットする。 表示ジオメトリは、仮想シーン内のオブジェクトの位置を、カメラ画像内の対応する位置の上に適切に重ね合わせるために必要です。 開始位置は、セッションの開始時にカメラが置かれる位置と向きを指定します。 ディスプレイのジオメトリは、仮想シーン内のオブジェクトの位置を、カメラ画像内の対応する位置の上に適切にオーバーレイするために必要です。 開始位置は、セッションの開始時にカメラが置かれる位置と向きを指定します。

## パラメータ（全てオプション） {#parameters-all-optional}

| なし                                                                    | タイプ                                                                               | デフォルト                                                                                                           | 説明                      |
| --------------------------------------------------------------------- | --------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- | ----------------------- |
| cam [オプション］                                 | `{pixelRectWidth, pixelRectHeight, nearClipPlane, farClipPlane}`. | {nearClipPlane: 0.01, farClipPlane: 1000.0}\\` | カメラの設定。                 |
| origin [オプション］                              | {x, y, z}\\`                                                                     | {x: 0, y: 2, z: 0}\\`                                          | シーン内のカメラの開始位置。          |
| facing [Optional] | w, x, y, z}\\`.                                                  | `{w:1, x: 0, y：0, z: 0}`                                                                                        | シーン内のカメラの開始方向（クォータニオン）。 |

cam\\`には以下のパラメータがある：

| パラメータ         | タイプ | 説明                         |
| ------------- | --- | -------------------------- |
| ピクセル矩形幅       | 番号  | カメラのフィードを表示するキャンバスの幅。      |
| ピクセル矩形の高さ     | 番号  | カメラのフィードを表示するキャンバスの高さ。     |
| nearClipPlane | 番号  | シーンオブジェクトが見える、カメラに最も近い距離。  |
| ファークリップ平面     | 番号  | シーンオブジェクトが見える、カメラから最も遠い距離。 |

## 戻り値 {#returns}

なし

## 例 {#example}

```javascript
XR8.XrController.updateCameraProjectionMatrix({
  origin: { x: 1, y: 4, z: 0 },
  facing: { w: 0.9856, x: 0, y: 0.169, z: 0 }
})
```
