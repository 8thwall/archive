# XR8.removeCameraPipelineModule()

`XR8.removeCameraPipelineModule(moduleName)`

## 説明 {#description}

カメラパイプラインからモジュールを削除する。

## パラメータ {#parameters}

| パラメータ      | タイプ      | 説明        |
| ---------- | -------- | --------- |
| moduleName | `String` | モジュールの名前。 |

## を返す {#returns}

なし

## 例 {#example}

```javascript
XR8.removeCameraPipelineModule('reality')
```
