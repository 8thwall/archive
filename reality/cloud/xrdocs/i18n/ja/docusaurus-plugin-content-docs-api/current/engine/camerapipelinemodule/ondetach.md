# オンデタッチ()

`onDetach: ({framework})`

## 説明 {#description}

`onDetach`は、モジュールが最後にフレーム更新を受信した後に呼ばれる。 これは、エンジンが停止した後か、モジュールがパイプラインから手動で取り外された後のどちらか早い方である。

## パラメータ {#parameters}

| パラメータ     | 説明                                       |
| --------- | ---------------------------------------- |
| framework | イベントをディスパッチするための、このモジュールのフレームワークバインディング。 |
