# XR8.clearCameraPipelineModules()

`XR8.clearCameraPipelineModules()`

## 説明 {#description}

カメラループからすべてのカメラパイプラインモジュールを取り外します。

## パラメータ {#parameters}

なし

## を返す {#returns}

なし

## 例 {#example}

```javascript
XR8.clearCameraPipelineModules()
```
