# onRemove()

`onRemove: ({framework})`

## 説明 {#description}

`onRemove`は、モジュールがパイプラインから削除されたときに呼び出される。

## パラメータ {#parameters}

| パラメータ     | 説明                                       |
| --------- | ---------------------------------------- |
| framework | イベントをディスパッチするための、このモジュールのフレームワークバインディング。 |
