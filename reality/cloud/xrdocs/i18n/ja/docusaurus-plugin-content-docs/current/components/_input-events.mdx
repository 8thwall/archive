## イベント

### UI_CLICK

同じUI要素に対してプレス(`UI_PRESSED`)とリリース(`UI_RELEASED`)の両方が発生した場合に発せられる。 このイベントは、完全なクリックまたはタップジェスチャを表します。 これは、両方のインタラクションが重なった要素にディスパッチされ、通常、ボタンの有効化やアクションのトリガーなど、ユーザーの意図を確認するために使用される。
このイベントは、開始(押された)と終了(リリースされた)eidの共通の祖先の最下位に発送されます。 マウスの動きはクリックイベントには影響しません。

イベントのペイロードは `Position` 型である。

#### 例

<EventExample
  ns="ecs.input"
  event="UI_CLICK"
  callbackBody={`console.log('UI click: ', event.data.x, event.data.y)`}
/>

### UI_PRESSED

ユーザーがUIエレメント上でタッチまたはポインターを下に動かすインタラクションを開始したときに発せられる。 直接押された要素にのみディスパッチされ、親要素にはバブルしない。 このイベントは、ユーザー入力の開始時に即時の視覚的フィードバックやインタラクション状態（ボタンのハイライトやアニメーションなど）をトリガーするのに便利です。

イベントのペイロードは `TouchEvent` 型である。

#### 例

<EventExample
  ns="ecs.input"
  event="UI_PRESSED"
  callbackBody={`console.log('UI pressed: ', event.data.position)`}
/>

### UI_HOVER_START

マウスが UI 要素の上にマウスポインターを合わせ始めたときに発生します。

イベントのペイロードは `HoverEvent` 型である。

#### 例

<EventExample
  ns="ecs.input"
  event="UI_HOVER_START"
  callbackBody={`console.log('UI hover start: ', event.data.x, event.data.y)`}
/>

### UI_HOVER_END

マウスが UI 要素上にホバリングを停止した場合に発生します。

イベントのペイロードは `HoverEvent` 型である。

#### 例

<EventExample
  ns="ecs.input"
  event="UI_HOVER_END"
  callbackBody={`console.log('UI hover end: ', event.data.x, event.data.y)`}
/>

:::note

- 複数のタッチポイントを同時にアクティブにすることができる。
- 一度に1つのタッチジェスチャー（シングルまたはマルチタッチ）のみがアクティブとして認識されます。
  :::
  :::
  :::

:::info
タッチイベントにターゲットがある場合、そのターゲット上で発せられ、親要素に伝搬し、最終的にはグローバルレベルにまで伝搬する。 つまり、親オブジェクトのタッチリスナーは、そのすべての子エレメントからのイベントを捕捉することになる。
:::

### スクリーン・タッチ・スタート

ユーザーが最初にスクリーンまたはターゲット・オブジェクトをタッチまたはクリックしたときに発する。

イベントのペイロードは `TouchEvent` 型である。

#### 例

<EventExample
  ns="ecs.input"
  event="SCREEN_TOUCH_START"
  callbackBody={`console.log('Screen touch start: ', event.data.position)`}
/>

イベントのペイロードは `TouchEvent` 型である。

#### 例

  callbackBody={`console.log('Screen touch move: ', event.data.position)`}
/>

### スクリーン_タッチ_終了

ユーザーがクリックを止めたり、画面から指を離したりしたときに発する。

イベントのペイロードは `TouchEvent` 型である。

#### 例

<EventExample
  ns="ecs.input"
  event="SCREEN_TOUCH_END"
  callbackBody={`console.log('Screen touch end: ', event.data.position)`}
/>

:::info
ジェスチャー・イベントは、ユーザーが携帯電話の画面上で「ジェスチャー」をしたときに発せられる。 ジェスチャーとは、複数の指を必要とする動作のこと。 ジェスチャー・イベントは、ユーザーが携帯電話の画面上で「ジェスチャー」をしたときに発せられる。 ジェスチャーとは、複数の指を必要とする動作のこと。 ユーザーが "ズーム "アクション（2本の指が互いに離れていく）で開始し、スクリーンにもう1本指を加えた場合、"ズーム "ジェスチャーは終了し、3本の指で新しいジェスチャーが開始されます。
:::
