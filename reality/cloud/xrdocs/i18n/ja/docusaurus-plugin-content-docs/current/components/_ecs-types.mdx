以下のデータ型は、カスタム・コンポーネントのスキーマ・プロパティや特定の型への参照を作成するのに便利です。

| タイプ         | 説明                          |
| ----------- | --------------------------- |
| ECS.EID     | エンティティ・リファレンス               |
| ecs.f32     | 32ビット浮動小数点数                 |
| ecs.f64     | 64ビット浮動小数点数                 |
| ecs.i32     | 32ビット整数                     |
| ecs.ui8     | 8ビット符号なし整数                  |
| ecs.ui32    | 32ビット符号なし整数                 |
| ecs.string  | ストリング                       |
| ecs.boolean | ブーリアン                       |
