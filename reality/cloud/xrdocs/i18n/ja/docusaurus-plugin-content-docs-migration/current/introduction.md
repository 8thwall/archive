# 私たちと一緒に建設していただきありがとうございます。

\*\*8th Wallは来年中に閉鎖され、ホストプロジェクトは2027年2月まで活動する。

![](/images/migration/intro.png)

7年以上という素晴らしい年月を経て、我々は8th Wallのサービスを終了するという難しい決断を下した。

この章は幕を閉じようとしているが、私たちは技術の遺産を継続させることに全力を注いでいる。 我々は、8thウォールのテクノロジーとドキュメンテーションの主要コンポーネントをオープンソース化し、コミュニティが利用できるようにするオプションを積極的に模索している。

私たちの目標は、現在進行中の作業を維持し、輸出を支援し、次世代のビルダーに技術を開放することによって、この移行を明確かつ予測可能なものにすることである。

## 何が起きているのか

ライブ・プロジェクト、ホストされた体験、アクティブなキャンペーンは、2027年2月28日までオンラインで完全に運用される。 一朝一夕には何も変わらない。 開発者は、プラットフォームが完全にシャットダウンする前にキャンペーンを完了し、作品をエクスポートする時間がある。

8th Wall Studio、Cloud Editor、Asset Labを含む全ての8th Wall製品とサービスは、段階的に閉鎖されます：

このスケジュールにより、既存のすべてのプロジェクトとキャンペーンが、ホスティングウィンドウの全期間を通じて継続されます。 この期間中、どのコンポーネントがオープンソース化されるのか、また、その詳細が明らかになり次第、リリースのタイムラインについても最新情報をお伝えします。

## ありがとう

第8の壁は、拡張現実はどこでも、誰にでも使えるべきだというシンプルなアイデアから始まった。

開発者、アーティスト、ストーリーテラーたちが、1つ1つのプロジェクトで未来を作り上げたのです。

ともに作り上げたものを誇りに思うと同時に、我々を信頼してくれたことに深く感謝している。 このビジョンを信じていただき、ARと3Dの可能性の限界に挑戦していただき、この7年間を忘れられないものにしていただき、ありがとうございました。
