# プロジェクトのエクスポート

## コード輸出

コード・エクスポートがスタジオとクラウド・エディターの両方のプロジェクトで利用できるようになりました。 これにより、現在8th Wallに存在するプロジェクトの完全なスナップショットをダウンロードすることができます。 このバージョンでは、**ワンクリックでコード、アセット、イメージターゲットをダウンロードできます**。

Code Exportは、オープンソースへの移行を進める際や、他のプラットフォームの利用を検討する際に、即座にアクセスしたりバックアップを取ったりできるように設計されています。

:::warning[Note]
このバージョンはビルド可能なプロジェクトを生成しない。 エクスポートされたコードをそのまま実行したり、デプロイしたりすることはできない。 建設可能な輸出はもうすぐだ。
:::

### プロジェクト詳細

Studioプロジェクトのコピーをダウンロードするには

1. Studioプロジェクトを開き、**Publish**をクリックします。

![](/images/migration/studio-publish.png)

2. エクスポート**セクションで**コード\*\*を選択し、**エクスポート**ボタンをクリックします。

![](/images/migration/studio-export-code.png)

3. イメージターゲット、コード、ファイルの入った.zipファイルが届きます。

![](/images/migration/studio-files.png)

### クラウド・エディター

Cloud Editorプロジェクトのコピーをダウンロードする：

1. Editorプロジェクトを開き、**Publish**をクリックする。

![](/images/migration/editor-publish.jpg)

2. エクスポート**セクションで**コード\*\*を選択し、**エクスポート**ボタンをクリックします。

![](/images/migration/editor-export-code.png)

3. イメージターゲット、コード、ファイルの入った.zipファイルが届きます。

![](/images/migration/editor-files.png)

## ビルド可能なコード・エクスポート

数週間後には、Buildable Code Exportをリリースする予定だ。 このバージョンには、あなたのプロジェクトのオフラインバージョンをビルドし、実行するために必要なものがすべて含まれています：

* プロジェクト・コードと資産
* イメージターゲット
* ビルド構成
* XRエンジン
* XRエクストラ
* ランディングページ

しかし、オフラインでの開発では、クラウドに依存する機能や、私たちが配布するライセンスを持っていない機能などは\*\*サポートされません：

* VPS / 地図
* ハンドトラッキング
* モジュール / バックエンド
