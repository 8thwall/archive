# XR8.XrConfig.camera()

Enumeration

## Description {#description}

Desired camera to use.

## Properties {#properties}

Property | Value | Description
-------- | ----- | -----------
FRONT | `'front'` | Use the front facing / selfie camera.
BACK | `'back'` | Use the rear facing / back camera.
