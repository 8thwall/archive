# xrlefteyebrowraised

## Description {#description}

This event is emitted by [`xrface`](/legacy/api/aframe/#face-effects) when a tracked face's left eyebrow is raised from its initial position when the face was found.

`xrlefteyebrowraised.detail : {id}`

Property  | Type | Description
--------- | ---- | -----------
id | `Number` | A numerical id of the located face.
