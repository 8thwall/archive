# xrrighteyebrowraised

## Description {#description}

This event is emitted by [`xrface`](/legacy/api/aframe/#face-effects) when a tracked face's right eyebrow is raised from its position when the face was found.

`xrrighteyebrowraised.detail : {id}`

Property  | Type | Description
--------- | ---- | -----------
id | `Number` | A numerical id of the located face.
