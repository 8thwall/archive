# xrmeshlost

## Description {#description}

This event is emitted by [`xrweb`](/legacy/api/aframe/#world-tracking-image-targets-andor-lightship-vps) when `recenter()` is called.

`xrmeshlost.detail : { id }`

Property  | Description
--------- | -----------
id | An id for this mesh that is stable within a session
