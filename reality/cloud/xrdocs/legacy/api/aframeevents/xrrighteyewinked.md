# xrrighteyewinked

## Description {#description}

This event is emitted by [`xrface`](/legacy/api/aframe/#face-effects) when a tracked face's right eye closes and opens within 750ms while the left eye remains open.

`xrrighteyewinked.detail : {id}`

Property  | Type | Description
--------- | ---- | -----------
id | `Number` | A numerical id of the located face.
