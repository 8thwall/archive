# xrrighteyebrowlowered

## Description {#description}

This event is emitted by [`xrface`](/legacy/api/aframe/#face-effects) when a tracked face's right eyebrow is lowered to its initial position when the face was found

`xrrighteyebrowlowered.detail : {id}`

Property  | Type | Description
--------- | ---- | -----------
id | `Number` | A numerical id of the located face.
