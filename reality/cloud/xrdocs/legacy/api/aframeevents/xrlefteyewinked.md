# xrlefteyewinked

## Description {#description}

This event is emitted by [`xrface`](/legacy/api/aframe/#face-effects) when a tracked face's left eye closes and opens within 750ms while the right eye remains open.

`xrlefteyewinked.detail : {id}`

Property  | Type | Description
--------- | ---- | -----------
id | `Number` | A numerical id of the located face.
