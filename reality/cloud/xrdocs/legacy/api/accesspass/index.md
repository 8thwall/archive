# AccessPass

## Description {#description}

Provides a module for monetizing your Web AR and Web VR experience. This is only available for 8th
Wall Hosted projects, and requires the [Payments Module](/legacy/guides/advanced-topics/monetize-with-8th-wall-payments).

```ts
import {AccessPass} from 'payments'
```

## Functions {#functions}

Function | Description
-------- | -----------
[requestPurchaseIfNeeded](requestpurchaseifneeded.md) | Opens a checkout window where the customer can securely make a payment for the provided access pass.
