# XR8.initialize()

`XR8.initialize()`

## Description {#description}

Returns a promise that is fulfilled when the AR Engine's WebAssembly is initialized.

## Parameters {#parameters}

None

## Returns {#returns}

None

## Example {#example}

```javascript
XR8.initialize().then(() => console.log(XR8.version())
```
