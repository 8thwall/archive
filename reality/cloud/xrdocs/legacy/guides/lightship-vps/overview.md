# Lightship VPS Overview {#lightship-vps-overview}
:::info
**VPS localization is not currently previewable in the
[Simulator](/legacy/getting-started/quick-start-guide/#simulator).**
:::

With Niantic’s Visual Positioning System (VPS), developers can create rich Location AR experiences 
that overlay virtual content on the physical world. WebAR content can be anchored to real-world 
locations, enabling virtual objects to interact with their physical surroundings. This makes the 
AR experience feel more personal, meaningful, and immersive, and gives users new reasons to explore 
the world around them.
