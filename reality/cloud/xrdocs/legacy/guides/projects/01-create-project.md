# Create Legacy Editor Project 

1. Login to your 8th Wall account, and click "Go to My Projects":

![GoToMyProjects](/images/account/go-to-my-projects.jpg)

2. Expand your workspace, if not already open.

![SelectWorkspace](/images/account/select-workspace.jpg)

3. Scroll to the bottom of the page, and click **Build with Legacy Editor**

![BuildWithLegacyEditor](/images/legacy/build-with-legacy-editor.jpg)

4. Enter the project URL and click **Create**.

![CreateLegacyProject](/images/legacy/legacy-create-project.jpg)

:::info
An **App Key** is used to develop locally and for self-hosted projects. This option is only available on a [Custom Plan](https://8thwall.com/custom).
:::
