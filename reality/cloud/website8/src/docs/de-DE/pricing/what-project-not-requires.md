---
title: Für welche Projekte ist keine kommerzielle Lizenz erforderlich?
priority: 2
---

Für ein Projekt ist keine kommerzielle Lizenz erforderlich, wenn es für den persönlichen Gebrauch oder für Bildungszwecke erstellt wurde, wenn es sich um ein nicht-kommerzielles künstlerisches Werk handelt, wenn es sich um eine private Demo handelt oder wenn es sich um ein Beispielprojekt handelt, das potenziellen Kunden vorgestellt werden soll. Wenn Sie spezielle Fragen zur Lizenzierung haben, senden Sie bitte eine E-Mail an licensing@8thwall.com.
