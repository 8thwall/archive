---
title: Für welche Projekte ist eine kommerzielle Lizenz erforderlich?
priority: 2
---

Ein Projekt erfordert eine kommerzielle Lizenz, wenn die URL öffentlich zugänglich sein wird und Sie planen, sie für kommerzielle Zwecke und/oder kostenpflichtige Anwendungen zu nutzen. Beispiel: Beispiel: 1) Sie sind eine Agentur und haben einen Vertrag zur Entwicklung einer WebAR-Kampagne für einen Kunden; 2) Sie sind ein Unternehmen, das ein Projekt aufbaut oder die 8th Wall-Technologie in seine Plattform integriert.

Dies gilt für Kampagnen, Aktivierungen, Showcases, Events, Verpackungen, einzelne bezahlte Projekte und langfristige Projekte. Wenn Sie spezielle Fragen zur Lizenzierung haben, senden Sie bitte eine E-Mail an licensing@8thwall.com.
