---
title: Was ist Virtual Reality?
defaultpriority: 0
webarpriority: 2
---

Virtual Reality (VR) ist eine Technologie, die es Benutzer_innen ermöglicht, eine vollständig simulierte oder virtuelle Welt zu erleben. In der Regel müssen Benutzer_innen ein Head-Mounted Display tragen, das die Sicht auf die reale Welt vollständig verdeckt. Einige VR-Erlebnisse sind jedoch auch auf 2D-Bildschirmen in 3D möglich.
