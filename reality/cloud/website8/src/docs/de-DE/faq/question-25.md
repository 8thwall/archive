Niantic VPS for Web bietet eine großzügige kostenlose Stufe an, die es Entwicklern ermöglicht, bis zu 25.00 Anrufe pro Monat ohne jegliche Kosten zu tätigen. Sobald ein Entwickler diesen Grenzwert überschreitet, wird er ab dem 25.001sten Anruf im ersten Monat in Rechnung gestellt. [Sehen Sie hier mehr](https://8th.io/vps-pricing).
