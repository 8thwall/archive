---
title: Kann ich 8th Wall-Projekte auf meinem eigenen Webserver hosten?
defaultpriority: 16
---

Wenn Sie einen kostenpflichtigen Pro- oder Enterprise-Plan haben, können Sie Ihre WebAR-Erfahrungen auf Ihrem eigenen Webserver hosten. Dazu müssen Sie eine Liste der zugelassenen URLs angeben, die für das Hosting Ihres Projekts zugelassen sind.
