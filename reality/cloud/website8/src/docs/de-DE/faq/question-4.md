---
title: Was ist WebVR?
defaultpriority: 0
webarpriority: 4
---

WebVR, oder webbasierte Virtual Reality, bezeichnet eine Technologie, die es ermöglicht, Virtual-Reality-Inhalte in einem Browser zu nutzen. WebVR kann auf Mobilgeräten, Tablets und Desktopcomputern als 3D-Erlebnis oder als Virtual-Reality-Erlebnis in Browsern über Head-Mounted Displays (HMDs) genutzt werden. Für WebVR muss keine App heruntergeladen werden. Benutzer\*innen greifen auf WebVR-Technologie zu, indem sie auf einen Link klicken oder einen QR-Code scannen, um eine VR-fähige Webseite in ihrem Browser zu öffnen.
