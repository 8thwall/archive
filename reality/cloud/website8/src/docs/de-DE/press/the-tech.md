---
title: Die Technik
---

8th Wall hat mit seinem für den Browser optimierten SLAM-System (Simultaneous Location and Mapping) im Jahr 2018 den Startschuss für WebAR gegeben. Die AR-Engine von 8th Wall, die vollständig auf standardkonformem JavaScript und WebGL basiert, ist eine vollständige Implementierung der SLAM-Engine (Simultaneous Localization and Mapping) von 8th Wall, die für interaktive, browserbasierte AR-Echtzeiterlebnisse optimiert wurde. Zu den Features gehören: Erfassung der sechs Freiheitsgrade (Six Degrees of Freedom, 6DoF), Licht- und Oberflächenschätzung, Bildzielerfassung (flach und gekrümmt), Lightship VPS und Gesichtseffekte.

Zusätzlich zur leistungsstarken AR-Engine von 8th Wall gibt es den Cloud Editor von 8th Wall – eine integrierte, cloudbasierte Entwicklungsumgebung, die speziell für die WebAR- und WebVR-Entwicklung konzipiert wurde. Neben den vielen Features bietet der 8th Wall Cloud Editor eine cloudbasierte Programmierumgebung mit integrierter Quellcodeverwaltung, Commitverlauf, Livevorschau, drahtlosem Remotedebugging und Veröffentlichung in einem globalen CDN auf Knopfdruck.
