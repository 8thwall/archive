title: What is WebVR?
defaultpriority: 0
webarpriority: 4
---

WebVR, or web-based virtual reality, refers to technology which enables virtual reality content to work within a browser. WebVR can be experienced on mobile, tablet and desktop computers as 3D experiences or as virtual reality experiences in browsers accessed via head-mounted devices or HMDs. WebVR requires no app to download. Users access WebVR by clicking a link or scanning a QR code to open a VR-enabled web page in the browser.
