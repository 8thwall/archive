---
title: Qu’est-ce qu’une vue ?
priority: 4
---

Pour les projets hébergés par 8th Wall, une « vue » est une demande d’un navigateur Web à la plateforme 8th Wall lorsqu’un utilisateur visite votre expérience WebAR. Pour les projets auto-hébergés, une « vue » est une requête d’un navigateur Web à la plateforme 8th Wall lorsqu’un utilisateur visite votre expérience WebAR et approuve les permissions pour lancer une session de AR.
