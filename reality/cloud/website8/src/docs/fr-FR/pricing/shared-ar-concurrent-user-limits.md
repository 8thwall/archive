---
title: Comment augmenter les limites d’utilisateurs simultanés pour mon application web qui utilise Shared AR ?
priority: 11
---

Le service RA partagé comprend des limites d’utilisateurs simultanés (CCU) par forfait. Si votre application Web est utilisée lors d’un événement où vous auriez un pic d’utilisation, veuillez nous contacter pour discuter des options de tarification et d’assistance. Nous pouvons prendre en charge votre application Web pour un plus grand nombre d’utilisateurs simultanés et potentiellement même pour différentes régions géographiques. Ces limites par forfait incluent l’utilisation de la bande passante.
