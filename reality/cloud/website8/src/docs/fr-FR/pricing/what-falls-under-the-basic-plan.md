---
title: Qu'est-ce qui relève du régime de base ?
priority: 8
---

Les expériences non commerciales et non marquées peuvent être réalisées avec le plan de base. Ce plan est idéal pour les étudiants, les artistes et les amateurs. Le plan de base permet de créer des expériences à l'aide de Niantic Studio, il n'inclut pas le Cloud Editor.
