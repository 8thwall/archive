---
title: Qu’est-ce que la technologie WebVR ?
defaultpriority: 0
webarpriority: 4
---

WebVR, ou réalité virtuelle basée sur le Web, désigne la technologie qui permet au contenu de réalité virtuelle de fonctionner dans un navigateur. Cette technologie peut être utilisée sur les téléphones portables, les tablettes et les ordinateurs de bureau comme expérience 3D ou comme expérience de réalité virtuelle dans les navigateurs accessibles via des dispositifs montés sur la tête (HMD). La technologie WebVR ne nécessite aucun téléchargement d’application. Les utilisateurs y accèdent en cliquant sur un lien ou en scannant un code QR pour ouvrir une page Web compatible avec la VR dans le navigateur.
