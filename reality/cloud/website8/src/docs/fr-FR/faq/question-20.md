---
title: La solution 8th Wall est-elle gratuite ?
defaultpriority: 20
---

8th Wall est gratuit pour les particuliers, les étudiants, les amateurs et les créateurs de jeux indépendants qui utilisent l'éditeur Niantic Studio. Pour les travaux commerciaux, un abonnement Pro est nécessaire ainsi qu'une licence commerciale. Commencez gratuitement [ici.](https://www.8thwall.com/get-started)
