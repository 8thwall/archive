---
title: Quel est le coût d'un SDV Niantic pour le Web ?
defaultpriority: 25
---

Niantic VPS for Web offre un niveau gratuit généreux, permettant aux développeurs d'effectuer jusqu'à 25 000 appels par mois sans frais. Lorsqu'un développeur dépasse cette limite, il est facturé à partir de son 25 001e appel au cours du premier mois. [Plus d'informations ici](https://8th.io/vps-pricing).
