---
title: Qu’est-ce que la technologie WebAR ?
defaultpriority: 0
webarpriority: 1
---

WebAR, également connu sous le nom de réalité augmentée basée sur le web, est une technologie innovante qui permet des expériences de réalité augmentée directement dans votre navigateur web sur n'importe quel appareil mobile, y compris les casques. Contrairement aux applications de RA traditionnelles, WebAR évite aux utilisateurs de télécharger des applications supplémentaires. Les utilisateurs n'ont qu'à cliquer sur un lien ou à scanner un QR code pour ouvrir une page web compatible avec la réalité augmentée directement dans leur navigateur, ce qui leur garantit une expérience de réalité augmentée fluide et efficace sans avoir à télécharger d'applications.
