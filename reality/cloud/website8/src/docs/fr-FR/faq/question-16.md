---
title: Puis-je héberger les projets 8th Wall sur mon propre serveur Web ?
defaultpriority: 16
---

Si vous bénéficiez d'un plan Pro ou Enterprise payant, vous pouvez héberger vos expériences WebAR sur votre propre serveur web. Pour ce faire, vous devez spécifier une liste d’URL approuvées pour héberger votre projet.
