defaultpriority: 20

8th Wall es gratuito para particulares, estudiantes, aficionados y creadores de juegos independientes que construyan con nuestro editor Niantic Studio. Para trabajos comerciales, se requiere una suscripción Pro junto con una licencia comercial. Empiece gratis [aquí](https://www.8thwall.com/get-started).
