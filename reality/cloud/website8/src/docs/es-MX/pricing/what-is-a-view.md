---
title: ¿Qué es una vista?
priority: 4
---

Para los proyectos alojados en 8th Wall, una "vista" es una solicitud de un navegador web a la plataforma 8th Wall cuando un usuario visita su experiencia WebAR. Para los proyectos autoalojados, una "vista" es una solicitud de un navegador web a la plataforma 8th Wall cuando un usuario visita su experiencia WebAR y aprueba los permisos para iniciar una sesión de RA.
