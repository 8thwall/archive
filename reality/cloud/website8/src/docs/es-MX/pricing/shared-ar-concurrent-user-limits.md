---
title: '¿Cómo puedo aumentar los límites de usuarios concurrentes para mi aplicación web que utiliza AR compartida?'
priority: 11
---

La AR compartida incluye límites de usuarios simultáneos (CCU) por plan. Si su aplicación web se utiliza en un evento en el que se produciría una explosión de uso, póngase en contacto con nosotros para hablar sobre las opciones de precios y asistencia. Podemos dar soporte a su aplicación web a través de más usuarios simultáneos y potencialmente incluso diferentes regiones geográficas. Estos límites por plan incluyen el uso de ancho de banda.
