---
title: ¿Qué proyectos no requieren licencia comercial?
priority: 3
---

Un proyecto no requiere una licencia comercial si se crea para uso personal o con fines educativos, una obra de expresión artística no comercial, una demostración privada o un proyecto de muestra para mostrar el trabajo a posibles clientes. Si tiene preguntas específicas sobre licencias, envíe un correo electrónico a licensing@8thwall.com.
