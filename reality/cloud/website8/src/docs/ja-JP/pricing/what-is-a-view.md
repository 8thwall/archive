8th Wallがホストするプロジェクトでは、View とは、ユーザーがあなたのWebARエクスペリエンスを訪問した際に、Webブラウザから8th Wallプラットフォームへのリクエストのことです。 セルフホスト型プロジェクトの場合、Viewとは、ユーザーがあなたのWebARエクスペリエンスを訪問し、ARセッションを開始するためのパーミッションを承認する際に、Webブラウザから8th Wallプラットフォームへのリクエストのことです。 セルフホスト型プロジェクトの場合、Viewとは、ユーザーがあなたのWebARエクスペリエンスを訪問し、ARセッションを開始するためのパーミッションを承認する際に、Webブラウザから8th Wallプラットフォームへのリクエストのことです。
