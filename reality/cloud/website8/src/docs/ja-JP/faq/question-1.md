title: 拡張現実（Augmented Reality）とは？
defaultpriority: 0
webarpriority: 2

AR（Augmented Reality：拡張現実）とは、現実世界とデジタル情報やオブジェクトを同時に体験できる技術です。 モバイル機器やウェアラブル機器などのディスプレイを通して現実世界を見る必要があり、コンピュータビジョン技術を利用して、ユーザーの世界体験をコンピュータグラフィックスで補強するものです。 モバイル機器やウェアラブル機器などのディスプレイを通して現実世界を見る必要があり、コンピュータビジョン技術を利用して、ユーザーの世界体験をコンピュータグラフィックスで補強するものです。
