# A-Frame: Light Estimation

This interactive example shows how to use light estimation to control lights in your scene.

Turn off the lights and watch the car lights turn on!

![](https://media.giphy.com/media/H5BcotxShNPRoFy1Ng/giphy.gif)

```xr-light``` controls a light's intensity value.
- min: minimum light intensity (default: 0)
- max: maximum light intensity (default: 2)

Learn more about this in our docs [here](https://www.8thwall.com/docs/web/#xr8xrcontroller)
