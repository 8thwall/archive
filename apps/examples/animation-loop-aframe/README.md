# A-Frame: Animation Mixer Loop / Replay

This example illustrates how to use the AFrame "animation-mixer" component to: 

1. automatically loop through a list of animation clips within a 3d model (loop.html)
2. play back animation clips after they complete (replay.html)

![](https://media.giphy.com/media/Su738nSVeEnp36q5iA/giphy.gif)

Learn more about the component [here](https://github.com/n5ro/aframe-extras/tree/master/src/loaders#animation) 
and check out the A-Frame: Animation Mixer project on the [Project Library](https://www.8thwall.com/8thwall/animation-mixer-aframe).
