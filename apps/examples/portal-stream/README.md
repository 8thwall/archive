# A-Frame: World Tracking Portal 360 Livestream Video

This example shows off the popular portal illusion in web AR while livestreaming a 360 video using
[hls.js](https://github.com/video-dev/hls.js/).

![](https://media.giphy.com/media/8d8e3AiH3lYfOpSECF/giphy.gif)

### NOTE: iOS 14.0 - 14.1 (and iPhone 12 devices) contain a bug that affects HLS streaming in WebGL, blocking HLS playback.
