# A-Frame: Custom Capture Button

This project demonstrates how to customize the media recorder capture button using Javascript and CSS.

To customize the capture button add `<xrextras-capture-button>` to the scene and inspect the page to view it's HTML element/styles. Inner HTML can be modified using Javascript as shown in app.js. Styles can be modifies by reseting and/or overriding the id/class of the container/button/progress as shown in main.css.
