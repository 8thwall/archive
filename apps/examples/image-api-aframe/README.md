# A-Frame: Endless Image Targets

This example uses the [image target API](https://www.8thwall.com/docs/web/#changing-active-image-targets) to set 10 image targets programmatically at init. 
When the last image in the sequence is scanned (rooster), it loads a new set of image targets (flightless birds).
