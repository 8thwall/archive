# three.js: Record Scene Audio with MediaRecorder

The goal of this project is to show how easy it is to add sounds in THREE.js that are
part of the MediaRecorder's audio output.

Learn more [here](https://www.8thwall.com/docs/web/#xr8mediarecorder).
