# A-Frame: Face Effects with Additive Blending

This A-Frame example shows how to use video materials with blending: additive to create a bloom or laser effect.

![](https://media.giphy.com/media/mC4lCZJKyuVhsbiOVu/giphy.gif)

### Setting Blending Additive on Materials

In this example project we use two videos (blue-laser.mp4 and color-laser.mp4) each video has a black background
which lends itself to work well when setting a material to additive blending. 

We then create two cylinders (overlapping each other to create a layering effect) and apply the videos as textures and autoplay them. 

You can take a look further at blending materials in Aframe [here] (https://aframe.io/docs/1.3.0/components/material.html)
