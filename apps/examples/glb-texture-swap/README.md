# GLB Texture Swap

Simple component that swaps a glb texture.

![](https://media.giphy.com/media/LkeSUF2eldYYF7apwM/giphy.gif)

NOTE: WHEN GRABBING A MESH NAME, ANY PERIOD USED IN THE NAMING CONVENTION IS 
IGNORED BY THREE.JS, AS AN EXAMPLE A MESH NAMED "CUBE.001" IN BLENDER WOULD BE 
GRABBED VIA THE NAME "CUBE001" INSTEAD
