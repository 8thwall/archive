# Lightship VPS: three.js Template

Use the Lightship VPS A-Frame Template to bootstrap VPS development.

To learn more about developing bespoke and procedural VPS experiences, see the [Lightship VPS Documentation](https://www.8thwall.com/docs/guides/lightship-vps/)

### Quick Start

1. Add a test scan or public Location to the project via the Geospatial Browser (https://www.8thwall.com/docs/web/#managing-wayspots)
2. Download the selected mesh as a glb from the geospatial browser, upload to the project as 'mesh.glb', then uncomment line 16 of `threejs-scene-init.js`
