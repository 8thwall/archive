# A-Frame: Place Model

This example project allows the user to place a "ghosted" version of a 3D model before placing it. 
You can change what is shown via the cursor by changing the cursor entity. 

![](https://media.giphy.com/media/haM0i40ABu1vGnAjE0/giphy.gif)

### Transparent Model

To learn more about how to create a transparent 3D model take a look at this sample project
[```here```](https://www.8thwall.com/playground/glass-materials-aframe).
