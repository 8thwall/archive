# three.js: Gummy Transmission

This example modifies scene rendering in a three.js r132 project to allow the 
[KHR_materials_transmission](https://github.com/KhronosGroup/glTF/blob/master/extensions/2.0/Khronos/KHR_materials_transmission/README.md) 
glTF extension to include the camera feed.

This scene also applies realtime reflections to the 3D model. Learn more [here](https://www.8thwall.com/playground/cubemap-realtime-threejs).

![](https://media.giphy.com/media/aY2iTlxwCwBgiIW7SH/giphy.gif)

__REQUIRES: three.js r131+ and a glb/glTF model with transmission properties__
