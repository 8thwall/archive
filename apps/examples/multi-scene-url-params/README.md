# A-Frame: Multi-Scene using URL Parameters

Simple scene showing how to use URL params to load different HTML scenes.

Useful for combining several scenes in the same project without using React, Vue, etc.

![](https://media.giphy.com/media/pT6dsVTMrgzTWPGf0p/giphy.gif)

Just type *red*, *blue*, or *green* after the `?color=` string in the URL to see the scene change.
