# A-Frame: Photo Frame

This project demonstrates how to add a photo frame to photo/video capture using the media recorder and foregroundCanvas parameter.

To include 2D elements in media capture, we manually draw to a seperate canvas and pass it to the CanvasScreenshot and MediaRecorder pipeline modules.
