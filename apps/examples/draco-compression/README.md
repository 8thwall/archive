# A-Frame: Draco Compression

Shows how to include the draco decoder in your A-Frame project, allowing the loading of draco-compressed gLTF models.

![](https://media.giphy.com/media/8CDIfbuLpNWKDkTm2r/giphy.gif)

Learn more about Draco compression [here](https://google.github.io/draco/).

A-Frame Documentation located [here](https://aframe.io/docs/1.0.0/components/gltf-model.html#scene-properties).
