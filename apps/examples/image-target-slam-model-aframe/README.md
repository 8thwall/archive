# A-Frame: Image Target + SLAM Manipulate 3D Model

This A-Frame example project showcases how to initiate a SLAM 3D model on an xrimagefound event. 
To learn more about image target events click [here] (https://www.youtube.com/watch?v=VwQk4msRPs4&t=461s)
