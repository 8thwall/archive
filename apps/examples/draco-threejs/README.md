# three.js: Draco Compression

Shows how to include the draco decoder in your three.js project, allowing the loading of draco-compressed gLTF models.

![](https://media.giphy.com/media/8CDIfbuLpNWKDkTm2r/giphy.gif)

Learn more about Draco compression [here](https://google.github.io/draco/).

THREE.DRACOLoader Documentation located [here](https://threejs.org/docs/#examples/en/loaders/DRACOLoader).
