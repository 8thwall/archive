# Demo Project to use webxr-layer in Three-js with NAE

Copied from this Three example [here](https://threejs.org/examples/webxr_vr_layers.html)

That example use quite a few dependencies that wasn't supported in NAE by May 2025. That's why we
forked it here for easier modification and testing

## Run Server

```sh
bazel run //apps/client/exploratory/webxr-layer:serve
```
