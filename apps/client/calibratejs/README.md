# What is needed

* Run `bazel run --platforms=//bzl:wasm32 //apps/client/calibratejs:serve`
* Make sure you have a token with the correct engine version for 8w/pipeline-tester project.
* Navigate your phone browser to `https://<your_ip>:8888/apps/client/calibratejs/index.html`.
* Point the camera at a printed chessboard.
