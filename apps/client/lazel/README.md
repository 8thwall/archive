# Lazel

A bazel GUI for people who have better things to do than type the same bazel commands over and over.

https://www.figma.com/design/2RIb05k97JTlRsR1dC9AKl/Lazel

## Planned Features

- Build/Run/Test using bazel
- Configure targets/flags
- View outputs
- Save, load, and share configurations
